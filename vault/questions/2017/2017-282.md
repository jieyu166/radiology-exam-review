#交換 #2017交換 #IR 關於 transarterial embolization (TAE) for post-partum hemorrhage (PPH) 下列何者錯誤?
(A) Bleeding sources 主要是 uterine arteries 或 ovarian arteries.
(B) 若 hysterectomy 後仍在出血, sole source 為 ovarian arteries.
(C) 使用 coils 較 gelfoam 容易失敗.
(D) 目前對 menopause 和 pregnancy 無顯著影響.
??
**Ans: A**
**主要出血來源為子宮動脈(uterine artery)**，卵巢/圓韌帶動脈為主要側枝吻合來源而非出血源。
(A) **錯誤（正解）**:主要出血源僅為uterine artery，非uterine與ovarian arteries並列。
(B) 正確:子宮切除後持續出血非單純卵巢動脈為唯一來源。
(C) 正確:coils較gelfoam容易栓塞失敗。
(D) 正確:目前對停經/懷孕無顯著不良影響。
