#交換 #2017交換 #NR Regarding calcification in a thyroid nodule, which of the following ultrasound finding indicates benign entity?
(A) calcification associated with comet tail artifact
(B) egg shell calcification
(C) punctate calcification
(D) coarse calcification
??
**Ans: A**
在cyst內看到，和 **colloid** 有關，強烈暗示為良性(benignity)  給0分
B. Peripheral calcification跟惡性關聯variable。但比macrocalcification較危險，TI-RADS算2分。如果邊緣不平整，有些文獻認為和protruding soft tissue有關，但specificity很低。在TI-RADS中，這些lesion會因為lobulated margin額外給2分。
D. “Macrocalciﬁcations”: coarse echogenic foci+acoustic shadowing。和malignancy關聯度mixed，可能有微弱關聯，故給1分。
