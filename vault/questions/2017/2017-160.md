#交換 #2017交換 #CV Which of the following adjustments is used to decrease beam-hardening artifact for cardiac CT?
(A) Increase the tube potential.
(B) Decrease the section thickness.
(C) Improve the spatial resolution.
(D) Improve the temporal resolution.
??
**Ans: A**
正解 **A**。**增加 tube potential（高 kVp）** 降低 beam-hardening artifact（高能光子較不受 beam hardening 影響）。亦可用濾片、虛擬單能量高能影像、iterative reconstruction。
