#交換 #2017交換 #CV The course of the short LAD is along the proximal AIVS; the long LAD has an epicardial course on the LV side of the proximal AIVS, reentering the distal AIVS. Which type of dual LAD is this?
(A) Type I.
(B) Type II.
(C) Type III.
(D) Type IV.
??
**Ans: A**
正解 **A（Type I）**。short LAD 走近端 AIVS、long LAD epicardial 於 LV 側再入遠端 AIVS,分支 septal＋diagonal ＝ Type I。
