#交換 #2019交換 #GU Regarding CT protocol for adrenal incidentaloma washout study, which of the following is NOT needed?
(1) Unenhanced CT
(2) 5-min delayed CT
(3) 15-min delayed CT
(4) Enhanced CT (60-70 seconds)
??
**Ans: 2**
正確答案為 **2（5-min delayed CT）**。標準 adrenal washout protocol 僅需 unenhanced + enhanced + 15-min delayed，不包含 5 分鐘延遲。Absolute washout ≥60% 或 relative washout ≥40% 提示 adenoma。詳見 [[adrenal-incidentaloma-washout]]。
