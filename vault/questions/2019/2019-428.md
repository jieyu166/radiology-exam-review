#交換 #2019交換 #PE Regarding meconium peritonitis, which statement is **WRONG**?
(A) It is a chemical peritonitis caused by in utero bowel perforation
(B) Fibroadhesive type is the most common
(C) Water-soluble contrast enema can be used for differential diagnosis
(D) No case requires surgery despite calcification, ascites, or pseudocyst
??
**Ans: D**
正確答案為 **D**。嚴重 meconium peritonitis（伴有 calcification + ascites + pseudocyst）仍可能需要手術。並非所有病例都不需手術。詳見 [[meconium-peritonitis]]。
