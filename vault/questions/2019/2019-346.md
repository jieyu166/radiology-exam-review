#交換 #2019交換 #ABD Which of the following is a **true** diverticulum of the small bowel?
(A) Meckel diverticulum
(B) Duodenal diverticulum
(C) Jejunal diverticulum
(D) Ileal diverticulum
??
**Ans: A**
正確答案為 **A**（Meckel diverticulum）。True diverticulum 包含腸壁所有層次（mucosa + submucosa + muscularis propria）。小腸中唯一常見的 true diverticulum 是 **Meckel diverticulum**（卵黃管殘留）。十二指腸、空腸、迴腸憩室皆為 false (pseudo) diverticulum，僅含黏膜與黏膜下層。
