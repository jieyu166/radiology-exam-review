#交換 #2019交換 #GI Which type of tracheoesophageal fistula (TEF) is the most common?
(A) Type A (pure EA, no fistula)
(B) Type B (EA + proximal fistula)
(C) Type C (EA + distal fistula)
(D) Type E (H-type fistula, no EA)
??
**Ans: C**
正確答案為 **C（Type C）**。Type C（食道閉鎖＋遠端瘔管）約占 85-86%，為最常見。記憶：Type C = most Common。詳見 [[tracheoesophageal-fistula]]。
