#交換 #2019交換 #CH 根據 revised ATS/ERS/JRS/ALAT diagnostic criteria for idiopathic pulmonary fibrosis，下列 pattern 何者錯誤?
(A) UIP pattern
(B) NSIP pattern
(C) Possible UIP pattern
(D) Inconsistent with UIP pattern
??
**Ans: B**
正確答案為 **B**。ATS/ERS/JRS/ALAT IPF 診斷標準的 HRCT pattern 分類為：UIP pattern、Possible UIP pattern、Inconsistent with UIP pattern。**NSIP pattern 不在此分類中** — NSIP 為獨立的 IIP 亞型。
