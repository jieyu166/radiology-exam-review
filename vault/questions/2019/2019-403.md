#交換 #2019交換 #IR 假設病患的右肱動脈血壓為 a mmHg，左肱動脈血壓為 b mmHg，右大腿血壓為 c mmHg，左大腿血壓為 d mmHg。請問關於 ABI (ankle-brachial pressure index) 之敘述下列何者錯誤?
(A) 右大腿之 ABI 為 c / (a+b)
(B) 如果右大腿之 ABI= 1.3，原因可能是右大腿動脈嚴重鈣化
(C) ABI < 0.9 表示動脈異常
(D) 以上皆正確
??
**Ans: A**
正確答案為 **A**。ABI = ankle SBP / 兩側 brachial SBP 中**較高者**（非兩側之和）。正確公式應為 c / max(a,b)。詳見 [[ankle-brachial-index]]。
