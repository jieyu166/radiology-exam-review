#交換 #2019交換 #ABD Which is **wrong** about image evaluation for biliary atresia?
(A) Biliary atresia can be ruled out if the entire extrahepatic bile duct is seen.
(B) The differential diagnosis of prolonged infant hyperbilirubinemia includes biliary atresia and neonatal hepatitis.
(C) The following image modality can help in diagnosing biliary atresia: sonography, scintigraphy, and MRI.
(D) The diagnosis of biliary atresia can be confirmed if we cannot see extrahepatic bile duct in MRCP study.
??
**Ans: D**
正確答案為 **D**（錯誤敘述）。膽道閉鎖的確診需 **intraoperative cholangiogram**，MRCP 看不到肝外膽管並不能確診（可能為技術限制或新生兒膽管太小）。A（見到完整膽管可排除）、B（鑑別含BA與新生兒肝炎）、C（US/scintigraphy/MRI有助診斷）皆正確。
