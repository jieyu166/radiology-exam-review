#交換 #2019交換 #BR How can we decrease radiation doses when performing film-screen mammography?
(A) No grid for breast with higher fatty content
(B) Use fast screen/film
(C) Extended processing
(D) All of the above
??
**Ans: D**
正確答案為 **D**。降低傳統底片式乳房攝影劑量的三種方法皆正確：(A) 脂肪型乳房散射少，可省略 grid 降低劑量；(B) 快速增感屏/底片系統感度高，降低所需曝光量；(C) 延長顯影處理可增加底片感度。
