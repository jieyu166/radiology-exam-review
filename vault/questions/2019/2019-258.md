#交換 #2019交換 #PE According to Salter-Harris classification, which type of fracture consists of a crush injury to part or all of a physis?
(A) Type 2
(B) Type 3
(C) Type 4
(D) Type 5
??
**Ans: D**
正確答案為 **D（Type 5）**。Salter-Harris Type V = 生長板壓碎傷（crush injury），X光可能初期正常，診斷常為回顧性。詳見 [[salter-harris]]。
