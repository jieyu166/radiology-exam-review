#交換 #2019交換 #CH Which is the most common type of congenital pulmonary adenomatoid malformation (CPAM)?
(A) Type I
(B) Type II
(C) Type III
(D) Type IV
??
**Ans: A**
正確答案為 **A（Type I）**。CPAM Type I 最常見（~65%），含大囊（>2 cm），預後最佳。詳見 [[cpam-congenital-pulmonary]]。
