#交換 #2019交換 #CV Which of the following is **true** regarding detection and quantification of coronary artery calcification (calcium scoring) by CT?
(A) Agatston scoring is traditionally performed on images acquired using 100kVp.
(B) Should be performed using retrospective ECG-gating
(C) Is performed using helical (spiral) acquisition
(D) Is performed using a slice thickness of 2.5-3.0 mm
??
**Ans: D**
正確答案為 **D**。冠狀動脈鈣化分數（Agatston score）以非顯影、前瞻性心電圖閘控（prospective ECG-gated）之單次屏氣掃描取得，管電壓固定 **120 kVp**（非100kVp），採 **step-and-shoot（非helical）**方式，層厚 **2.5–3.0 mm**。詳見 [[calcium-score]]。
