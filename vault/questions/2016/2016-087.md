#交換 #2016交換 #NR In the pediatric population, which type is most common in the spinal cord AVM?
(A) Type 1, dural AVM
(B) Type 2, glomus AVM
(C) Type 3, juvenile AVM
(D) Type 4, perimedullary AVM
??
**Ans: B**
兒童以 **type 2 glomus(髓內)** AVM 最常見。
(A) Type 1 dural:整體最常見但好發老年男性。
(B) **Type 2 glomus(正解)**:髓內、年輕/兒童最常見。
(C) Type 3 juvenile:罕。
(D) Type 4 perimedullary AVF。
