#交換 #2016交換 #ABD Which is **incorrect** about congenital biliary cysts (Todani classification)?
(A) Type V is Caroli disease.
(B) Type I is choledochal cyst and accounts for 77-78% of cases.
(C) Choledochocele is type IV.
(D) Choledochal diverticulum is type II.
??
**Ans: C**
正解 **C**（不正確）。**choledochocele 是 type III**（非 IV）。Type V＝Caroli(A)、Type I 最常見 ~77–78%(B)、Type II＝膽總管憩室(D)皆正確。![[ADD Todani classification.png]]
