#交換 #2016交換 #CV 關於 Endoleaks of the aorta 下列敘述何者正確?
(A) 最常見的是 type I
(B) 其中 type II 與 aortic rupture 有關
(C) 其中 type III 與 graft porosity 相關
(D) 造成 type I 的原因多是因為 device failure
??
**Ans: D**
消去法:最常見是 type II(非 I)、與破裂相關是 type I/III(非 II)、porosity 是 type IV(非 III) → 唯 (D) type I=封合/裝置失敗正確。
(A) 最常見 type I:錯(type II 最常見)。
(B) type II 與破裂:錯(type I/III)。
(C) type III 與 graft porosity:錯(type IV)。
(D) **type I 因 device 失敗(正解)**:type I=附著處封合不全。
