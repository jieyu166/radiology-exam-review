#交換 #2020交換 #NR Which type of the cavernous-carotid fistula is the most common?
(A) Type A
(B) Type B
(C) Type C
(D) Type D
??
**Ans: A**
**Barrow type A（直接型carotid-cavernous fistula）最常見**，故(A)為正解。
(A) **對（正解）**：Type A為直接型（頸內動脈與海綿竇直接相通），最常見類型；主因為外傷或動脈瘤破裂。
(B)(C)(D) 錯：Type B/C/D為間接型（dural AVF），繼發於海綿竇血栓合併再血管化，較少見。
