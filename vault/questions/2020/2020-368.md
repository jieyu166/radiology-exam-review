#交換 #2020交換 #MS Which statement is **true** about rotational malleolar fracture?
(A) Type A is an infrasyndesmotic injury, in which the fibular fracture is transverse and located below the syndesmosis.
(B) Type B is the most common type of ankle fracture, describing a trans-syndesmotic injury.
(C) Type C is a suprasyndesmotic injury, resulting from pronation-external rotation force.
(D) All of above.
??
**Ans: D**
旋轉性內外踝骨折依Weber(AO)分類：Type A為韌帶聯合下方損傷(infrasyndesmotic)，腓骨骨折呈橫向且位於韌帶聯合下方；Type B為最常見之踝關節骨折型態，描述經韌帶聯合損傷(trans-syndesmotic)；Type C為韌帶聯合上方損傷(suprasyndesmotic)，由旋前外旋力(pronation-external rotation)造成。三者描述皆正確，故答案為D。
