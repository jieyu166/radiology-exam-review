#交換 #2020交換 #NR Which of the following disease is NOT shown as multifocal hypointense foci on T2* GRE images?
(A) Cavernous malformations
(B) Diffuse axonal injury
(C) Neurofibromatosis type I
(D) Cerebral amyloid angiopathy
??
**Ans: C**
**T2* GRE多發低訊號焦點(microbleeds)之常見病因為cavernous malformations(Zabramski type IV)、diffuse axonal injury、慢性高血壓性腦病變、cerebral amyloid angiopathy；neurofibromatosis type I並非此類典型病因**，故(C)為正解。
(A) 錯：cavernous malformations為T2* GRE多發低訊號焦點常見病因（Zabramski type IV）。
(B) 錯：diffuse axonal injury為常見病因之一。
(C) **對（正解）**：neurofibromatosis type I並非T2* GRE多發低訊號焦點之典型病因。
(D) 錯：cerebral amyloid angiopathy為常見病因之一。
