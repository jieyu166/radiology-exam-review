#交換 #2018交換 #MSK 在 odontoid fracture 分類中，何種type為unstable?
(A) type 1
(B) type 2
(C) type 3
(D) a+b
(E) b+c
??
**Ans: E**
依本題來源 Tintinalli's Emergency Medicine，Anderson-D'Alonzo type II 與 type III 皆列為 unstable（b+c）。
> [!warning] 此分類與部分骨科 Tier 1 文獻（如 Acta Biomed 系列，見 [[odontoid-fracture]]）認為 type III 因骨折面積大、血供佳而相對穩定、易癒合的描述有出入，屬文獻分歧，建議對照原始考卷與兩方文獻脈絡確認。
