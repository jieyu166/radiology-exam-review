#交換 #2018交換 #IR Which one of the followings is the **wrong** description about endoleak after endovascular aneurysmal repair?
(A) Type 1 endoleak reaches the aneurysm around the end of stent graft
(B) Type 2 endoleak reaches the aneurysm by retrograde flow through lumbar or inferior mesenteric branches
(C) Type 3 endoleak reaches the aneurysm through a defect in the wall of stent-graft
(D) Type 2 endoleak should be treated as soon as possible
??
**Ans: D**
type II endoleak多為良性、常自行消退，僅在瘤囊持續增大(>5mm)時才介入，非「盡快治療」。
