# pancreatic-anatomic-variants

**胰臟最常見的解剖變異是 pancreas divisum（背側與腹側胰管未融合），環狀胰為次常見但知名度高的變異；環狀胰約 1/3 為 incomplete type（未完全環繞十二指腸），十二指腸阻塞風險與 complete type 相近。** 判讀分水嶺：**「最常見」變異＝pancreas divisum，非 annular pancreas**——考題常以環狀胰的知名度誤導。

## Summary
- **胰臟最常見解剖變異：pancreas divisum**（背側胰管 duct of Santorini 與腹側胰管 duct of Wirsung 未正常融合，背側胰管經副乳頭獨立引流大部分胰液）；其餘選項（partial dorsal agenesis、anomalous pancreatobiliary duct union、annular pancreas）皆非最常見。[^1]
- **Annular pancreas（環狀胰）**：胰腺組織異常環繞十二指腸第二段（降部），可致十二指腸阻塞；依環繞完整度分 **complete type**（完全環繞）與 **incomplete type**（未完全環繞）。[^2]
- **Incomplete type 比例約 1/3**：影像學上判定為 incomplete 環狀胰者約占**1/3**病例。[^2]
- **阻塞風險不因型別而異**：**incomplete type 之十二指腸阻塞風險（約33%）與 complete type（約40%）相近**（p=0.75，無統計差異）——故「incomplete 型阻塞風險較低」為錯誤觀念，不可用環繞完整度預測阻塞風險。[^2]

## 判讀骨架
- **Pancreas divisum 影像**：MRCP/ERCP 可見背側胰管（Santorini duct）經副乳頭獨立引流、未與腹側胰管（Wirsung duct）匯合；為最常見胰臟先天變異（一般族群約 4–10%），多數無症狀，少數併發復發性胰臟炎。[^1]
- **環狀胰影像**：CT/MRI 見胰腺組織環繞十二指腸第二段，需評估環繞完整度（complete vs incomplete）與是否合併十二指腸狹窄/阻塞徵象;完整度不等於臨床阻塞風險，兩型阻塞率相近。[^2]
