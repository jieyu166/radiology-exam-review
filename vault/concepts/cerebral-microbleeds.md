# cerebral-microbleeds

**T2* GRE/SWI上多發低訊號焦點(cerebral microbleeds)的常見病因四大類：cavernous malformations（Zabramski type IV）、慢性高血壓性腦病變、cerebral amyloid angiopathy、diffuse axonal injury——四者才是常見病因，神經纖維瘤病第一型（NF1）並不在此常見病因清單中。**

## Summary
- **本質**：小灶局部腦內出血，常僅在**susceptibility-sensitive MRI序列**（T2* GRE、SWI）上可見，稱為cerebral microhemorrhages/microbleeds。[^1]
- **常見病因**：
  - **Cavernous malformations（Zabramski type IV）**[^1]
  - **慢性高血壓性腦病變（chronic hypertensive encephalopathy）**[^1]
  - **Cerebral amyloid angiopathy**[^1]
  - **Diffuse axonal injury (DAI)**[^1]
- **少見病因**：出血性微轉移（黑色素瘤或腎細胞癌）、急性出血性白質腦炎(AHLE)、放射誘發腦血管病變、腦血管炎、血栓性微血管病變、顱內栓塞（脂肪、氣體、敗血性）、COL4A1/2腦小血管病。[^1]
- **神經纖維瘤病第一型(NF1)不屬於典型病因**——NF1相關腦部異常主要為unidentified bright objects (UBOs)於T2/FLAIR高訊號，非以T2* GRE多發低訊號微出血為特徵表現。[^2]

> [!note] 考點：何者「非」T2* GRE多發低訊號焦點之病因?
> **Neurofibromatosis type I（C）**——非典型病因。Cavernous malformations、diffuse axonal injury、cerebral amyloid angiopathy皆為常見病因。[^1][^2]

### 參考來源
[^1]: 官方2020詳解（題目所引）；標準腦微出血病因分類：常見病因（cavernous malformations Zabramski IV、慢性高血壓性腦病變、cerebral amyloid angiopathy、diffuse axonal injury）與少見病因（出血性微轉移、AHLE、放射誘發血管病變、腦血管炎、血栓性微血管病變、顱內栓塞、COL4A1/2小血管病）。
[^2]: 官方2020詳解（題目所引，交叉查證）：NF1腦部影像典型表現為T2/FLAIR高訊號之unidentified bright objects (UBOs)，非T2* GRE多發低訊號微出血病因。
