# renal-av-shunt-classification

**腎動靜脈瘻管血管攝影分類靠「創傷史＋血管構造」兩軸判讀：外傷性（traumatic）呈單一動脈對單一引流靜脈的直接瘻管，常合併假性動脈瘤；非外傷性再依供血動脈數目與引流複雜度分三型（I：單/少數動脈對單一擴張引流靜脈；II：多條小動脈對單一擴張引流靜脈；III：多重動靜脈間分流形成複雜血管網）。**

## Summary
- **Traumatic renal AV shunt（外傷性）**：血管攝影呈**單一動脈與單一引流靜脈間的直接瘻管形成**，**常合併假性動脈瘤（pseudoaneurysm）並存**——本題考點所在，此組合特徵即定義為外傷性。[^1]
- **Nontraumatic renal AV shunt（非外傷性）分三型**：[^1]
  - **Type I**：單一或少數幾條動脈直接分流至**單一擴張的引流靜脈**。
  - **Type II**：**多條小動脈**分流至單一擴張的引流靜脈（形成較大靜脈囊，多條供血動脈匯入）。
  - **Type III**：**小動脈與小靜脈間多重分流**，形成複雜血管網（nidus），通常由多條供血動脈與多條引流靜脈組成。

## 判讀骨架
- **判讀第一步：問病史**——有無外傷史、有無假性動脈瘤並存，決定是否歸為 traumatic 型。
- **判讀第二步（無外傷史時）：數供血動脈與引流靜脈的複雜度**——單純單一供血血管對單一引流靜脈為 Type I；多條供血小動脈匯聚單一引流靜脈為 Type II；分流本身呈複雜網狀（非單一匯聚點）為 Type III。
- **臨床意義**：分類直接影響經導管栓塞（transcatheter embolization）之技術策略選擇（單純瘻管栓塞 vs 複雜血管網之栓塞劑選擇與進路規劃）。

### 參考來源
