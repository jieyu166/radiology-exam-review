# ivc-leiomyosarcoma

**IVC leiomyosarcoma依侵犯節段分三段，各自對應一個特徵性臨床症候群：上段（肝靜脈到右心房）→Budd-Chiari症候群；中段（肝靜脈到腎靜脈）→腎病症候群；下段（腎靜脈以下）→下肢水腫。記憶鉤：「愈往上愈靠近心臟愈危險（Budd-Chiari），愈往下愈單純（腿腫）」。**

## Summary
- **上段（upper segment）**：肝靜脈至右心房之間 → 阻塞肝靜脈回流 → **Budd-Chiari症候群**。[^1]
- **中段（middle segment）**：肝靜脈至腎靜脈之間 → 阻塞腎靜脈回流 → **腎病症候群（nephrotic syndrome）**。[^1]
- **下段（lower segment）**：腎靜脈以下（infrarenal） → 阻塞下肢靜脈回流 → **下肢水腫**。[^1]
- 上述三段-症候群對應關係為官方詳解確認之正確敘述（本題官方答案為「以上皆對」）。[^1]

> [!note] 考點：IVC leiomyosarcoma三段分類敘述何者「不正確」？
> **以上敘述皆正確（D）**——上段(肝靜脈-右心房)致Budd-Chiari症候群、中段(肝靜脈-腎靜脈)致腎病症候群、下段(腎靜脈以下)致下肢水腫，三者皆為正確描述。[^1]
