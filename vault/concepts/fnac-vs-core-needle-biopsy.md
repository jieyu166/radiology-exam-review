# fnac-vs-core-needle-biopsy

**乳房FNAC最適合的導引方式是超音波（快、安全，亦可用於水泡抽出），而非MR-guided——MR-guided活檢通常保留給core needle biopsy；FNAC整體敏感度仍低於core needle biopsy，且高度仰賴cytopathologist經驗，這三個限制點是此題型的核心考點。**

## Summary
- **FNAC最適合之導引方式**：**超音波導引**（快、安全，亦可用於水泡/囊腫之抽出液），**非**MR-guided FNAC；MR-guided較常用於需組織學（core needle biopsy）之場合。[^1]
- **準確度取決於判讀者經驗**：當有**經驗豐富之cytopathologist**判讀時，FNAC準確度方能提高——此為FNAC準確度高度仰賴人員經驗之限制。[^1]
- **FNAC vs Core needle biopsy敏感度**：**FNAC敏感度低於core needle biopsy**。超音波導引之core needle biopsy以14G切片針取樣：**取一條檢體準確度約70%，取三條檢體準確度可達95%**（取樣數量顯著影響準確度）。[^1]
- **FNAC相對優勢族群**：於**B2/C2**病灶（如fibroadenoma）表現優越，構成FNA門診之主要病例。[^1]
- **FNAC角色限制**：於gray zone病灶（**B3/C3、B4/C4**）角色有限；多數**B3/C3以上**病灶（如atypical ductal hyperplasia、in-situ carcinoma、papillary lesions）需明確組織學診斷以利後續處置（含是否手術/化療），FNAC於此類病灶有其侷限性。[^1]

> [!note] 考點：關於breast FNAC與core needle biopsy,何者為非？
> 「**FNAC最適用於MR-guided之FNAC**」為非——FNAC最適合超音波導引。經驗豐富cytopathologist判讀時FNAC準確度高、FNAC敏感度低於core needle biopsy（14G取三條準確度95% vs一條70%）皆為正確敘述。[^1]
