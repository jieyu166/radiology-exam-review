# gi-infection-location-pattern

**腸胃道感染依病原體有特徵性的解剖分布：結核、typhlitis、阿米巴以『迴結腸(ileocolonic，遠端迴腸+盲腸)』為主;但Giardia(梨形鞭毛蟲)主要侵犯『近端小腸』，並非以迴結腸為主。** 判讀分水嶺：**近端小腸寄生蟲(Giardia、Strongyloides)vs遠端小腸細菌(Salmonella、Shigella、Yersinia);迴結腸為主(TB、typhlitis、amoebiasis)vs結腸特定節段(Yersinia/Salmonella好發升結腸、Shigella好發降結腸);全結腸炎(pancolitis)可見於C.difficile、CMV、E.coli;C.difficile的CT特徵為accordion sign。**

## Summary
- **迴結腸為主（ileocolonic-predominant）疾病**：**結核(tuberculosis)、typhlitis（盲腸炎）、阿米巴病(amebiasis)**最常侵犯**遠端迴腸與盲腸**。[^1]
- **右側結腸(right colon)常見致病菌**：Salmonella、Shigella、**Campylobacter**、Yersinia enterocolitica。[^1]
- **左側結腸與直乙狀結腸(left colon/rectosigmoid)**：Schistosomiasis（左結腸）;HSV、Gonorrhea（直乙狀結腸）。[^1]
- **小腸感染的近端 vs 遠端分布**：**寄生蟲性腸炎（Giardia、Strongyloides）最常侵犯近端小腸**;**細菌性（Salmonella、Shigella、Yersinia）則侵犯遠端小腸**。[^1]
- **結腸內的細菌分布**：Yersinia與Salmonella好發**升結腸**;Shigella好發**降結腸**。[^1]
- **全結腸炎（pancolitis）**：可見於**Clostridium difficile、cytomegalovirus、Escherichia coli**;**C. difficile 的CT相對特異徵象為accordion sign（手風琴徵）**。[^1]

> [!note] 考點：何者「非」以迴結腸(ileocolonic)為主要侵犯部位的疾病?
> **Giardia（D）**——梨形鞭毛蟲主要侵犯**近端小腸**，非迴結腸為主。Tuberculosis、Crohn disease、typhlitis 皆好發迴結腸區域（Campylobacter則好發右側結腸,亦非典型迴結腸描述但非本題重點）。[^1]
