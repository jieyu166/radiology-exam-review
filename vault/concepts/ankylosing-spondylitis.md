# ankylosing-spondylitis

**AS 是「由下向上、由後向前」融合的脊椎病——對稱薦腸關節炎先，細直 syndesmophytes 後，最終 bamboo spine。** 男性為主、HLA-B27 相關；椎體方形化（squaring）與 shiny corner（Romanus 病灶）是早期特徵；enthesitis 為全身表現。考試分水嶺：**arthritis mutilans（pencil-in-cup）屬乾癬性關節炎，不是 AS**——這幾乎是每次必考的「非 AS 特徵」。

## Summary
- **薦腸關節炎**：通常**對稱**、為早期表現。[^1]
- **Syndesmophytes**：典型為**細、垂直（縱向）**的椎旁骨贅 → 廣泛融合呈 **bamboo spine（竹節狀脊椎）**。[^1]
- **椎體方形化（squaring）**、**shiny corner sign（Romanus 病灶）**、**Andersson 病灶（非感染性脊椎椎間盤炎）**、**dagger sign**。[^1]
- **Enthesopathy / enthesitis**：肌腱韌帶附著點發炎。[^1]
- **流行病學**：**男性為主**、年輕成人、**HLA-B27** 相關。[^1]
- **Arthritis mutilans 不是 AS 特徵**：屬**乾癬性關節炎**（及類風濕）之嚴重破壞性（如 pencil-in-cup）→ 故為題目「非 AS 特徵」之正解（2016-107 正解 C）。[^1][^2]

> [!note] 考點：何者「非」AS 的影像特徵？
> 「**Arthritis mutilans**」非 AS——是**乾癬性關節炎**的破壞性表現。AS 特徵＝對稱薦腸關節炎、（細直）syndesmophytes、enthesopathy、男性為主。[^1][^2]
