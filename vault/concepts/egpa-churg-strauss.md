# egpa-churg-strauss

**EGPA 的三聯是氣喘＋嗜酸性球增多＋壞死性血管炎，ANCA 陽性時記得是 pANCA（MPO），不是 cANCA。** cANCA（PR3）屬於 GPA/Wegener，兩者在考題常互換作陷阱。記憶鉤：**「EGPA 有氣喘和好多嗜酸球（p 型），GPA 靠 c 型抗體」**——p vs c 對應 EGPA vs GPA，背起來就不混淆。

## Summary
- **三聯**：**氣喘、周邊/組織嗜酸性球增多、壞死性血管炎**(常伴肉芽腫)。[^1]
- **ANCA**：約 40% 陽性,典型為 **pANCA(MPO-ANCA)**;**cANCA(PR3) 屬 GPA/Wegener**,非 EGPA。[^1]
- **肺影像**：游走性/周邊性 consolidation 或 GGO、小葉中心結節;可累及心臟、周邊神經、鼻竇。[^1]

> [!note] 考點：Churg-Strauss 的特徵「除外」何者?
> **Serum cANCA positive**——EGPA 為 **pANCA(MPO)**;cANCA(PR3) 是 GPA/Wegener。asthma、hypereosinophilia、necrotizing vasculitis 皆為 EGPA 特徵。[^1]

### 參考來源
