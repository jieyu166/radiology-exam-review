# carotid-cavernous-fistula

**頸動脈海綿竇瘻管（CCF）動脈化血流常見流出路徑包括下岩靜脈竇、淺蝶顳（sylvian）皮質靜脈與翼狀靜脈叢；丘紋靜脈（thalamostriate vein）並非CCF常見的流出路徑。**

## Summary
- **常見流出路徑**：inferior petrosal sinus（下岩靜脈竇）、superficial sylvian cortical veins（淺蝶顳皮質靜脈）、pterygoid venous plexus（翼狀靜脈叢），另可經 superior ophthalmic vein 等海綿竇周邊靜脈系統。[^1]
- **非常見路徑**：**Thalamostriate vein（丘紋靜脈）並非CCF動脈化血流常見流出路徑**——丘紋靜脈屬深部靜脈系統，與海綿竇無直接常見連通。[^1]
- **外傷性CCF血流速率（2019交換）**：外傷性/直接型（Barrow type A）CCF是頸內動脈與海綿竇之間**直接**相通，通常呈**高流速（high-flow）**廔管，並非低流速（slow-flow）——低流速廔管較常見於間接型（dural/indirect，Barrow type B-D）CCF。臨床上外傷或動脈瘤破裂是直接型CCF最常見病因。[^2]

## 陷阱（外傷性CCF）
- 「Slow flow fistula between cavernous ICA and sinus」為錯誤敘述——外傷性/直接型CCF為**高流速**廔管。[^2]

### 參考來源
[^1]: 2019 交換考詳解；ASNR Core Curriculum Course 1994, p.49。
[^2]: 2019 交換考詳解；Diagnostic Imaging, Brain, P. I-2-63：直接型（Barrow type A）CCF為頸內動脈與海綿竇直接相通之高流速廔管，外傷/動脈瘤破裂為最常見病因；enlarged extra-ocular muscles、皮質靜脈填充為危險徵象。
