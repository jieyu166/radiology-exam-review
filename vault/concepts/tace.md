# tace

**TACE（經動脈化療栓塞）是 BCLC 中期 HCC 首選；其禁忌/風險的核心在『肝功能儲備與門脈血流』——主門脈阻塞、高膽紅素、TIPS（門脈血流被分流）皆增加肝衰竭風險；膽道異常增加肝膿瘍風險。** 判讀分水嶺一：**TIPS 存在 → TACE 後肝衰竭風險『高』（非低）**；判讀分水嶺二：**膽道擴張/乳頭切開/膽腸吻合 → 肝膿瘍風險升高**。

## Summary
- **主門脈阻塞**：傳統 TACE 之（相對）禁忌——栓塞肝動脈時失去雙重血供致肝梗塞/衰竭。[^1]
- **血清膽紅素 >3.0 mg/dL**：肝功能不足，屬禁忌（肝衰竭風險）。[^1]
- **TIPS 存在 → 肝衰竭風險『高』**（門脈血流經分流分走，栓塞肝動脈後肝灌流更不足）；故「TIPS 與低風險相關」**錯誤**＝題目正解。[^1]
- **膽道擴張 / 內視鏡乳頭切開史 / 膽腸吻合**：膽道與腸道相通、細菌易上行 → **肝膿瘍風險升高**（建議預防性抗生素）。[^1]

> [!note] 考點：何者敘述「正確/錯誤」？
> 「**TIPS 與 TACE 後低肝衰竭風險相關**」**錯**——TIPS 反而**增加**肝衰竭風險。主門脈阻塞、膽紅素>3 為禁忌；膽道異常增肝膿瘍風險。[^1]
