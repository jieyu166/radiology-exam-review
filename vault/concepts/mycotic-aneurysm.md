# mycotic-aneurysm

**感染性（mycotic）動脈瘤是病原體感染破壞管壁形成的偽動脈瘤（pseudoaneurysm），影像關鍵不在「形狀」而在「演變」：囊狀/分葉、偏心、好發非典型部位、且短期內快速增大或新生。** 主動脈段判讀分水嶺：**囊狀/偏心瘤＋鈣化環中斷＋主動脈旁發炎/軟組織/積氣＋快速增大**；顱內段判讀分水嶺：**遠端分支（distal MCA/PCA）的不規則/梭狀瘤、常多發、可在數天到數週內新生或快速擴大**。記憶鉤：退化性真性動脈瘤＝鈣化環完整、緩慢、好發腎下主動脈近端大血管；mycotic＝管壁「被吃掉」、快速、好發非典型遠端位置。

## Summary

### 定義與機轉
- **本質為偽動脈瘤**：感染破壞動脈壁全層，由周邊組織包裹而成的 contained rupture，而非真性動脈瘤的管壁三層擴張。[^1][^2]
- **病因**：細菌（Salmonella、*Staphylococcus aureus*、Clostridium 等）經菌血症血行播散種植於內皮；亦可鄰近感染蔓延、心內膜炎敗血性栓子；少數黴菌/分枝桿菌。**梅毒性主動脈炎亦屬感染性主動脈病變之一**（故「全部都是非梅毒感染所致」為錯誤敘述）。[^1][^3]

### 主動脈段（CTA 為首選）
- **CT/CTA 為首選模態**：快速、高解析、容積對比影像；可顯示囊狀/分葉/偏心瘤體、**主動脈壁鈣化環中斷（管壁被侵蝕）**、**主動脈旁軟組織增厚/積液/積氣**及鄰近膿瘍。[^1][^2]
- **判讀重點徵象**：囊狀或偏心（saccular/lobulated）瘤、缺乏退化性瘤常見的環狀鈣化、瘤周發炎反應；可併血栓、遠端栓塞。[^2]
- **動態變化是最重要陷阱**：感染瘤**短期快速增大、易破裂**，回顧前次影像若見短時間內瘤體擴大或新生，強烈支持感染性而非退化性。[^1][^2]
- **FDG-PET/CT 為功能性輔助**：瘤體 FDG 高攝取（攝取高於肝臟背景）對感染瘤敏感度高，但**特異度低**——發炎性動脈瘤（inflammatory aneurysm）等可偽陽性，須結合臨床、實驗室與微生物學判讀。[^4]
- **好發部位**：胸主動脈與腎上（suprarenal）主動脈等非典型位置，與退化性瘤好發腎下主動脈不同。[^1]

### 顱內段（infectious / mycotic intracranial aneurysm）
- **流行病學**：占所有顱內動脈瘤約 0.7–5.4%；見於約 2–10% 感染性心內膜炎（IE）病人，多為敗血性栓子所致。[^5]
- **位置與形態**：**好發遠端分支（secondary/tertiary branches）**，以**遠端 MCA、PCA**為主；形態**不規則或梭狀（fusiform）**、亦可囊狀，**常多發**；多數體積小，但可快速擴大甚至成 giant。[^5]
- **時間窗與動態陷阱**：IE 發病後約 **2–5 週**為破裂高峰；瘤可在 CTA/MRA/DSA **首次檢查陰性後數天內新生或快速長大並破裂**——故 IE 病人須**連續神經影像追蹤**。[^5]
- **DSA 為金標準**：即使 CTA/MRA 等非侵入檢查陰性，仍建議 DSA 確認。[^5]
- **MRI 輔助偵測未破裂瘤**：T2*/GRE 之 **susceptibility vessel sign**（敏感度約 88%）配合 FLAIR（管腔/鄰近皮質高訊號），兩者合用偵測未破裂感染性顱內瘤的敏感度與特異度可達 100%。[^6]

### 鑑別（DDx）
- **退化性（atherosclerotic）真性動脈瘤**：fusiform、管壁**環狀鈣化完整**、緩慢增大、好發腎下主動脈/近端大血管；感染瘤反之為囊狀偏心、鈣化中斷、快速變化、好發非典型遠端部位。[^1][^2]
- **發炎性動脈瘤（inflammatory aneurysm）**：瘤周亦可發炎/FDG 攝取，是 PET 偽陽性主因，須靠臨床與微生物學區分。[^4]
- **顱內：典型囊狀（berry）動脈瘤**：好發 Willis 環近端分叉處、單發、不會數天內新生；感染性則遠端、多發、快速變化。[^5]

> [!note] 考點:何者「錯」?
> 「**全部都是非梅毒感染所致**」錯——梅毒性主動脈瘤亦屬感染性。好發胸/腎上主動脈、鈣化環中斷、主動脈旁發炎/積氣皆正確。
