# prune-belly-syndrome

**梅干腹（prune belly / Eagle-Barrett / triad）症候群三聯＝①前腹壁肌發育不良（腹直肌分離,「梅干」外觀）②泌尿道異常（腎盂輸尿管擴張+腎發育不良,hydroureteronephrosis）③雙側隱睪（cryptorchidism）。『脊柱裂（spina bifida）』不在三聯內。** 判讀分水嶺一：**腹壁＋泌尿＋隱睪三聯**;判讀分水嶺二：**幾乎只見於男嬰**。

## Summary
- **三聯**：[^1]
  - **前腹壁肌發育不良**（腹直肌廣泛分離,「prune belly」外觀）(option A 對)
  - **泌尿道異常**：**腎盂輸尿管擴張＋腎發育不良（hydroureteronephrosis）**(option C 對)
  - **雙側隱睪（cryptorchidism）**(option D 對)
- **不屬三聯**：**spina bifida（脊柱裂）** 不在三聯內 → 故為「不屬三聯」之正解（2016-188 正解 B）。[^1]
- 幾乎全為男嬰;可合併膀胱擴張、肺發育不良（羊水過少）、骨骼/心臟異常。[^1]

> [!note] 考點：何者「不」屬 prune belly 三聯?
> **Spina bifida（脊柱裂）**。三聯＝腹壁肌發育不良、hydroureteronephrosis、隱睪。[^1]
