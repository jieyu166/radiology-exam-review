# renal-artery-stenosis

**腎動脈狹窄 Doppler 的因果鏈：狹窄→局部 PSV ↑（>180 cm/s）→下游水流緩慢（tardus-parvus，AT >0.07 s）；但萎縮小腎（<8 cm）＋高實質 RI（>0.8）代表末期腎硬化，修通血管也無效，即使看到狹窄也不介入。** 判讀分水嶺一：**PSV >180–200 cm/s、RAR >3.5、AT >0.07 s → 血流動力學顯著狹窄**；判讀分水嶺二：**小萎縮腎＋高 RI → 腎實質已纖維化，血管重建無效 → 不介入**。

## Summary

### 病因型態（判讀第一步：定位狹窄在哪一段）
- **動脈粥狀硬化（atherosclerotic，約占 RAS 90%）**：好發**近端／開口（ostial/proximal）**，常為主動脈斑塊延伸至腎動脈開口；典型為年長、有心血管危險因子病人，常見鈣化斑塊。[^2][^3]
- **纖維肌肉發育不良（FMD, fibromuscular dysplasia，約占 10–20%）**：好發**中遠段（mid-to-distal）**，**很少侵犯開口／近端**（與粥狀硬化的鑑別重點）；典型為**年輕女性**；medial fibroplasia 型在血管攝影／CTA／MRA 呈經典 **string of beads（串珠狀）** ——交替的狹窄與小動脈瘤樣擴張。[^2][^3]

### 都卜勒超音波判讀
- **狹窄直接徵象**：狹窄處 **PSV >180–200 cm/s**（>200 cm/s 對 ≥60% 狹窄較具特異性）、**renal-aortic ratio (RAR) ≥3.5**（預測 ≥60% 狹窄，敏感度約 84–91%、特異度約 95–97%）、狹窄後 **turbulence/aliasing**；end-diastolic velocity >150 cm/s 提示 >80% 狹窄。[^1][^2]
- **下游間接徵象（腎內葉間動脈）**：**tardus-parvus 波形**（收縮上升延遲且低平、early systolic peak 消失）、**acceleration time (AT) >0.07 s（>70 ms）**、acceleration index <3 m/s²。[^1][^2]
- **resistive index (RI)**：正常腎門 RI 約 0.65、葉間動脈約 0.54；**RI ≥0.7 為病理閾值**，**RI >0.80 預示血管重建後反應不佳**（腎實質已纖維化）。[^1]
- **RAR 限制**：當主動脈 prerenal PSV <40 cm/s 時 RAR 不可靠。[^1]

### CTA / MRA
- **CTA**：直接顯示管腔狹窄、開口處鈣化斑塊、**狹窄後擴張（post-stenotic dilation）**；FMD 呈 string of beads。[^2][^3]
- **MRA（contrast-enhanced）**：對偵測 FMD 的 string of beads 外觀敏感度高（約 97%），但對「>50% 狹窄」量化的敏感度較低（約 68%）；對 ≥50% 粥狀硬化狹窄 gadolinium-enhanced MRA 敏感度可達約 96%。可同時做腎實質灌流的功能性評估。[^2][^3]

### 功能性評估
- **captopril renography（卡托普利腎圖）**：用於評估狹窄之功能意義與**預測血管重建是否有效**；對 50–99% 狹窄敏感度約 85%、特異度約 71%。[^4]

### 介入適應與排除
- **小而萎縮的腎（雙側 5–6 cm，<8 cm）＋ 實質 RI 高（~0.9，>0.8）** → 提示**末期腎硬化／腎實質病變**,血管重建難改善 → **不介入**。[^1]
- 題例 PSV 130／160 cm/s **未達 >180 狹窄閾值**（非血流動力學顯著狹窄）,加上雙側小腎、RI 0.9 → **無介入適應症**。[^1]

### DDx 與判讀陷阱
- **副腎動脈漏看（accessory renal artery）**：約 20–30% 病人有副腎動脈，其顯著狹窄是隱性腎血管性高血壓來源；主腎動脈直徑 ≤4.15 mm 提示可能存在副腎動脈，應主動往下找。[^1]
- **tardus-parvus 偽陰／偽陽**：tardus-parvus 出現有助診斷，但**其不出現不能排除 RAS**；粥狀硬化致血管順應性下降時，parvus-tardus 波形可能變得不明顯（偽陰）；近端主動脈病變或心臟瓣膜疾病亦可造成全腎波形低平（偽陽，非單側狹窄）。[^1]
- **技術因素**：腸氣、肥胖、深位血管、呼吸動作致主腎動脈直接顯像率僅約 84–90%；完全阻塞時血流消失易誤判為無法顯像。[^1]
- **gadolinium MRA 的 NSF 顧慮**：腎功能不良者使用 gadolinium 有腎源性全身性纖維化（NSF）風險，**GFR ≤30 mL/min/1.73 m² 不建議施打 gadolinium MRA**。[^2][^4]
- **stent 後（in-stent restenosis）評估**：支架使血管硬度改變，需提高速度閾值——in-stent **PSV >180 cm/s** 敏感度高但特異度低（約 74%），**PSV >226 cm/s** 特異度較佳（約 90%）；可併用 RAR >2.7。[^1]

### RAS 病因辨異（考點）
- **最常見 RAS 病因＝動脈粥狀硬化（~75%，部分文獻達 90%）**,非 FMD。[^3][^5]
- **FMD 亞型**：**medial fibroplasia（中膜型）為最常見,約占 FMD 90%**,呈 string of beads;其餘為 intimal/perimedial/adventitial 等少見型。[^3][^5]
- **Takayasu aortitis**：**女性為主（F:M ≈ 9:1）、好發 <50 歲（常 <40）**;為大血管炎,可累及主動脈及其分支致腎動脈開口狹窄（非男性、非 >35 歲為主）。[^5]
- **Middle aortic syndrome（中主動脈症候群）**：為**獨立罕見病種**,好發兒童／青少年,特徵為**腹主動脈嚴重狹窄合併腎及內臟分支進行性受累**;**並非 Takayasu 的亞型**。[^5]

> [!note] 考點：雙側小腎（5–6 cm）、RI 0.9、PSV 130／160 cm/s,該如何處置?
> **不介入（No intervention）**。PSV 未達狹窄閾值（>180）、且雙側萎縮腎＋高 RI（腎硬化）→ 血管重建無益。[^1]

> [!note] 考點：關於 RAS／FMD／Takayasu,何者正確?
> 「**Medial dysplasia 為最常見 FMD 型（~90%）**」正確。最常見 RAS 病因為粥狀硬化（非 FMD）、Takayasu 為女性為主＜50 歲（非男性＞35）、midaortic syndrome 為獨立病種（非 Takayasu 亞型）。[^3][^5]
