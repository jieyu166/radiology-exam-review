# aortic-arch-anomalies

**主動脈弓變異最重要的兩條規則：「最常見」不代表「最會造成症狀」；「右弓鏡像」才是先天心臟病的強烈暗示。** 最常見的變異是左弓＋迷走右鎖骨下動脈，從食道後方偷偷繞過去，多數人終身無症狀（偶爾吞嚥困難 dysphagia lusoria）；考試愛把「常造成壓迫」設為錯誤選項。次常見的右弓鏡像分枝型則高度關聯 DiGeorge 症候群、TOF 等 conotruncal 心臟病——看到右弓就要想到先心篩查。判讀的核心就三件事：**分支型態（哪一條最後出來、有無 retroesophageal 走行）、氣管／食道有無壓迫、有沒有形成完整血管環（complete vascular ring）。**

## Summary
- **左弓＋迷走右鎖骨下動脈（aberrant right SCA, arteria lusoria）**：**最常見**的弓變異（盛行率約 0.5–2%）；分支順序為右總頸→左總頸→左鎖骨下→**最後**才是迷走右鎖骨下動脈，走**食道後方（retroesophageal，>80%）**；**多無症狀**，故「常造成壓迫症狀」**錯誤**（2016-100 正解 A）。[^1][^2]
- **Kommerell 憩室**：迷走鎖骨下動脈起始處的囊狀膨出（第四對背側主動脈弓殘餘），可壓迫**食道（後方）或氣管**，是壓迫症狀與動脈瘤／剝離風險的來源。[^1][^3]
- **Bovine arch（牛弓）**：**最常見的分支變異**（非真正異常），盛行率約 7–35%（非裔／南美裔較高）；無名動脈與左總頸動脈**共同起源**（Type I 共幹、Type II 左總頸自無名動脈近端發出）；多無臨床意義，但介入或主動脈手術前須認知。[^4]
- **Double aortic arch（雙主動脈弓）**：**最常見、最完整的血管環**；CTA 見**兩側各自獨立發出一支頸動脈＋一支鎖骨下動脈（four-vessel sign）**，環繞並壓迫**氣管與食道**，嬰幼兒常以**喘鳴（stridor）或吞嚥困難**表現；部分後段（多為左弓）為閉鎖節（atretic），但仍維持血管環。[^2][^5]
- **右弓鏡像分枝（right arch, mirror-image）**：**第二常見**的右弓型；分支為鏡像對稱、**不走食道後方故不形成血管環**；與**先天性心臟病高度相關（約 98%，以 TOF 為主，亦見 truncus、DiGeorge 等 conotruncal 異常）**——看到此型要主動篩先心。[^2][^6]
- **右弓＋迷走左鎖骨下動脈（aberrant left SCA）**：右弓中較常見者；分支順序為左總頸→右總頸→右鎖骨下→**最後**迷走左鎖骨下動脈走**食道後方**，起始處常見 **Kommerell 憩室**；憩室＋動脈韌帶（ligamentum arteriosum）可形成**完整血管環**而壓迫氣管食道；**較少合併先心**。[^2][^3][^6]
- **Cervical aortic arch（頸位主動脈弓）**：罕見；弓位異常上升**至鎖骨上方（約 C6/C7）**，臨床可表現為**鎖骨上搏動性腫塊**；CT/MR 見高位弓、降主動脈／弓結位置異常、氣管或食道後方壓痕。[^7]
- **Coarctation（主動脈窄縮）**：弓遠端／降主動脈交界（峽部）狹窄；CTA／MRA 見**升主動脈與降主動脈口徑落差、內乳動脈與肋間動脈側枝擴大**（肋骨下緣 rib notching 為其 X 光對應徵象）。[^2]

## 影像判讀骨架（CTA／MRA 為主）
- **第一步看分支數與順序**：正常左弓三分支（無名→左總頸→左鎖骨下）。出現**第四分支且走食道後方** → 想迷走鎖骨下動脈；**四血管徵（four-vessel sign）** → 雙主動脈弓；**頸動脈共幹** → bovine arch。[^2][^4]
- **第二步看弓的側別與走行**：氣管偏移方向、弓跨在主支氣管哪一側可提示左／右弓；弓上升至鎖骨上 → cervical arch。[^2][^7]
- **第三步評估氣管／食道壓迫**：橫斷面看 retroesophageal 結構對食道後壁的壓痕、氣管管徑與偏移；嬰幼兒喘鳴／反覆呼吸道症狀要特別找血管環。[^2][^5]
- **第四步判定有無完整血管環**：雙主動脈弓、右弓＋迷走左鎖骨下＋Kommerell 憩室＋動脈韌帶 → 多形成完整環；單純左弓＋迷走右鎖骨下動脈**通常不構成完整環**。[^2][^3][^5]
- **臨床關聯連動**：右弓鏡像 → 先心（TOF 等）篩查；Kommerell 憩室 → 評估憩室大小與動脈瘤／剝離風險；coarctation → 找側枝與合併二葉式主動脈瓣。[^2][^6]

> [!note] 考點：何者「為假」？
> 「左弓＋迷走右 SCA 最常見且**常造成壓迫症狀**」假——最常見但**多無症狀**。Kommerell 壓食道、右弓鏡像次常見且關聯 DiGeorge／TOF 皆正確。[^1] 延伸辨別：**血管環**主要由雙主動脈弓、右弓＋迷走左 SCA（＋Kommerell＋動脈韌帶）形成；**先心關聯**最強者為右弓鏡像型。[^2][^6]

## 題目
> [!question]- Which statement of right arch with aberrant left subclavian artery is wrong? (2018-063)
> **The first branch of aorta is left subclavian artery（A）**——右弓+迷走左鎖骨下動脈之分支順序為**左總頸動脈(left CCA)→右總頸→右鎖骨下→最後迷走左鎖骨下動脈**，故第一分支為left CCA非left subclavian。約佔右弓39.5%(B對)、常伴Kommerell憩室(C對)、較少合併其他心血管異常(D對)皆為正確描述。[^2]
