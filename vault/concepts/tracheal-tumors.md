# tracheal-tumors

**成人的氣管腫瘤多半是惡性的；良性最常見是 squamous cell papilloma（孤立性），惡性最常見是 squamous cell carcinoma 與 adenoid cystic carcinoma。** 記憶鉤：良性→多發/平滑/廣泛；惡性→單一/不規則/穿壁。Adenoid cystic 特點是「沿黏膜下潛行」，影像上範圍常比外觀更廣。判讀分水嶺一：**成人單一不規則氣管腔內腫塊 + 管壁外侵犯 → 惡性**；判讀分水嶺二：**多發 polypoid 結節（尤其兒童） → 先想 papillomatosis（HPV）**。

## Summary
- **最常見良性氣管腫瘤 = squamous cell papilloma**（孤立性、成人）。[^3]
- **Papillomatosis**：多發 papilloma，與 HPV（recurrent respiratory papillomatosis）相關、好發兒童；CT 多發 polypoid 結節，可向下播散至肺。[^1]
- **Hamartoma**：含脂肪/軟骨之良性腫瘤。[^1]
- **最常見原發惡性 = squamous cell carcinoma + adenoid cystic carcinoma**；SCC 與抽菸相關、好發 >50 歲。[^2]
- **良性 vs 惡性線索**：廣泛平滑/結節增厚或多發結節→良性；單一不規則、管壁外侵犯、淋巴結→惡性。[^2]

## 良性
- **Squamous cell papilloma**：**最常見良性氣管腫瘤**（孤立性）。[^3][^1]
- **Tracheobronchial papillomatosis**：多發（HPV/RRP）、兒童常見；可肺內播散（空洞結節）。[^1]
- **Hamartoma**：含脂肪/軟骨（CT 可見脂肪/popcorn 鈣化）。[^1]
- 其他間葉性（leiomyoma、chondroma、lipoma 等）少見。[^1]

## 惡性
- **Squamous cell carcinoma**：最常見原發氣管惡性之一；抽菸相關、>50 歲。[^2]
- **Adenoid cystic carcinoma**：另一最常見原發氣管惡性；沿黏膜下/神經周圍浸潤。[^2]

> [!note] 後壁膜「累及 vs 保留」鑑別（瀰漫氣道病）
> **保留氣管後壁膜**：tracheobronchopathia osteochondroplastica（TBO）、relapsing polychondritis。**累及後壁膜**：amyloidosis（見 [[pulmonary-amyloidosis]]）。[^2]

## 鑑別與報告要點
| 類別 | 代表 | 重點 |
| --- | --- | --- |
| 良性（最常見）| **Squamous cell papilloma** | 孤立、成人[^3] |
| 良性（多發）| Papillomatosis（HPV）| 多發、兒童、可肺播散[^1] |
| 良性（含脂/軟骨）| Hamartoma | 脂肪/popcorn 鈣化[^1] |
