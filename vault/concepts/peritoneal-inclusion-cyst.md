# peritoneal-inclusion-cyst

**腹膜包涵囊腫＝非腫瘤性反應性間皮增生,好發『停經前(premenopausal)』有骨盆手術/PID/子宮內膜異位史的婦女。** 判讀分水嶺：**停經前婦女、有腹膜沾黏因子(手術/外傷/PID/子宮內膜異位);卵巢被包繞於液體中、卵巢正常;『spider web』分隔外觀;無強化實質、無惡性潛能**。

## Summary
- **本質**：**非腫瘤性反應性間皮增生**;由腹膜沾黏陷住卵巢分泌的液體。[^1]
- **族群**：**停經前(premenopausal)婦女**,有骨盆手術、外傷、PID 或子宮內膜異位史。[^1]
- **影像**：圍繞 GYN 器官的清澈液體、卵巢正常;**spider web appearance**（分隔）;**無強化實質、無惡性潛能**。[^1]

> [!note] 考點：關於 peritoneal inclusion cyst,何者「錯誤」?
> 「**最常見於停經後(postmenopausal)婦女**」錯誤——好發**停經前(premenopausal)**有骨盆手術/PID/內膜異位史者。spider web 外觀(B)、無惡性潛能(C)、無強化實質(D)皆正確。[^1]

### 參考來源
