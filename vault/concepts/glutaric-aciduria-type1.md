# glutaric-aciduria-type1

**第一型戊二酸尿症（Glutaric Aciduria Type 1, GA1）為體染色體隱性遺傳之有機酸代謝疾病，典型影像特徵包含寬大之外側裂（wide operculae）、基底節於某些序列呈現亮訊號（bright basal ganglia）、早期巨頭畸形與晚期腦萎縮；嚴重個案常合併髓鞘化延遲（delayed myelination），並非正常髓鞘化。**

## Summary
- **遺傳模式**：**體染色體隱性遺傳（autosomal recessive）**。[^1]
- **影像特徵**：**外側裂寬大（wide operculae）**、**基底節亮訊號（bright basal ganglia）**。[^1]
- **病程**：**早期巨頭畸形（early macrocephaly）、晚期腦萎縮（late atrophy）**。[^1]
- **髓鞘化**：**嚴重受影響之嬰兒常合併髓鞘化延遲（delay in myelination）**，並非「正常髓鞘化」——此為考題易混淆之錯誤敘述。[^1]
- **鑑別診斷（BESS，良性蛛網膜下腔擴大）**：BESS雖常見凸面（convexity）蛛網膜下腔明顯擴大，但**不會**合併外側裂（sylvian fissure）或前顳極（anterior temporal poles）之CSF空間明顯擴大，且**無實質病變**——可作為與GA1鑑別之參考。[^1]

## 陷阱
- 「Normal myelination（正常髓鞘化）」為錯誤敘述——嚴重個案應呈**髓鞘化延遲**。
