# peripheral-avm-classification

**Cho-Do classification 依據 nidus 分 4 型，Type II（多條 arteriole 匯入單一 outflow vein）最佳栓塞方式為 transvenous approach 或 direct puncture（非 transarterial）。**

## Summary
- Type I：≤3 feeders + 1 outflow vein → transarterial [^1]
- Type II：多條 arterioles + single outflow vein → **transvenous 或 direct puncture** [^1]
- Type IIIa：多支 feeder + multiple non-dilated shunts → direct puncture + transarterial [^1]
- Type IIIb：多支 feeder + multiple dilated shunts → direct puncture + transarterial [^1]

### 參考來源
[^1]: 2019 交換考詳解。
