# true-diverticulum

**True diverticulum（真憩室）包含腸壁所有層次（黏膜、黏膜下層、肌肉層），小腸唯一常見的 true diverticulum 是 Meckel diverticulum；其餘常見小腸憩室（十二指腸憩室、空腸憩室、迴腸憩室）均為 false diverticulum（假憩室），僅含黏膜與黏膜下層突出。**

## Summary
- **True diverticulum**：壁含所有層次（mucosa + submucosa + muscularis propria）[^1]
- **False (pseudo) diverticulum**：僅黏膜與黏膜下層經肌肉層薄弱處突出 [^1]
- **小腸**：**Meckel diverticulum 是唯一常見的 true diverticulum**，為卵黃管（omphalomesenteric duct）殘留 [^1]
- 十二指腸憩室（最常見小腸憩室）、空腸與迴腸憩室皆為 false diverticulum [^1]
- 大腸：結腸憩室幾乎全為 false diverticulum（除少見先天性 true diverticulum 外）[^1]
