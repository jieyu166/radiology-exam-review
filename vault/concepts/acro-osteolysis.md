# acro-osteolysis

**Acro-osteolysis（指/趾端骨溶解）常見於：scleroderma、psoriatic arthritis、hyperparathyroidism、thermal injury、PVC exposure、leprosy 等。NF1 不是典型原因。**

## Summary
- **Scleroderma**：tuftal resorption，常伴 soft tissue calcification [^1]
- **Psoriatic arthritis**：DIP 關節 pencil-in-cup，可合併 tuft erosion [^1]
- **Hyperparathyroidism**：subperiosteal resorption 可延伸至 tuft [^1]
- **不典型**：Neurofibromatosis type 1（NF1 主要造成長骨 bowing、pseudarthrosis、sphenoid dysplasia，非 acro-osteolysis） [^1]
