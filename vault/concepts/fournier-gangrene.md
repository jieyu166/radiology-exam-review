# fournier-gangrene

**Fournier gangrene 是會陰/陰囊的壞死性筋膜炎，感染源幾乎都找得到——最常是肛門直腸（廔管/膿瘍），其次泌尿生殖道或會陰創傷,「找不到原發感染源」是錯誤選項。** 判讀分水嶺：**CT 是判定範圍（extent）的必要工具**（找皮下氣體沿筋膜面擴散範圍,決定清創範圍）,不是用來找感染源。

## Summary
- **本質**：致命的多重菌感染性壞死性筋膜炎，好發於陰囊、會陰、肛周，可迅速沿筋膜面擴散至腹壁、大腿；死亡率高，需緊急外科清創。[^1]
- **好發族群**：**糖尿病**病人風險顯著升高，其餘危險因子含慢性酒精濫用、免疫抑制、肥胖、周邊血管疾病。[^1]
- **感染源多可被找到**：最常見為**肛門直腸來源**（perianal fistula 或 abscess），其次為泌尿生殖道感染或會陰部創傷；僅少數個案找不到明確原發灶。[^1]
- **CT 角色**：CT 為判定疾病**擴散範圍**（皮下氣體沿筋膜面分布、有無延伸至腹壁/腹膜後/大腿）之必要工具，據以規劃清創範圍，而非用於找感染源本身。[^1]

## 影像判讀骨架
- **CT/US 皮下氣體**：沿陰囊/會陰筋膜面呈線狀或簇狀氣體影，為壞死性筋膜炎特徵性所見，可早於臨床明顯變化出現。[^1]
- **軟組織腫脹與筋膜增厚**：皮下脂肪 stranding、筋膜面增厚與液體積聚，界定感染沿筋膜擴散的路徑。[^1]
- **範圍評估**：CT 追蹤氣體/液體是否延伸至腹壁、腹股溝、大腿內側或腹膜後，直接影響清創範圍決策。[^1]
