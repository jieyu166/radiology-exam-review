# dural-venous-sinus-thrombosis

**靜脈竇血栓的影像陷阱多：急性血栓 T2 低訊號和正常 flow void 長得一樣，容易漏診；顯影後才浮現 empty delta（強化硬膜包圍不強化的血栓）。** DWI 並非萬用——並非所有血栓都呈現擴散受限，「DWI 全看得到」是錯誤敘述。橫竇本身常發育不全，訊號缺乏時要與真血栓區分。記憶鉤：**「急性竇栓＝T2 低→像 flow void→靠顯影 empty delta 揭露」**。

## Summary
- **Empty delta sign（空三角）**：上矢狀竇後段於**顯影 CT/MR** 上,強化硬膜環繞中央無強化血栓 → 三角形充盈缺損(選項 A 真)。[^1]
- **急性血栓訊號**：**T1 等訊號、T2 低訊號**,可**模擬正常竇的 flow void**(易漏診)(選項 B 真);亞急性轉 T1 高訊號。[^1]
- **DWI**：血栓**『可能』(may)顯示擴散受限,並非全部血栓於 DWI 可見** → 故「**所有阻塞血栓都能在 DWI 上看到**」**錯誤**＝題目正解(2016-127 正解 C)。[^1]
- **橫竇 hypoplasia**：**常見發育不全/低流變異區**,可呈現缺流、**模擬血栓**(選項 D 真);須與真血栓區別(看 CTV/相位、骨溝大小)。[^1]

> [!note] 考點：何者「有誤」?
> 「**所有阻塞血管內的血栓都能在 DWI 上看到**」有誤——血栓 DWI 訊號多變,**並非全部可見**。empty delta、急性 T2 低訊號模擬 flow void、橫竇 hypoplasia 模擬血栓皆正確。[^1]
