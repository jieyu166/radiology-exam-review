# ccam

**CCAM/CPAM 是肺的錯構瘤樣畸形，終末呼吸單位腺瘤樣過度增生，多單側、僅侵犯單一肺葉。** Stocker 分型以囊的大小決定：Type I（最常見，70%）是大囊（2–10 cm），預後最好；Type II 囊 <2 cm、常伴其他畸形；Type III 是微囊（<5 mm），超音波看起來像實質腫塊。記憶鉤：「I 號大、II 號小、III 號小到看不到囊」——囊的尺寸由大到微，對應 I→III。

## Summary
- **本質**：肺的**錯構瘤樣畸形(hamartomatous)**,終末呼吸單位腺瘤樣過度增生(選項 A 真)。[^1]
- **位置**：**多單側、侵犯單一肺葉**(選項 B 真);無明確肺葉偏好。[^1]
- **Stocker 分型(依囊大小)**：[^1]
  - **Type I（最常見,70%）**：**大囊,優勢囊 2–10 cm**,預後佳。
  - **Type II（15–20%）**：囊 **<2 cm**,常合併其他異常(腎發育不全、肺隔離症、先心)。
  - **Type III（~10%）**：**微囊 <5 mm**,**超音波呈實質(solid)樣**(microcystic)(選項 C 真)。
  - Type 0（罕、致死)、Type 4。
- **題目**：選項 D「**Type I 囊通常 <2 cm**、預後佳」**錯誤**——type I 為**大囊(2–10 cm)**,不是<2cm(<2cm 是 type II)（2016-130 正解 D）。[^1]

> [!note] 考點：何者「為錯」?
> 「**Type I CCAM 囊通常 <2 cm**」錯——type I 為**大囊(2–10 cm)、最常見、預後佳**(<2cm 是 type II)。錯構瘤樣、多單側單葉、microcystic(III)呈實質樣皆對。[^1]
