# pterygopalatine-fossa

**翼腭窩（PPF）是頭頸周神經/感染擴散的十字路口,連通：inferior orbital fissure（眼眶）、foramen rotundum（中顱窩,V2）、vidian canal（破裂孔/中顱窩）、sphenopalatine foramen（鼻腔）、pterygomaxillary fissure（顳下窩）、palatine canals（口腔）。『foramen ovale』通 V3 至咀嚼肌間隙,『不』與 PPF 相連。** 判讀分水嶺一：**PPF 連 IOF/rotundum/vidian/sphenopalatine**;判讀分水嶺二：**foramen ovale（V3）通咀嚼肌間隙,非 PPF**。

## Summary
- **PPF 連通**：[^1]
  - **inferior orbital fissure** → 眼眶(B)
  - **foramen rotundum** → 中顱窩（V2）(D)
  - **vidian（pterygoid）canal** → 破裂孔/中顱窩(C)
  - **sphenopalatine foramen** → 鼻腔
  - pterygomaxillary fissure → 顳下窩;greater/lesser palatine canals → 口腔
- **不與 PPF 相連**：**foramen ovale**（傳 V3,連中顱窩↔咀嚼肌間隙/顳下窩）**不直接與 PPF 相連**＝題目正解（2016-183 正解 A）。[^1]
- 臨床：周神經腫瘤擴散（如 adenoid cystic carcinoma 沿 V2/vidian）、juvenile angiofibroma 侵犯。[^1]

> [!note] 考點：何者「未」與翼腭窩相連?
> **Foramen ovale**（傳 V3,通咀嚼肌間隙）。inferior orbital fissure、foramen rotundum、vidian canal、sphenopalatine foramen 皆與 PPF 相連。[^1]
