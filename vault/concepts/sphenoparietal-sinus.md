# sphenoparietal-sinus

**蝶頂竇沿蝶骨小翼後下緣走行、接收 superficial middle cerebral vein（Sylvian vein），有三種典型『深部引流』型；『向上至 MCV 再入上矢狀竇』不是其中之一。** 判讀分水嶺一：**屬支是 superficial MCV（流向竇內），不是竇向上流入 SSS**；判讀分水嶺二：三型＝海綿竇、翼叢（basal emissary）、沿中顱窩底向後越岩錐入岩下/橫竇。

## Summary
- **解剖**：沿蝶骨小翼後下緣；接收 **superficial middle cerebral vein（Sylvian vein）**、middle meningeal vein（frontal ramus）、anterior temporal diploic vein；典型引流入**海綿竇**。[^1]
- **三種典型（深部）引流型**（依題目來源 Takahashi）：[^2]
  1. 入**海綿竇**（最常見、前內側）。
  2. 經 basal emissary veins 入**翼叢（pterygoid plexus）**（sphenobasal 型）。
  3. **沿中顱窩底向後、越岩錐**入**inferior petrosal vein/sinus 或 transverse sinus**（sphenopetrosal 型）。
- **非屬上述三型（題目 EXCEPT）**：「直接向上至 middle cerebral vein 再入上矢狀竇」——方向相反（MCV 是流『入』蝶頂竇的屬支），非其引流型。[^1][^2]

> [!note] 考點：除哪一項外？
> 三型＝海綿竇／翼叢（basal emissary）／沿中顱窩底越岩錐入岩下或橫竇。**「向上至 MCV 再入 SSS」非其引流型**（MCV 為其屬支、流向相反）。[^1][^2]
