# cardioembolic-stroke

**腦栓塞來自「左心」——左心的血栓直接打入體循環、進入腦動脈；右心的栓子走肺動脈，造成 PE 而非腦梗。** 因此 AF（左心耳）、AMI（左室附壁血栓）、感染性心內膜炎（左側瓣膜）都是腦栓塞來源；ARVD/ARVC 雖引起心律不整卻是「右室」病，栓子打往肺，是猝死原因而非腦中風原因。記憶鉤：「左心→腦、右心→肺，記住方向不搞混」。

## Summary
- **心房顫動(AF)**：左心房/左心耳血栓 → **最常見心源性腦栓塞來源之一**(選項 B 是中風原因)。[^1]
- **急性心肌梗塞(AMI)**：左室壁運動異常/附壁血栓 → 腦栓塞(選項 A 是中風原因)。[^1]
- **感染性心內膜炎**：左側瓣膜贅生物 → 敗血性腦栓塞/黴菌性動脈瘤(選項 D 是中風原因)。[^1]
- **ARVD/ARVC**：**右心室**纖維脂肪置換致**室性心律不整與猝死**;栓子源於右心→**肺**(PE),**非腦缺血中風來源** → 故「ARVD 是腦缺血中風原因」**錯誤**＝題目正解(2016-128 正解 C)。[^2]

> [!note] 考點：哪個心臟疾病「不是」腦缺血中風的原因?
> **Arrhythmogenic right ventricular dysplasia (ARVD/ARVC)**——右室心律不整/猝死,非腦栓塞源。AF、AMI、感染性心內膜炎皆為心源性腦栓塞來源。[^1][^2]

# 影像判讀

## 定義與機轉
- **心源性腦栓塞**：栓子源自左心（左心耳血栓、左室附壁血栓、瓣膜贅生物、PFO/ASD 之矛盾栓塞），順血流打入體循環、卡在腦動脈近端分叉處，造成突發、完全性的血管阻塞。屬 TOAST 分型中的 **cardioembolism**。[^3]
- 栓子成分多為 **紅色血栓**（富含紅血球與纖維蛋白），與大動脈粥狀硬化的原位白色血栓（血小板/纖維蛋白為主、紅血球少）組成不同，是兩者影像徵象差異的病理基礎。[^4]

## 影像分布特徵（最具鑑別力）
- **多血管領域 / 雙側散在**：栓子陣雨（embolic shower）或反覆栓塞造成**多發、跨血管領域、雙側散在**的梗塞;單一影像同時見到多領域病灶對心源性最具特異性。[^3]
- **皮質為主、楔形**：栓子多卡在皮質支，形成沿血管分布的**楔形皮質（cortical）/ 灰白質交界**梗塞;此分布與分水嶺（深部白質）及腔隙（深部灰質/穿支）不同。[^3]
- **大血管近端阻塞**：常見 M1/M2、基底動脈、頸內動脈末端等近端大血管突然阻塞，梗塞體積大。[^4]
- **出血性轉化傾向**：心源性梗塞體積大、再灌流時血腦障壁破壞，**出血性轉化（HT）率高於粥狀硬化阻塞**（一研究 27.8% vs 7.6%）。[^4]

## CTA / MRI 徵象
- **DWI**：多發、散在、跨領域的高訊號小病灶（embolic pattern）是心源性的典型表現;DWI 對小梗塞較 CT/其他序列敏感。[^3]
- **磁感受性血管徵象（susceptibility vessel sign, SVS / T2*·SWI·GRE 上的血管低訊號；CT 對應 hyperdense MCA sign）**：栓子內去氧血紅素造成。心源性栓子的 SVS **直徑較大、長度較短**（一研究 5.4 vs 3.9 mm 直徑、11.2 vs 16.6 mm 長度），粥狀硬化白色血栓則細而長;心源性 SVS 陽性率亦較高（100% vs 68.2%）。**SVS 直徑越大、越易出血性轉化**。[^4]
- **出血性轉化偵測**：SWI/GRE 對早期 HT 與微出血最敏感。[^4]

## 鑑別診斷（與其他中風機轉）
- **大動脈粥狀硬化（LAA）/ 分水嶺**：分水嶺（border zone）梗塞位於主要動脈交界——皮質型（ACA-MCA、MCA-PCA 交界）與內側型（深部白質串珠狀），反映近端大血管狹窄之**血流動力學低灌流**;SVS 細長。[^3][^4]
- **腔隙（lacunar）/ 小血管病**：單一穿支（lenticulostriate、視丘穿支、橋腦穿支）供應區之**小（≤20 mm）、深部、單發**梗塞，位於基底核/視丘/腦幹/深部白質;與心源性的皮質多發分布相反。[^5]
- **分布即線索**：多領域皮質散在 → 心源性;單一深部小病灶 → 腔隙;主要動脈交界串珠/楔形 → 分水嶺/LAA。[^3]

## 判讀陷阱
- 心源性與粥狀硬化在 SVS 的差異是**統計傾向（直徑/長度分布重疊）**，單一個案不可僅憑 SVS 形態斷定來源。[^4]
- MRA 易把重度狹窄誤判為阻塞，使 LAA 之 SVS 出現假陰性，造成兩組差異被高估。[^4]
- 出血性轉化需與原發性腦出血鑑別;HT 多沿先前梗塞區（皮質/灰白質交界）分布，並有對應 DWI 缺血灶。[^4]
- 約 1/4「看似腔隙」的小皮質下梗塞其實有大血管機轉（穿支開口被母動脈粥狀斑塊阻塞，branch atheromatous disease），不應一律歸為小血管病。[^5]
