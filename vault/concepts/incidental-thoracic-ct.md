# incidental-thoracic-ct

**Main pulmonary artery (MPA) ≥3.0 cm 或直徑等於 ascending aorta 即應報告為擴張。** 其他重點：lymph node 15mm short-axis threshold、前縱膈純囊性病灶免追蹤、pericardial cyst（HU<10、液體<50ml、無鈣化）免追蹤。

## Summary
- MPA dilated：**≥3.0 cm** 或 ≥ ascending aorta 直徑 [^1]
- 淋巴結：short-axis ≥15 mm 需進一步處理 [^1]
- 前縱膈純囊性病灶（小、無症狀）：免追蹤 [^1]
- Pericardial cyst（HU<10、無增厚心包膜、液體<50ml、無鈣化）：免追蹤 [^1]
