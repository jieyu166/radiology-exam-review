# choledochal-cyst

**先天膽道囊腫,白話就是膽道某一段「鼓起來」變成囊腫;Todani 把它分成五型。記憶主線一句話:從膽總管本身鼓大（type I,最常見）→ 側壁凸個憩室（II）→ 末端鑽進十二指腸壁（III）→ 多處都有（IV）→ 一路長進肝內（V）。** 判讀分水嶺一:**Type I 膽總管囊狀/梭狀擴張最常見(~77–80%),看到膽總管整段腫先想它**;判讀分水嶺二(考試最愛的陷阱):**choledochocele 是 type III（十二指腸內遠端膽總管囊腫）,不是 IV;Type II＝膽總管憩室、Type IVa/b＝多發肝內外或多發肝外、Type V＝Caroli disease（肝內膽管囊狀擴張）**。
![Todani classification](https://img-hosting.jieyu166.workers.dev/i/MGf6prC.png) from Radiopaedia

## Summary
- **Type I（最常見,~77–80%）**：膽總管囊狀/梭狀擴張——「主幹整段腫起來」,最常見所以最先想到。[^1]
- **Type II**：膽總管**憩室(diverticulum)**——側壁凸一個包。[^1]
- **Type III＝Choledochocele**：遠端膽總管於**十二指腸壁內**囊狀擴張——位置最特別（鑽進腸壁）,所以自成一型,也最常被誤記成 IV。[^1]
- **Type IV**：**多發**(IVa 肝內＋肝外;IVb 多發肝外)——關鍵字是「不只一顆」。[^1]
- **Type V＝Caroli disease**：囊狀擴張長在**肝內膽管**——記「最深、長進肝裡」。[^1]

> [!note] 考點：關於 Todani 分類,何者「不正確」?
> 「**choledochocele 是 type IV**」不正確——**choledochocele 是 type III**。Type V＝Caroli(A)、Type I 最常見 ~77–78%(B)、Type II＝膽總管憩室(D)皆正確。[^1]

## 影像判讀骨架

### Todani 分型各自的影像型態
- **Type I（最常見，西方成人系列佔約 64%）**：肝外膽總管囊狀（IA）或梭狀（IC）擴張，膽囊管常匯入囊腫。影像判讀重點是「囊腫與膽管樹相連通」——能用造影或 MRCP 把囊腔追溯到膽管系統，才能與單純肝囊腫區分。[^2][^3]
- **Type II**：膽總管側壁長出真性憩室（diverticulum），主膽管口徑大致正常，憩室以細頸與膽總管相連。[^2]
- **Type III＝Choledochocele**：遠端膽總管在十二指腸壁內囊狀膨出，造影下呈「十二指腸內充盈缺損／囊袋」，與膽總管囊腫鑑別。[^2]
- **Type IV**：多發囊腫。IVa＝肝內＋肝外皆有（與 type V 的鑑別在於 type IV 同時有肝外擴張）；IVb＝僅肝外多發。西方成人系列 type IV 約佔 22%，為第二常見。[^2][^3]
- **Type V＝Caroli disease**：肝內膽管囊狀／梭狀擴張，可瀰漫雙葉或局限。瀰漫型常合併先天性肝纖維化（即 Caroli syndrome）。[^4]

### 各模態判讀
- **超音波（US）**：常為第一線篩檢工具，西方成人系列中診斷最常用 US，其次 CT。可顯示與膽管相連的肝門部／肝外無回音囊腔、肝內膽管囊狀擴張，產前可見 Caroli 的肝橫切「horn comb（角梳）」狀排列。[^3][^5]
- **MRCP**：評估膽管樹解剖的首選非侵入性檢查；對於偵測膽胰管解剖異常（APBJ）、勾勒囊腫與膽管樹連通關係優於 CT，且不需注射對比劑或內視鏡介入。[^6][^4]
- **CT**：用於評估囊腫範圍、是否有惡性轉化結節、規劃手術可切除性；多平面重組可顯示膽胰管匯合。[^4]
- **HIDA／肝膽核醫**：示蹤劑在囊腔內滯留／充填可證實囊腔與膽管樹相通，傳統用於與不相通的肝囊腫鑑別（兒科與膽道閉鎖鑑別時尤其有用）。[^2]
- **ERCP／PTC／術中膽道造影**：屬侵入性，是顯示膽管樹與膽胰管匯合解剖細節的決定性檢查，但因感染風險，僅在選定病例使用。[^6][^4]

### Anomalous pancreaticobiliary junction（APBJ／PBM）
- 胰管與膽管在十二指腸壁外匯合，形成過長的共同管（common channel），失去 Oddi 括約肌正常調控。「共同管理論」認為胰液逆流入膽管造成管壁慢性損傷、擴張，是膽總管囊腫（尤其 type I／IV）與惡性化的關鍵機轉。[^6][^7]
- 共同管長度閾值文獻有分歧：直接膽胰管造影常以 **≥15 mm** 為診斷標準[^6]，部分文獻採 **>10 mm**[^7]——數字並列、判讀時以該報告所用定義為準。MRCP 與高解析 MDCT 多平面重組可非侵入性顯示此長共同管。[^6]
- **好發族群**：**PBM於亞洲族群之盛行率遠高於西方族群**。[^9]
- **MRCP診斷特徵**：MR cholangiopancreatography為診斷PBM之重要影像工具，特徵為**膽胰管匯合處位於十二指腸壁外**（outside the duodenal wall）。[^9]
- **不伴膽管擴張仍可診斷**：**PBM即使不伴隨膽道擴張，仍可診斷**（並非「無膽道擴張即無法診斷」）——此類無擴張型PBM因缺乏典型影像徵象，臨床上較難診斷，許多成人病例是因合併之膽道癌臨床徵象才被發現；膽囊壁增厚可作為癌症發生前之早期診斷線索。[^9]
- **惡性風險**：PBM與**較高之膽道癌(biliary cancer)風險**相關。[^9]
- **併發症**：急性胰臟炎（兒童較常見，約30%病例，與是否合併膽道擴張無關）、慢性胰臟炎（較少見，約3%）、膽道結石、膽管炎、膽道系統癌（膽管癌、膽囊癌等）。[^9]

### 惡性風險判讀
- 膽總管囊腫具膽道惡性（以膽管癌 cholangiocarcinoma 為主）風險：西方成人切除系列合併惡性率約 **10.9%**。[^3]
- 即使完整切除仍無法完全消除風險（曾切除者中仍有惡性報告），故影像追蹤需警覺囊壁增厚、腔內結節、局部狹窄合併上游膽管擴張等可疑徵象。[^3]
- **Caroli disease／syndrome** 的肝內膽管癌風險較常人增加約 100 倍，但術前影像極難偵測——惡性多為手術標本的偶然發現，整體合併膽管癌發生率約 6.6%（各系列 2.7–37.5%）。[^4]

### 判讀陷阱與鑑別診斷（DDx）
- **central dot sign（中央點徵象）**：Caroli disease 特徵，擴張的肝內膽管包繞門靜脈分支，在對比後 CT／MRI 橫切面呈「囊腔中央一個強化小點」；瀰漫型 Caroli 約 43% 可見，並伴周邊「漏斗狀（funnel-shaped）」徵象。產前 MRI 對偵測 central dot sign 有優勢。[^8][^5]
- **與其他膽管擴張原因鑑別**：阻塞性近端膽管狹窄、原發性肝內結石所致的局限性肝內膽管擴張，可與局限型 Caroli 混淆——後者幾乎都伴肝內結石、左葉萎縮，且缺少瀰漫型的囊狀＋梭狀混合與漏斗徵象。[^8]
- **與肝囊腫鑑別**：肝囊腫不與膽管樹相通；choledochal cyst／Caroli 的囊腔與膽管連通（MRCP、HIDA 或造影可證），是關鍵分水嶺。[^2]
- **與膽道閉鎖鑑別**（嬰兒）：膽道閉鎖膽管樹不通、HIDA 無腸道排泄；type I 囊腫則可見與膽管相連的囊腔，臨床上需在新生兒黃疸鑑別中區分。[^2]

## 題目
> [!question]- Which of the following descriptions is wrong? （關於PBM/Pancreaticobiliary maljunction） (2020-279)
> **PBM cannot be diagnosed without the presence of biliary dilatation（D）**為錯誤——PBM即使不伴膽道擴張仍可診斷，惟此類無擴張型較難診斷，多數成人病例經由膽道癌臨床徵象發現。PBM通常形成過長共同管、亞洲族群盛行率遠高於西方、MRCP特徵為膽胰管匯合位於十二指腸壁外、PBM與較高膽道癌風險相關，皆為正確敘述。[^9]
