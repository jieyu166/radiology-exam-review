# choledochal-cyst

**先天膽道囊腫以 Todani 分五型;最常見為 type I,choledochocele 是 type III（非 IV）。** 判讀分水嶺一：**Type I（膽總管囊狀/梭狀擴張）最常見(~77–80%)**;判讀分水嶺二：**Type II＝膽總管憩室;Type III＝choledochocele（十二指腸內遠端膽總管囊腫）;Type IVa/b＝多發肝內外或多發肝外;Type V＝Caroli disease（肝內膽管囊狀擴張）**。
![[ADD Todani classification.png]] from Radiopaedia

## Summary
- **Type I**：膽總管囊狀/梭狀擴張,**最常見(~77–80%)**。[^1]
- **Type II**：膽總管**憩室(diverticulum)**。[^1]
- **Type III**：**Choledochocele**——遠端膽總管於十二指腸壁內囊狀擴張。[^1]
- **Type IV**：多發(IVa 肝內＋肝外;IVb 多發肝外)。[^1]
- **Type V**：**Caroli disease**（肝內膽管囊狀擴張）。[^1]

> [!note] 考點：關於 Todani 分類,何者「不正確」?
> 「**choledochocele 是 type IV**」不正確——**choledochocele 是 type III**。Type V＝Caroli(A)、Type I 最常見 ~77–78%(B)、Type II＝膽總管憩室(D)皆正確。[^1]
