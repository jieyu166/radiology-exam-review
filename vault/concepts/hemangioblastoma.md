# hemangioblastoma

**成人後顱窩囊加強化壁結節，首先想 hemangioblastoma（兒童同樣型態則想 pilocytic astrocytoma）；結節富血管、貼軟膜、可見 flow void，多發時要排 VHL。** 判讀分水嶺一：**成人＋小腦囊＋強化壁結節（貼軟膜）＝hemangioblastoma**；判讀分水嶺二：**結節有 flow void（富血管）；多發或年輕成人 → 排 von Hippel-Lindau（VHL）**。記憶鉤：年齡配型態——成人（hemangioblastoma）vs 兒童（pilocytic astrocytoma）各守一方；兩者長相一樣，年齡是關鍵。

## Summary
- **族群**：**成人**小腦最常見原發腫瘤；好發小腦半球/蚓部；多發或年輕者想 **von Hippel-Lindau**。[^1]
- **型態**：**囊腫＋強化壁結節（cyst with mural nodule）**；結節**強烈強化、貼軟膜表面、可見 flow void**（富血管）。[^1]
- 臨床：頭痛、**ataxia、nystagmus**（小腦症狀）；可分泌 EPO 致紅血球增多。[^1]
- 與本題相符：30 歲男、ataxia/nystagmus、後顱窩 cystic mass＋enhancing solid component → **hemangioblastoma**（2016-090 正解 B）。[^1]
- 鑑別：**pilocytic astrocytoma**（兒童囊+結節，第一/二十年）、轉移（成人，但通常實質）、GBM（罕見於小腦）。[^1]

> [!note] 考點：成人後顱窩囊+強化壁結節？
> **Hemangioblastoma**（成人小腦最常見原發；多發想 VHL）。兒童同型態則想 pilocytic astrocytoma。[^1]
