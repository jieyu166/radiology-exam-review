# bronchiectasis

**支氣管擴張的鑑別核心是「哪一層級的氣道受累」——中央、中段（第 4–6 級）、周邊各有對應病因與症候群。** 記憶三個具名症候群：**Williams-Campbell＝中段（mid-order）軟骨缺損，選擇性中段擴張**；**Mounier-Kuhn＝中央氣道（氣管/主支氣管）擴大，非中段**；**Swyer-James＝感染後 bronchiolitis obliterans，單側透亮肺**。記憶鉤：**「W（中段）、M（中央）、S（單側）」——W/M/S 層級由內到外顛倒」**。

## Summary
- **Williams-Campbell syndrome**：先天**第 4–6 級（mid-order/subsegmental）支氣管軟骨缺損** → **選擇性中段支氣管擴張**；吸氣時氣囊狀膨大、呼氣時塌陷。[^1]
- **Mounier-Kuhn syndrome（tracheobronchomegaly）**：**中央氣道（氣管、主支氣管）異常擴大**——是中央、非中段。[^1]
- **Swyer-James syndrome**：感染後 **bronchiolitis obliterans** → 單側透亮肺 + 不對稱支氣管擴張（整側肺）。[^1]
- **Bronchiolitis obliterans**：**小氣道**病變，非選擇性中段支氣管擴張之因。[^1]
- 分布線索：中央 → ABPA（中央性）、Mounier-Kuhn；上肺 → CF、ABPA、TB、sarcoid；下肺 → 吸入、免疫缺乏、primary ciliary dyskinesia。[^1]

## 具名症候群（依層級）
| 症候群 | 機轉 | 影像層級/特徵 |
| --- | --- | --- |
| **Williams-Campbell** | 第 4–6 級支氣管**軟骨缺損** | **中段（mid-order）選擇性**；吸氣膨大/呼氣塌陷[^1] |
| **Mounier-Kuhn** | 氣管支氣管壁彈性/肌肉缺陷 | **中央氣道擴大**（tracheobronchomegaly）[^1] |
| **Swyer-James** | 感染後 bronchiolitis obliterans | **單側透亮肺** + 不對稱支氣管擴張[^1] |
| ABPA | 過敏性麴菌 | **中央性**支氣管擴張、黏液栓（可高密度）[^1] |

> [!note] 一句話定位
> **中段選擇性 → Williams-Campbell（軟骨缺損）**；中央氣道擴大 → Mounier-Kuhn；單側透亮肺 → Swyer-James。[^1]
