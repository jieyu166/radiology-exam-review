# tree-in-bud

**樹芽徵（tree-in-bud）是 CT 上細支氣管或小肺動脈被充填後的空間輪廓，絕大多數來自小氣道感染（TB/NTM 最典型），少數是腫瘤微栓塞或 IV 藥物賦形劑堵塞小血管。** 記憶鉤：看到「發芽的樹枝」先想「小氣道被膿塞住了」，若有惡性病史且無感染線索，才轉換思路到「小血管被瘤栓堵住了」。判讀分水嶺一：**centrilobular 微結節 + 線狀分支（兩型）→ 氣道型（感染、分泌物、吸入）佔多數，最初描述於 endobronchial TB**；判讀分水嶺二：**血管型（tumor microemboli、IV talcosis）→ 隨機分布、惡性病史、無感染跡象時須考慮，易被漏看**。

## Summary

### 徵象定義與病理基礎
- **影像定義**：薄層 CT 上多發 **centrilobular（小葉中心）soft-tissue micronodules**,連接到多條口徑相近、**源自單一柄（single stalk）的分支線狀結構**,外觀像春天萌芽的樹枝。[^rg2005][^radiopaedia]
- **解剖對應**：小葉中心位置反映**末端細支氣管（terminal/respiratory bronchiole）與其伴行的小肺動脈**;線狀分支＝被填充而擴張的細支氣管或小血管沿軸向走行。[^rg2005]
- **病理基礎（氣道型）**：細支氣管管腔被**黏液、膿、發炎滲出物或肉芽**充填擴張,管壁可增厚;組織學最初描述於**結核的 endobronchial spread**(乾酪性物質充填),故又稱「endobronchial spread」徵象。[^rg2005]
- **與氣道病變高度相關**：原始薄層 CT 系列中,**27 例 tree-in-bud 有 26 例(96%)合併支氣管擴張或近端氣道管壁增厚**,僅 1 例(急性吸入)氣道正常 → 看到 tree-in-bud 應同時找近端氣道異常佐證氣道型病因。[^aquino1996]

### HRCT 特徵與分布
- **典型分布**：氣道型多為**斑塊狀、不對稱、以重力依賴/下肺與支氣管周圍**為主;隨小氣道走行成簇。[^rg2005]
- **常見伴隨**：支氣管擴張、氣道壁增厚、小葉中心結節、consolidation;TB/NTM 另可見空洞、淋巴結鈣化。[^aquino1996][^ntm2005]
- **TB vs NTM 取向(同屬分枝桿菌但側重不同)**:tree-in-bud 與小結節兩者皆常見、重疊大;**MDR-TB 偏多發空洞、較年輕**;**NTM(尤其 MAC/M. abscessus)偏 nodular bronchiectatic form、廣泛支氣管擴張、年長女性**。[^chung2006][^ntm2005]

### DDx 分類（依 RadioGraphics 經典 review 架構）
| 大類 | 代表病因 | 影像/臨床鑑別線索 |
|---|---|---|
| **感染（最常見）** | 分枝桿菌(TB endobronchial spread、NTM)、細菌性 bronchopneumonia、病毒、黴菌、寄生蟲 | 急性病程、發燒、痰菌;TB 看上肺空洞、淋巴結鈣化;NTM 看中葉/舌葉 bronchiectasis[^rg2005][^aquino1996] |
| **先天性** | 囊性纖維化(CF)、原發性纖毛運動不良(PCD/Kartagener) | 年輕、上肺為主(CF)、內臟轉位＋鼻竇炎(Kartagener)[^rg2005] |
| **吸入／誤吸** | aspiration、foreign substance inhalation | 重力依賴後段/下葉;吞嚥障礙、意識病史[^rg2005][^aquino1996] |
| **特發性氣道病** | diffuse panbronchiolitis(DPB)、obliterative bronchiolitis | DPB:東亞、慢性鼻竇炎、瀰漫小葉中心結節[^rg2005] |
| **免疫／過敏** | ABPA、過敏 | ABPA:氣喘＋中央性支氣管擴張＋黏液栓(finger-in-glove)[^rg2005] |
| **結締組織病** | 相關小氣道炎 | 已知 CTD 病史[^rg2005] |
| **周邊肺血管型** | **腫瘤性肺微栓塞／PTTM**、IV talcosis | **惡性病史、無近端氣道病變、增強 CT 無大血管 PE**[^rg2005][^pttm2023] |

### 鑑別關鍵與陷阱：血管型 tree-in-bud
- **核心觀念**：因肺小動脈與細支氣管平行走行,**小動脈被瘤栓填充或內膜增生擴張時也會呈 tree-in-bud**,故此徵象**不再被視為小氣道病變專屬**。[^pttm2023]
- **PTTM(pulmonary tumour thrombotic microangiopathy)**:瘤細胞栓塞肺小動脈,誘發內皮損傷→纖維細胞性內膜增生＋血栓;HRCT 見 **vascular tree-in-bud＋ground-glass centrilobular 結節＋小葉間隔增厚**,常合併縱隔/肺門淋巴結病變與右心衰竭徵象。[^pttm2023]
- **最大陷阱(易漏看)**:**癌症病史病人急性惡化呼吸衰竭、但增強 CT 在大肺動脈找不到栓子**時,要把 PTTM／血管型 tree-in-bud 列入,而非僅當作感染或單純 PE;PTTM 進展極快(發病後常 1–2 週內死亡),早期診斷影響預後。[^pttm2023]
- **好發原發癌**:腺癌最多,**胃癌最常見(約 26%)**,其次乳癌(21%)、攝護腺(13%)、肺(10%)。[^pttm2023]
- **其他血管型**:IV 藥物賦形劑肉芽腫(靜脈 talcosis、口服藥磨碎注射之 microcrystalline cellulose）→ 隨機分布、注射毒癮史。[^rg2005]

> [!note] 收斂思路
> tree-in-bud 先想**小氣道感染（TB/NTM）**;若有惡性病史、無感染證據、隨機分布,須想**腫瘤微栓塞（血管型）**或 IV 藥物 talcosis。配合分布、空洞、淋巴結、臨床。
