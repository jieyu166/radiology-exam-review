# uvc-catheter-route

**UVC 從臍靜脈進入後，正常路徑依序經過 portal sinus → ductus venosus → IVC → right atrium。UVC 不應經過 portal vein（左右門脈分支），若導管進入 portal vein 則為 malposition。**

## Summary
- 正常 UVC 路徑 [^1]：
  1. Umbilical vein → 進入肝門
  2. → **Portal sinus**（左門脈水平段）
  3. → **Ductus venosus**（旁路肝臟）
  4. → **IVC**（下腔靜脈）
  5. → Right atrium
- **Portal vein**（右門脈分支等）不在正常 UVC 路徑上 [^1]
- UVC tip 理想位置：IVC-RA junction（T8-T10 level on XR）[^1]

## 陷阱
- Portal sinus ≠ portal vein：portal sinus 是左門脈水平段（UVC 會經過），而 portal vein branch 不在 UVC 正常路徑上
