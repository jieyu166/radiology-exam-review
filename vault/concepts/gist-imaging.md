# gist-imaging

**GIST最常見部位依序為胃(55.6%)>小腸(31.8%)>大腸直腸(6.0%)>其他/多處(5.5%)>食道(0.7%)，胃為最常見但小腸位居第二，並非大腸；大型胃GIST合併中央壞死潰瘍須與胃淋巴瘤鑑別，淋巴結轉移少見，對Imatinib治療反應後可呈囊性變合併黏液樣退化與疤痕化。**

## Summary
- **好發部位頻率**：**胃（55.6%）> 小腸（31.8%）> 大腸直腸（6.0%）> 其他/多處位置（5.5%）> 食道（0.7%）**——小腸為第二常見，並非大腸/後腹腔/腸繫膜。[^1]
- **大型胃GIST**：合併中央壞死與潰瘍時，須與**胃淋巴瘤（gastric lymphoma）**鑑別。[^1]
- **淋巴結轉移**：對GIST而言**不常見（uncommon）**。[^1]
- **治療反應**：對Imatinib治療反應良好之GIST，影像上可呈**囊性變（cystic change）合併黏液樣退化（myxoid degeneration）與疤痕化（scarring）**。[^1]

## 陷阱
- 常見好發部位排序易誤植為「胃>大腸>後腹腔>腸繫膜>小腸」——正確應為**胃>小腸>大腸直腸>其他>食道**。[^1]
