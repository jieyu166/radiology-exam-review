# gamekeeper-thumb

**拇指被往外（尺）掰傷的是「尺側副韌帶（UCL）」，橈側副韌帶完好無損——Gamekeeper's/Skier's thumb 傷的永遠是 UCL，這是核心口訣：UCL = U Can't Lie（不能說「橈側」）。** 判讀分水嶺一：**受傷結構必是第一 MCP 尺側副韌帶，可伴近端指骨基部撕脫骨折。** 判讀分水嶺二：**Stener lesion = UCL 斷端被內收肌腱膜夾住無法癒合 → 必須手術；無 Stener 可保守治療。**

## Summary

### 定義與機轉
- **受傷結構**：拇指第一 **MCP 關節的尺側副韌帶（ulnar collateral ligament, UCL）** 撕脫/斷裂;故「**橈側副韌帶（radial collateral ligament）**」**未受傷**＝題目正解（2016-133 正解 B）。[^1]
- **機轉**：拇指被強力外展（橈偏）使 UCL 過度拉伸而斷裂;**Gamekeeper's**＝慢性反覆外展應力（原指獵場看守人扭兔頸的累積傷害）、**Skier's**＝急性（滑雪杖把手把外展力傳到拇指）。[^1][^2]
- **解剖要點**：UCL 由 proper（MCP 屈曲時緊）與 accessory（伸直時緊）兩束組成;遠端止點在近端指骨基部尺側，多數撕裂發生在**遠端止點**。[^2][^3]

### X 光（撕脫骨折）
- 標準攝影：拇指**正位（PA）＋側位（±斜位）**平片，主要找**近端指骨基部尺側撕脫骨折**（option C 可受傷）;單純韌帶撕裂平片可正常。[^2][^4]
- **應力位（stress view）**：對受傷拇指施加橈向（valgus）應力，量測 MCP 張開角;**MCP 屈曲 30° 下 valgus laxity > 35°、或較健側多 > 15°** 支持完全斷裂（accessory UCL／背側關節囊亦受損）。文獻角度門檻分歧（約 20–45° 皆有報告），並列供參。[^3][^4]
- **應力位陷阱**：施力本身可能把**未移位的撕脫片或 nondisplaced UCL 推成移位／Stener lesion**，故部分醫師對疑似完全斷裂者改以影像（US/MRI）取代或先行影像。[^3][^4]

### US／MRI 對 UCL 與 Stener lesion 判讀
- **正常 UCL**：US 呈韌帶自 metacarpal head 連續延伸至近端指骨基部的纖維狀結構;MRI 上為低訊號條索。撕裂時呈不連續、增厚、訊號上升或波浪狀回縮。[^1][^2]
- **Stener lesion 定義**：完全斷裂後 UCL 近端斷端**回縮並移位至內收肌腱膜（adductor aponeurosis）淺層**，被腱膜阻隔卡住、無法回到原止點而**不能癒合 → 必須手術**;見於約 **64–88% 的完全 UCL 斷裂**。[^1][^2]
- **「yo-yo on a string」徵象**：回縮成球狀的 UCL（yo-yo）位於 metacarpal head 旁、淺於內收肌腱膜（the string＝增厚的腱膜），US 與 MRI 皆可見;亦描述為 tadpole（蝌蚪）外形——頭＝回縮 UCL、尾＝腱膜。[^1][^2]
- **US 動態優勢**：US 快速、可雙側比較且**動態**;以拇指 IP/MCP 被動屈曲動作可區分**未移位撕裂**（韌帶仍貼著骨）與 **Stener lesion**（球狀斷端卡在腱膜淺層、不隨關節移動回位）。[^1][^2]
- **MRI**：對 UCL 完整度、accessory UCL、關節囊與隱匿撕脫骨折判讀佳;MR arthrography 與應力位皆曾用於鑑別 nondisplaced 與 displaced（Stener）撕裂。[^2][^5]

### 鑑別診斷（DDx）
- **橈側副韌帶（RCL）損傷**：受力方向相反（拇指被內收／尺偏），位於 MCP 橈側;Gamekeeper/Skier 不傷 RCL（題目正解依據）。[^1]
- **近端指骨基部撕脫骨折 vs Stener**：兩者可並存;撕脫骨折在平片即可見，Stener 需 US/MRI 判定斷端是否移位至腱膜淺層。[^2][^4]
- **第一 MCP 退化／OA、籽骨（sesamoid）相關病變**：慢性 gamekeeper 反覆損傷可致 MCP 不穩、疼痛與續發性 OA。[^3]

### 判讀陷阱
- **Stener lesion 必須辨識**：它決定保守 vs 手術;漏判 → 慢性不穩、疼痛、續發 OA。平片對 Stener **無診斷力**，務必加做 US 或 MRI。[^2][^4]
- **完全斷裂的回縮 UCL 可能不在原位**：MRI／US 若只看止點處會誤判，須往近端 metacarpal head 旁找回縮球狀斷端與腱膜關係。[^1][^2]
- **應力位前先評估有無撕脫骨折／移位**：避免施力造成醫源性移位（見上）。[^3][^4]

> [!note] 考點：Gamekeeper's thumb 何者「未」受傷?
> **橈側副韌帶（radial collateral ligament）**——受傷的是**尺側副韌帶（UCL）**於第一 MCP 關節;可伴近端指骨基部撕脫骨折、Stener lesion。[^1]

## 題目
> [!question]- In a Stener lesion, what structure entrapped the ulnar collateral ligament of first metacarpophalangeal joint, which resulted in the need for surgery? (2020-292)
> **Adductor aponeurosis（C）**——完全斷裂之UCL近端斷端回縮並移位至內收肌腱膜(adductor pollicis aponeurosis)淺層，被腱膜阻隔卡住無法回到原止點，故無法自行癒合須手術。[^1][^2]
