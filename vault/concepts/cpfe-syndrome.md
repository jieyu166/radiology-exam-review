# cpfe-syndrome

**CPFE＝『上肺氣腫＋下肺纖維化』常共存於吸菸者，被視為一個獨立疾病實體，凸顯單純把肺氣腫與纖維化二分的侷限;常合併肺高壓、預後差。** 判讀分水嶺：**上肺中央小葉/間隔旁肺氣腫＋下肺 UIP 或 NSIP 型纖維化;肺量測『假性正常』（氣腫使肺容積維持但 DLCO 顯著下降）;常合併嚴重肺高壓、肺癌風險高;非「只是兩種常見病剛好同時出現」,而是有共同（吸菸）易感性的獨立症候群。**

## Summary
- **本質**：肺氣腫與間質纖維化**常共存（尤吸菸者）,應視為一個獨立疾病實體（CPFE syndrome）**——凸顯單純二分肺氣腫/纖維化的侷限。[^1]
- **分布**：**上肺為主的中央小葉/間隔旁肺氣腫**＋**下肺為主的纖維化（UIP 或 NSIP 型）**。[^1]
- **生理**：肺容積/肺量測可『假性正常』（氣腫抵消纖維化的限制），但 **DLCO 顯著下降**。[^1]
- **併發/預後**：常合併**嚴重肺高壓**、肺癌風險高、預後差。[^1]
- **病因**：推測為部分病人對香菸的內在易感性（吸菸誘發纖維化）。[^1]

> [!note] 考點：關於肺間質纖維化與肺氣腫，何者「正確」?
> 「**兩者常共存且應視為一個獨立疾病實體（CPFE）**」正確。肺氣腫非較少見、非總在上肺、非「只是兩病剛好同時出現」。[^1]
