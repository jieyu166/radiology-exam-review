# ameloblastoma

**Ameloblastoma 是最常見的 odontogenic tumor，組織學上為良性但具有 locally aggressive 特性。最常發生於 mandible（下顎骨，約80%），而非 maxilla。典型影像為 multilocular radiolucent lesion（soap bubble appearance）。**

## Summary
- **最常見位置**：mandible 後段（posterior body / ramus），約 80% [^1]
- **非 maxilla**（上顎骨最常見的 odontogenic tumor 是 odontoma）[^1]
- 組織學良性但 locally aggressive，復發率高 [^1]
- 影像：multilocular radiolucent lesion = **soap bubble** / honeycomb pattern on panoramic radiograph [^1]
- 臨床：口腔/顏面腫脹、牙齒鬆動 [^1]

## 陷阱
- 最常見在 **mandible**，不是 maxilla — 這是常見考題陷阱
