# anomalous-pulmonary-venous-return

**肺靜脈沒接回左心房，而是拐彎引流到右側循環——這就是 APVR 的核心。** 正常四條肺靜脈「左2右2」全開口於左心房；TAPVR 依引流終點分四型：supracardiac（上縱膈）、cardiac（冠狀竇/右房）、infracardiac（橫膈下）、mixed。記憶鉤：阻塞風險最高的是「往下鑽過橫膈」的 infracardiac 型，因靜脈需穿越橫膈孔，易受壓；supracardiac 型阻塞點在垂直靜脈「跨越左肺動脈」前方；cardiac 型外觀可酷似 cor triatriatum，兩者都是左心房被膜或異常靜脈隔開。

## Summary
- **分型**：**supracardiac、cardiac、infracardiac、mixed**。[^1]
- **正常肺靜脈數**：**左 2 ＋ 右 2 ＝ 4 條開口於左心房**;故「檢查左2右3開口排除診斷」**錯誤**（正常非右3）。[^1]
- **阻塞**：**infracardiac 型最常合併肺靜脈阻塞**;**supracardiac 型阻塞處在垂直靜脈(ascending vein)跨越左肺動脈處**。[^1]
- **Cardiac 型 TAPVR**：引流入冠狀竇/右房,**可酷似 cor triatriatum**。[^1]

### TAPVR 四型判讀骨架（CXR + CTA/MR）
- **Type I — supracardiac（上心型，最常見約 50%）**：四條肺靜脈先匯成 confluence,經**左側垂直/上行靜脈（vertical/ascending vein）→ 左無名靜脈（brachiocephalic vein）→ SVC → 右房**;少數引流入 azygos vein。[^2][^3] CXR 典型「**雪人徵 / figure-of-8 / cottage-loaf heart**」——雪人頭＝擴張的垂直靜脈＋左無名靜脈＋SVC,雪人身＝擴大的右房與肺動脈幹;此徵象通常於數月後才形成,新生兒期尚不明顯。[^2][^3]
- **Type II — cardiac（心型）**：肺靜脈匯入**冠狀竇（最常）或直接入右房/SVC**;判讀可見冠狀竇擴大,**外觀可酷似 cor triatriatum**（左房被異常靜脈/隔膜分隔）。[^1][^3]
- **Type III — infracardiac（下心型）**：肺靜脈匯成垂直 confluence,以**下行靜脈穿過橫膈裂孔（多經 ductus venosus/門脈系統）→ 肝靜脈/門脈/IVC**;**幾乎必然合併阻塞**——靜脈須穿越橫膈孔且血流先經肝實質,故新生兒早期即以**嚴重肺水腫／肺靜脈高壓**表現,CXR 心臟不大但雙肺呈瀰漫性間質水腫。[^1][^2][^3]
- **Type IV — mixed（混合型，最少見）**：不同肺葉靜脈引流至多個不同終點。[^1][^3]
- **CTA / MR 角色**：勝在直接描繪**完整引流路徑、confluence 位置與阻塞點**;因新生兒需早期手術,**MDCT 血管攝影（快速、高空間解析度）常優先於 MR**。[^2][^3]

### 阻塞點與「血流鉗夾（hemodynamic vise）」
- **Supracardiac 阻塞點**：上行垂直靜脈走行於**左肺動脈與左主支氣管之間**時受夾擠（「hemodynamic/vascular vise」）;肺靜脈壓上升→肺動脈壓上升→更壓迫垂直靜脈,形成惡性循環。判讀時應沿垂直靜脈追蹤至此鉗夾段找最窄處。[^4]
- 影像阻塞徵象：confluence 或引流靜脈局部狹窄、近端擴張、**肺靜脈高壓／瀰漫間質性肺水腫、Kerley B、毛玻璃**;此型若 type I 因 vise 嚴重阻塞,CXR 可酷似 type III（小心臟＋肺水腫,而非雪人）。[^4]

### PAPVR 與 scimitar syndrome
- **PAPVR 最常型**：**右上肺靜脈引流入 SVC**,**80–90% 兒童（成人約 47%）合併 sinus venosus 型 ASD**;左上肺靜脈型則經垂直靜脈入左無名靜脈。[^5]
- **Scimitar syndrome（彎刀症候群／congenital pulmonary venolobar syndrome）**：**右肺（常右下葉）靜脈呈彎刀狀引流入 IVC**,CXR 見右心緣旁向下漸寬的彎曲血管影（彎刀徵）;合併**右肺發育不全、心臟右移（dextroposition）、患側肺動脈細小、來自腹主動脈的體循環供血**。[^5]
- **與 sinus venosus ASD 區辨**：scimitar 的腔靜脈接合正常、**通常不形成 ASD**;而一般右上肺靜脈型 PAPVR 多合併 sinus venosus ASD。[^5]
- **判讀重點（PAPVR）**：找異常垂直靜脈／彎刀血管追終點、評估有無 sinus venosus ASD、量化分流（Qp/Qs >1.5 才血流上重要,故常較晚發病）、看**右心容積負荷**（右房右室擴大、肺動脈幹擴張）。[^5]

### 判讀重點與 DDx
- **系統性追蹤四條肺靜脈終點**:確認是否全數（TAPVR）或部分（PAPVR）未入左房;找 confluence、垂直/下行靜脈與其引流終點。[^1][^3]
- **找阻塞徵象**:垂直靜脈跨左肺動脈處（supracardiac）或橫膈裂孔（infracardiac）的局部狹窄＋肺靜脈高壓。[^2][^4]
- **找合併病灶**:ASD（PAPVR 之 sinus venosus 型）、右心負荷、左房偏小（TAPVR 因血流不入左房）。[^3][^5]
- **DDx**：**cor triatriatum**（左房內隔膜,肺靜脈仍入左房,與 cardiac 型 TAPVR 影像相似）、肺靜脈狹窄/閉鎖、**PA-IVS／其他造成新生兒肺水腫之原因**;infracardiac 阻塞型須與**新生兒呼吸窘迫症候群、肺淋巴管擴張症**等瀰漫肺病鑑別。[^1][^3]

> [!note] 考點：關於肺靜脈回流異常診斷,何者「不對」?
> 「**須確認左2右3肺靜脈開口以排除診斷**」不對——正常為**左2右2（共4條）**。四型分類、infracardiac 最易阻塞、supracardiac 阻塞於垂直靜脈跨左肺動脈、cardiac 型似 cor triatriatum 皆正確。[^1]
