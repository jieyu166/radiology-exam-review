# ti-rads

**ACR TI-RADS 的五個評估項目為：composition、echogenicity、shape、margin、echogenic foci。Vascularity（血流）不包含在 TI-RADS 評估項目中。**

## Summary
- ACR TI-RADS 五大評估項目 [^1]：
  1. **Composition**：cystic / spongiform / mixed / solid
  2. **Echogenicity**：anechoic / hyperechoic / isoechoic / hypoechoic / very hypoechoic
  3. **Shape**：wider-than-tall / taller-than-wide
  4. **Margin**：smooth / ill-defined / lobulated / irregular / extra-thyroidal extension
  5. **Echogenic foci**：none / large comet-tail / macrocalcification / peripheral calcification / punctate echogenic foci
- **Vascularity 不是 TI-RADS 評估項目**（舊版 Kwak TI-RADS 曾納入，ACR TI-RADS 已移除）[^1]

## 陷阱
- 不要把 vascularity 當成 TI-RADS 項目，這是常見考題陷阱
