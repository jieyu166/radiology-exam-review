# mcdonald-ms-criteria

**McDonald（2010）MS 的『空間多發性（DIS）』＝在以下『四』個中樞部位中 ≥2 處各有 ≥1 個 T2 病灶：periventricular、juxtacortical、infratentorial、spinal cord。『視神經（optic nerve）』『不』在 2010 DIS 部位之列。** 判讀分水嶺一：**DIS＝四部位中 ≥2 處**;判讀分水嶺二：**optic nerve 非 2010 DIS 部位（2017 版亦未納入為 DIS）**。

## Summary
- **DIS（dissemination in space,2010）四部位**：**periventricular、juxtacortical、infratentorial、spinal cord**;≥2 部位各 ≥1 T2 病灶即滿足。[^1]
- **optic nerve『不』屬 DIS 部位** → 故為「不算入 DIS」之正解（2016-181 正解 C）。[^1]
- **DIT（時間多發性）**：同時存在強化與非強化病灶,或追蹤出現新 T2/強化病灶。[^1]
- 2017 修訂：juxtacortical 擴為 cortical/juxtacortical、CSF oligoclonal bands 可替代 DIT;**optic nerve 仍非 DIS 部位**。[^1]

> [!note] 考點：2010 McDonald DIS 不含哪個部位?
> **Optic nerve（視神經）**。DIS 四部位＝periventricular、juxtacortical、infratentorial、spinal cord。[^1]

### 參考來源
[^1]: *McDonald diagnostic criteria for multiple sclerosis*（2010 版,Polman 2011）;DIS＝四部位（**periventricular、juxtacortical、infratentorial、spinal cord**）中 ≥2 處各 ≥1 T2 病灶;**optic nerve 非 DIS 部位**（標準 MS 影像準則）。
