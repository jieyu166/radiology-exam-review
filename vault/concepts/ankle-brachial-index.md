# ankle-brachial-index

**ABI = 踝動脈收縮壓 / 兩側肱動脈收縮壓中較高者（非兩側之和）。ABI < 0.9 為異常（PAD）；ABI > 1.3 提示動脈鈣化（不可壓縮）。**

## Summary
- 公式：ABI = ankle SBP / **higher** of the two brachial SBPs [^1]
- 正常：1.0 – 1.3 [^1]
- **< 0.9**：peripheral arterial disease [^1]
- **> 1.3**：動脈嚴重鈣化（不可壓縮），多見於糖尿病/CKD [^1]
