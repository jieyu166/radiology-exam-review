# ccta

**CCTA 的核心價值是「排除」而非「確診」：NPV 約 99%，陰性幾乎可放心排除阻塞性冠狀動脈狹窄，避免侵入性心導管。** 但 PPV 偏低（≥70% 狹窄時僅 48%），因為鈣化 blooming 產生偽陽性——calcium score >400 會顯著拉低 specificity，陽性結果常需侵入性攝影確認。記憶鉤：「陰性信，陽性疑——CCTA 是守門員，不是裁判」。

## Summary
- **定位**：對中等前測機率、無已知 CAD 的胸痛病人，CCTA 是 rule-out 阻塞性冠狀動脈狹窄的有效非侵入替代（守門員角色）。[^1]
- **診斷效能（ACCURACY trial，64-MDCT，patient-based）**：≥50% 狹窄 Sens 95% / Spec 83% / PPV 64% / NPV 99%；**≥70% 狹窄 Sens 94% / Spec 83% / PPV 48% / NPV 99%**。[^1]
- **rule-out > rule-in**：99% NPV 使 CCTA 能可靠排除狹窄；但 PPV 偏低 → 陽性需侵入性攝影確認。[^1]
- **鈣化陷阱**：calcium score >400 顯著降低 specificity；重度鈣化的 **blooming artifact** 造成管腔高估／偽陽性。[^1][^2]
- **報告框架**：CAD-RADS 2.0（2022）以「狹窄度（0–5）＋斑塊負荷（P1–P4）＋modifiers（N/HRP/I/S/G/E）」標準化分級與處置建議。[^2][^5]
- **技術骨架**：心率控制（β-blocker，目標 ≤60 bpm）＋舌下 nitroglycerin（掃描前 5 分鐘）；心率規律且 <65 bpm 用 prospective gating（低劑量，預設），需心室功能或心率無法控制時才用 retrospective gating。[^6]
- **進階判讀**：calcium score（Agatston）量化鈣化負荷與風險分層；FFR-CT 以 ≤0.80 判定病灶特異性缺血（lesion-specific ischemia）；冠狀動脈異常起源之 interarterial（惡性）走行為猝死高風險，CT 為定義起源與走行的首選。[^5][^7][^8]

## 定義與角色
CCTA 為心電圖閘控的多切 CT 冠狀動脈血管攝影。在無已知 CAD、中等前測機率的胸痛族群，其臨床定位是**非侵入性排除**阻塞性冠狀動脈狹窄。[^1]

## 診斷效能 — ACCURACY trial（核心實證）
ACCURACY 為前瞻、多中心（16 site）試驗，納入 230 名無已知 CAD 的胸痛病人，同時接受 64-MDCT CCTA 與侵入性冠狀動脈攝影（ICA，以 QCA 為標準），未因 baseline calcium score 或 BMI 排除任何受試者。[^1]

| 狹窄門檻 | Sensitivity | Specificity | PPV | NPV |
| --- | --- | --- | --- | --- |
| **≥50%** | 95% | 83% | 64% | 99% |
| **≥70%** | 94% | 83% | **48%** | **99%** |

- 肥胖 vs 非肥胖、心率 ≤65 vs >65 bpm 之間 Sens/Spec 無顯著差異；**但 calcium score >400 顯著降低 specificity**。[^1]
- 結論：99% NPV（病人與血管層級）確立 CCTA 為 rule-out 阻塞性狹窄的有效非侵入替代。[^1]

> [!note] 為何 PPV 低、NPV 高？
> CCTA 強在「陰性可信」（高 NPV → 排除）；偽陽性偏多（鈣化 blooming、運動假影）使 PPV 偏低 → **陽性結果常需 ICA 確認**。臨床價值在 rule-out 而非 rule-in。[^1][^2]

## 陷阱與報告
- **Calcium blooming artifact**：重度鈣化斑塊在 CTA 上 blooming，造成管腔高估／偽陽性；以 calcium-specific window width/level 可減少 blooming、改善管腔顯示。[^2][^3][^4]
- **CAD-RADS 0–5**：標準化冠狀動脈狹窄分級與處置建議的報告框架。[^2]

## 技術 — 適應症與掃描骨架
- **適應症**：低-中度前測機率、無已知 CAD 的急/慢性胸痛 rule-out；急診胸痛（acute chest pain）排除阻塞性狹窄。[^1][^6]
- **心率控制**：多數受檢者需口服/靜脈 β-blocker 把心率與心率變異降下來，**目標 ≤60 bpm**，以利冠狀動脈清晰顯影（尤其 prospective triggering）。[^6]
- **Nitroglycerin**：舌下含片或噴霧（平滑肌鬆弛、擴張冠狀動脈），**掃描前約 5 分鐘**給予、待最大效果後再取像，原則上為預設流程。[^6]
- **Prospective（前瞻 ECG-triggered, step-and-shoot）vs retrospective（回溯 ECG-gated, helical）**：
  - Prospective 為心率規律且 **<65 bpm** 多數病人的**預設**模式，輻射劑量顯著較低。[^6]
  - Retrospective 取像涵蓋整個 R-R cycle，**需要心室功能評估**或**心率無法充分壓低/不規律**時才用；劑量較高。[^6]
- **心律/心率影響**：心率過快或心律不整（如心房顫動、早期收縮）增加 motion 與 step artifact 風險，是影像品質的主要決定因子。[^6]
- **造影時相**：擷取最少運動的舒張中期（mid-diastole，~70–75% R-R，較慢心率）或收縮末期（end-systole，較快心率）以凍結冠狀動脈運動。[^6]

## 判讀 — CAD-RADS 2.0（2022）
標準化報告：每例給「狹窄度（0–5）＋斑塊負荷（P1–P4）＋modifiers」。[^5]

| 分級 | 最大狹窄 | 說明／處置要點 |
| --- | --- | --- |
| **0** | 0% | 無斑塊或狹窄；無需後續檢查。[^5] |
| **1** | 1–24% | 極輕度；無需後續。[^5] |
| **2** | 25–49% | 輕度；無需後續。[^5] |
| **3** | 50–69% | 中度；考慮 CT-FFR／壓力測試／ICA。[^5] |
| **4A** | 70–99% | 重度（單/雙血管）；考慮 ICA 或功能評估。[^5] |
| **4B** | left main >50% 或三血管 70–99% | 高負荷；建議 ICA。[^5] |
| **5** | 100% | 完全阻塞；建議 ICA。[^5] |

- **斑塊負荷 P1–P4**：mild / moderate / extensive / very-extensive，與 calcium score（Agatston）對應，獨立於狹窄度描述整體斑塊量（CAD-RADS 0 無 P0）。[^5]
- **Modifiers**：
  - **N**：nondiagnostic，一或多節段無法評估。[^5]
  - **HRP**：高風險斑塊（≥2 項特徵，見下）。[^5]
  - **I**：缺血 modifier（CT-FFR 或 stress CTP）；I+ 有缺血、I- 無、I± 不確定。[^5]
  - **S**：支架（stent）；**G**：繞道（graft）。[^5]
  - **E**：exception，非動脈硬化異常（如異常起源、瘤、夾層、肌橋等）。[^5]
- **斑塊性質**：calcified / non-calcified（軟斑塊，較易破裂）/ mixed。[^2][^5]
- **高風險（vulnerable）斑塊徵象（HRP，≥2 項）**：[^5]
  - **Positive remodeling**：病灶處血管壁向外代償性擴大、管腔保留。
  - **Low-attenuation plaque**：密度 **<30 HU**（富脂核心）。
  - **Napkin-ring sign**：中央低密度核心、周邊環狀較高密度「餐巾環」。
  - **Spotty calcification**：點狀鈣化（**<3 mm**）。

## 判讀 — 進階：calcium score / FFR-CT / 異常起源
- **Calcium score（Agatston）**：量化鈣化斑塊負荷做心血管風險分層；calcium score **>400** 顯著降低 CCTA specificity（blooming 偽陽性），與 P 負荷分級對應。[^1][^5]
- **FFR-CT（CT-derived fractional flow reserve）**：對解剖中度狹窄（CAD-RADS 3 / 4A）加做的功能性評估，於狹窄遠端約 2 cm 量測；**>0.80 正常、≤0.80（多採 <0.80 cutoff）判定病灶特異性缺血**，0.76–0.80 為灰帶需進一步風險分層（DISCOVER-FLOW/DeFACTO/HeartFlow NXT 系列驗證）。[^7]
- **冠狀動脈異常起源/走行**：CT 為定義起源與全程走行的首選（優於傳統血管攝影）。**Interarterial（昔稱 malignant）走行**——冠狀動脈起自對側竇、行經主動脈根與肺動脈/右室流出道之間——為**猝死高風險**。CT 惡性解剖徵象包含：**起自對側竇之高位起源（high take-off，位於主-肺動脈間受兩血管收縮期擴張壓迫）**、**急性銳角 take-off**、**壁內（intramural）走行/近端管腔受壓變窄呈橢圓**；**左冠脈起自右竇之 interarterial 走行猝死風險最高，常不論症狀建議手術；右冠脈異常預後較佳，多於有缺血證據時才手術**。[^8]

## 判讀陷阱與支架/繞道評估
- **Calcium blooming artifact**：重度鈣化斑塊在 CTA 上 blooming，造成管腔高估／偽陽性；以 calcium-specific（wider）window width/level 與 sharp（dedicated）kernel 可減少 blooming、改善管腔顯示。[^2][^3][^4]
- **Motion artifact（運動假影）**：心率過快/不規律或屏氣不佳造成血管模糊，可被誤判為狹窄或假性異常起源（如惡性 RCA anomaly 可被 motion 模擬）。[^6]
- **Step artifact（階梯/錯位假影）**：prospective step-and-shoot 各區段心動週期/呼吸不一致時，相鄰 slab 接縫處出現錯位，可截斷或假性中斷血管。[^6]
- **心率過快**：是 motion/step artifact 的根本驅動因子，故術前心率控制至關重要。[^6]
- **支架（S）評估**：blooming/struts 限制 in-stent lumen 判讀；**支架直徑 <3 mm 顯著降低偵測 in-stent restenosis 的準確度（≤3 mm ~54% vs >3 mm ~86%），小支架不建議以 CT 評估**；理想對象為 ≥3 mm 與近端節段支架；sharp kernel + wider window 改善。[^3][^4]
- **繞道（G）評估**：CCTA 對 bypass graft 通暢度評估準確度高（graft 口徑大、運動少）；報告須涵蓋 graft 本體、anastomosis 與被繞道之原生（native）血管遠端流出道。[^2][^5]
