# mccune-albright

**McCune-Albright 三聯靠一個鉤記牢：「骨（多骨性 FD）、皮（coast-of-Maine 咖啡牛奶斑）、荷爾蒙（性早熟）」，其中性早熟以女孩為主——「男性為主」恰好是考題的假選項。** 判讀分水嶺一：**café-au-lait 邊界不規則（coast of Maine）**，與神經纖維瘤病（光滑邊界）不同；判讀分水嶺二：**性早熟＝最典型內分泌異常、女性偏多**（GNAS 鑲嵌突變）。

## Summary

### 定義與診斷三聯
- **McCune-Albright（MAS）經典三聯**：① 多骨性纖維異常增生（polyostotic fibrous dysplasia）、② café-au-lait 斑、③ 內分泌過度功能（典型為性早熟）。[^1][^2]
- **診斷門檻**：臨床上**三聯具備其二**即可診斷；因鑲嵌性（mosaicism）導致偽陰性，基因檢測臨床效用有限，**以臨床特徵為主**。[^2]
- **機轉**：**GNAS 體細胞活化突變（鑲嵌型）**，於胚胎期合子後（post-zygotic）發生（常見 Arg201），使 Gsα 持續活化、cAMP 上升，驅動骨、皮膚、內分泌組織異常。[^1][^2]

### Fibrous dysplasia 的 X 光判讀（MAS 的骨病灶基礎）
- **基本徵象**：髓內**膨脹性病變＋ground-glass（霧玻璃）基質＋皮質變薄但外緣平滑＋無 periosteal reaction**。[^1]
- **rind sign**：病灶外圍**厚硬化緣**（厚骨皮質殼），近端股骨最常見。[^1]
- **shepherd's crook deformity**：近端股骨因反覆病理性骨折逐漸內翻/彎曲變形，為長骨 FD 的典型併發。[^1]
- **分布差異**：**顱面骨病灶偏緻密硬化**、四肢/中軸骨偏透亮；可見 endosteal scalloping 但皮質完整性保留。[^1]

### CT / MRI 訊號與陷阱
- **CT**：基質衰減約 **60–140 HU**；較 X 光更能界定膨脹、皮質與顱面病灶範圍，為評估骨變形/範圍之首選；**打對比劑會強化**。[^1]
- **MRI 訊號非特異**：T1 多為**中等至低訊號**；**T2 變動（中等至高訊號）**，T2 越高代表骨小樑越少；含囊變區會出現亮 T2。[^1]
- **判讀陷阱（易誤為侵襲性）**：MRI 訊號與強化型態（中央斑點、邊緣、均質或混合）**無法可靠區分良惡性**，活性病灶可顯著強化；囊變/老化的顱面病灶在 CT 出現透亮囊狀區易誤為 ABC 或惡性。**邊界清楚（sharply demarcated）有助判良性**。[^1]
- **PET 陷阱**：FD 病灶 18F-FDG／18F-NaF 攝取多變，可能模擬惡性或轉移。[^1]
- **進階序列**：DWI/ADC 與細胞密度相關，對顱骨難辨病灶可協助縮小鑑別。[^1]

### café-au-lait 與內分泌
- **café-au-lait 斑**：邊界**不規則的「coast of Maine」鋸齒緣**（對比神經纖維瘤病的光滑「coast of California」緣）、**通常不越中線**、且**與骨病灶同側（ipsilateral）**。[^2]
- **內分泌**：**性早熟為最常見內分泌病變**，**女孩較常見且發病較早**（常以陰道出血、乳房發育表現），多為**性腺刺激素非依賴型（周邊型）**；其他包括甲狀腺機能亢進、肢端肥大／生長激素過多、高泌乳素血症、磷酸鹽流失、Cushing 症候群。[^2]
- **性別與考點**：本症因性早熟以女孩多見而呈**女性為主**；故「**男性為主（male predominance）**」**非** MAS 特徵。[^2]

### 鑑別診斷（FD vs.）
| 病變 | 鑑別要點 |
| --- | --- |
| **Paget disease** | 年長者；「cotton-wool」棉絮樣外觀；骨小樑粗化[^1] |
| **Non-ossifying fibroma（NOF）** | 偏心、皮質為基底、年輕、自限性[^1] |
| **Ossifying fibroma** | 邊界較侵襲、皮質膨脹、中央較溶骨[^1] |
| **Low-grade osteosarcoma** | 皮質破壞、軟組織腫塊；**MDM2/CDK4 陽性**（FD 陰性）[^1] |
| **Adamantinoma** | 脛骨前側骨幹、多房「soap-bubble」型[^1] |

> [!note] 考點：何者「非」McCune-Albright 的特徵？
> 「**男性為主（male predominance）**」非其特徵——本症**女性為主**。三聯（性早熟、café-au-lait、polyostotic fibrous dysplasia）皆為特徵。[^2]
