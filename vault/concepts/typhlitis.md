# Typhlitis (Neutropenic Enterocolitis)

- 好發於**化療後嗜中性白血球減少症**患者（ANC < 500）
- 主要侵犯盲腸（cecum）— "typhlitis" 源自希臘文 typhlon（盲腸）
- 也可延伸至升結腸、迴腸末端
- 影像特徵：
  - CT：盲腸壁增厚（>3mm）、周圍脂肪浸潤
  - 可見腸壁內氣體（pneumatosis）
  - 腸壁水腫、腹水
- 臨床：右下腹痛、發燒、腹瀉（可血便）
- 併發症：穿孔、敗血症
- 治療：保守治療為主（抗生素＋腸道休息），穿孔則手術
