# intra-arterial-thrombectomy

**動脈內取栓術（EVT）適應症評估：ASPECTS ≥6、NIHSS ≥6、pre-stroke mRS 0-1（美國建議）、GCS 無特定門檻。mRS = 6 = 死亡，為絕對禁忌症。**

## Summary
- **ASPECTS** ≤6 預測 3 個月功能預後較差且出血風險高（美國建議 ASPECTS ≥6）。[^1]
- **NIHSS** 6-30：符合治療範圍（需 ≥8 且 ≤30 依國衛院標準）。[^1]
- **pre-stroke mRS**：美國建議 <2（即 0 或 1）；台灣無此限制。[^1]
- **mRS = 6 = 死亡**，為絕對禁忌症。[^1]
