# odontoid-fracture

**齒突骨折 Anderson-D'Alonzo 分型從上往下記：I 尖（撕脫、穩定、罕見）→ II 基部（最常見、不穩定、血供差易 nonunion）→ III 延入 C2 椎體（較易癒合）。** 判讀分水嶺一：**Type II 骨折線位於齒突「基部」橫切，是最常見且臨床最重要的型態**；判讀分水嶺二：**Type II 血供差（基部為分水嶺區）→ nonunion 風險高，常需手術固定**。記憶鉤：骨折線越低越穩定——II 在腰（基部）最危險，III 進椎體反而癒合較好。

## Summary

### 定義與流行病學
- 齒突（dens / odontoid process）骨折是 **C2（axis）最常見的骨折**，也是 **>65 歲族群最常見的頸椎骨折**；佔成人頸椎骨折約 1/5。[^2][^3]
- 年齡呈雙峰分布：年輕族群多為高能量創傷（車禍、墜落），年長族群常因低能量跌倒（osteoporosis 背景）。[^1][^3]
- 機轉為過屈/過伸暴力作用於頭頸交界（craniocervical junction）。[^1]

### Anderson-D'Alonzo 分型（依骨折線高度，由上往下）
- **Type I**：齒突『尖端』斜行/撕脫骨折，為 alar ligament 附著處撕脫；**少見、多屬穩定**；但若合併寰枕分離（atlanto-occipital dissociation）則屬不穩定，須警覺。[^1][^4]
- **Type II**：骨折線橫過齒突『**基部（waist）**』、位於 C2 椎體上方；**最常見、不穩定**。基部為相對乏血管的分水嶺區（watershed），加上骨折面積小，**癒合差、nonunion 風險最高**（2016-149 正解 D）。[^1][^2][^4]
- **Type III**：骨折線向下延伸進入 **C2 椎體（含側塊/上關節面）**；骨折面為鬆質骨、血供佳，**較易癒合**。[^1][^4]
- 補充：Type II 可再用 **Grauer 亞型**（依骨折線方向：橫向 IIA、前上斜→後下 IIB、前下斜→後上 IIC）細分，影響前路 vs 後路螺釘固定的選擇。[^5]

### X 光徵象（open-mouth odontoid view + lateral）
- **Open-mouth（張口齒突位 / peg view）**：正面評估齒突輪廓、皮質中斷與寰齒間隙（ADI）、C1 側塊與 C2 對位是否對稱。[^6]
- **Lateral**：看齒突後緣連續性、ADI、椎前軟組織腫脹（C2 前緣 prevertebral soft tissue 增厚為間接徵象）。[^6]
- X 光敏感度有限：齒突可被牙齒、枕骨或 Mach 效應重疊遮蔽；**移位不明顯或無移位的骨折在平片上可能看不到，需靠 CT 確診**。[^6]

### CT（首選 / 確診工具）
- 多排 CT 加薄層多平面重組（MPR，含矢狀面/冠狀面）是頸椎創傷骨性評估的**首選與確診工具**；高能量創傷、明顯神經學缺損、多處骨折者有絕對 CT 指徵。[^1][^7]
- CT 用於：精確定位骨折線高度以分型（決定 Anderson-D'Alonzo type I/II/III）、評估移位/成角/粉碎程度、判讀骨折面方向（Grauer 亞型）。這些都是手術 vs 保守決策依據。[^1][^5]
- **年長/骨質疏鬆病人即使 MRI STIR 陰性，仍須以 CT 形態與臨床病史判定急性骨折**（見下方 MRI 陷阱）。[^7]

### MRI（韌帶與脊髓）
- MRI 用於評估**橫韌帶（transverse ligament）/翼狀韌帶（alar）等韌帶完整性、椎前/硬膜外血腫、脊髓挫傷或受壓**，補 CT 不足；疑似 SCIWORA、中央索症候群、血管損傷時有指徵。[^1]
- **STIR 急性判讀陷阱**：STIR 高訊號（骨髓水腫）常用來判定骨折急性度，但在 **≥57 歲、特別是骨質疏鬆病人，STIR 偵測 type II 骨折的敏感度顯著下降（57 歲以上 54% vs 57 歲以下 82%）**；觀察者間一致性也隨年齡下降。故年長者 **STIR 陰性不能排除急性骨折**，須回頭對照 CT 形態與臨床史。[^7]

### 鑑別診斷與判讀陷阱
- **Os odontoideum（游離齒突小骨）**：與 type II 骨折的關鍵鑑別。os odontoideum 為與 C2 體分離的圓滑、皮質完整骨片（多被視為兒童 subdental synchondrosis 損傷後的舊 pseudarthrosis），**邊緣有皮質、間隙平滑**；急性骨折則為**邊緣銳利、無皮質的鋸齒狀骨折線**。鑑別困難時用**動態屈伸 X 光（flexion-extension）在透視下評估不穩定**。[^8][^1]
- **Mach 效應（假骨折）**：張口位上枕骨後緣、後弓或牙齒與齒突重疊，因視覺邊緣增強形成貫穿齒突基部的透亮線、模擬 type II 骨折；其特徵是延伸超出骨緣、無皮質中斷對應，必要時以 CT 釐清。[^6]
- 兒童 **subdental synchondrosis**（軟骨結合，約 5–7 歲前可見）位於齒突基部，邊緣規則、有硬化緣，勿誤判為骨折。[^6]

### 臨床重要性（為何要正確分型）
- Type II nonunion 風險高：移位、成角、粉碎、年齡大、診治延遲皆為不癒合危險因子，是手術考量的關鍵。[^4][^2]
- 統合分析顯示 >65 歲 type II 病人手術組癒合率（74%）高於保守組（40%），但保守組「穩定纖維性不癒合（stable fibrous nonunion）」常為可接受的治療目標，<5% 保守者最終需翻修手術。[^3]

> [!note] 考點：最常見的齒突骨折型?
> **Type II（齒突基部橫向骨折）**——最常見、不穩定、易 nonunion。[^1][^4]

## 何種 type 為 unstable（2018-160）
- **Tintinalli's Emergency Medicine 教材觀點**：Type II 與 Type III 皆列為 unstable（本題官方答案 E＝b+c，即 type 2＋type 3）。[^9]
- **與本概念上方 Tier 1 文獻觀點的差異（文獻分歧，並列供醫師參考）**：上方 Anderson-D'Alonzo 分型整理（來自 Acta Biomed 等 Tier 1 骨科/創傷文獻）認為 **Type III 因骨折面積大、血供佳而相對穩定、易癒合**，僅 Type II 為明確不穩定。急診教材（Tintinalli's）之分類可能較保守（將 Type III 亦視為潛在不穩定以策安全），與骨科文獻對「癒合率高」與「穩定度」的定義著重點不同。
- **暫依本題來源（Tintinalli's）標記答案為 E（b+c，即 type II+III）**，惟此與骨科文獻對 Type III 穩定性之描述有出入，建議醫師對照原始考卷與兩方文獻脈絡確認採用何種分類標準作答。
