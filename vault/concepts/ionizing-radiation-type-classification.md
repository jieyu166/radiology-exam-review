# ionizing-radiation-type-classification

**游離輻射依型態分為粒子型與電磁波型——X光與γ射線本質皆為電磁波（僅來源不同：X光來自軌域電子躍遷/制動輻射，γ射線來自原子核衰變），屬電磁波型，並非粒子型；帶電粒子(如α、β)為直接游離輻射，不帶電之光子與中子則為間接游離輻射，須先與物質作用產生荷電粒子才致游離。**

## Summary
- **粒子型游離輻射**：由具足夠動能之粒子（如α、β粒子）本身之動能或電位能，使受作用物質產生游離或制動輻射。[^1]
- **電磁波型游離輻射（X光、γ射線）**：純粹為一種能量，將能量傳遞給物質使其產生激發與游離；**X光與γ射線本質相同（皆為電磁波），僅來源不同**——X光源自原子軌域電子躍遷或制動輻射(bremsstrahlung)，γ射線源自原子核衰變。[^1]
- **直接游離輻射 vs 間接游離輻射**：[^1]
  - **直接游離輻射**：具足夠動能、碰撞時能直接引起游離的**帶電粒子**（如α、β）組成之輻射。
  - **間接游離輻射**：**光子（X光/γ）和中子**須先與物質作用（光電效應、康普吞效應、成對發生效應、核反應、非彈性散射），產生荷電粒子後，再由此荷電粒子產生游離作用。
  - **中子射線**多由核反應產生、不帶電荷，屬間接游離輻射。

> [!note] 考點：游離輻射型態分類，何者敘述為非？
> **X-ray與γ-ray型態上完全相同、屬粒子型游離輻射（A）為錯誤敘述**——X光與γ射線本質皆為電磁波（僅來源不同），屬電磁波型游離輻射，並非粒子型。粒子型輻射藉粒子動能/電位能致游離、電磁波型輻射藉能量傳遞致激發游離、中子不帶電荷多由核反應產生屬間接游離輻射，皆為正確敘述。[^1]

### 參考來源
[^1]: 官方2020詳解；Ref: 原子、輻射、與輻射防護, Ch.3, p.75（題目所引）——游離輻射分粒子型（帶電粒子動能/電位能致游離）與電磁波型（能量傳遞致激發游離）；X光與γ射線本質皆為電磁波，僅來源不同；直接游離輻射為帶電粒子、間接游離輻射為光子與中子（須先產生荷電粒子）。

## 題目

> **X-ray與r-ray型態上完全相同，具高度穿透能力，為粒子型游離輻射（A）為錯誤敘述**——X光與γ射線本質皆為電磁波（僅來源不同），屬電磁波型游離輻射，非粒子型。[^1]
