**分枝桿菌感染影像表現**涵蓋原發性、續發性、NTM、與粟粒性結核的鑑別重點。

## Summary

各類型分枝桿菌感染影像特徵：[^1]

| 類型 | 特徵 |
|------|------|
| Primary TB | Lymphadenopathy, consolidation, pleural effusion, miliary nodules |
| Postprimary TB | Apical/upper lobe consolidation, nodules, cavitation |
| TB lymphadenopathy | Low-attenuation center + **peripheral rim enhancement** |
| Classic NTM | Cavitary, 與 postprimary TB 難以區分 |
| **Miliary TB** | **血行散播（hematogenous）**，random distribution |

## 陷阱

- **Miliary TB 為血行散播（hematogenous dissemination）**，非支氣管內散播（endobronchial）。
- Miliary nodules 為 **random distribution**（非 centrilobular），因為是經血液散播。
- Endobronchial spread 會產生 **centrilobular nodules / tree-in-bud**，不是 miliary pattern。
