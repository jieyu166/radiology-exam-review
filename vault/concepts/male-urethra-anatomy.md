# male-urethra-anatomy

**男性尿道由膀胱到尿道口的順序口訣：前列腺部→膜部→Pars Nuda→球部→陰莖部。Pars Nuda是球海綿體肌的一個小肌腱性吊帶，正是球部尿道進入泌尿生殖膈延續為膜部尿道的錐狀交界點。**

## Summary
- **順序（由近端至遠端）**：**前列腺部尿道(prostatic urethra)** → **膜部尿道(membranous urethra)** → **Pars Nuda（壓迫肌，musculus compressor nudae）** → **球部尿道(bulbous urethra)** → 陰莖部尿道(penile urethra) → 舟狀窩(fossa navicularis)。[^1]
- **Pars Nuda**：為球海綿體肌(bulbocavernosus)的一個小型橫紋肌腱性吊帶，使近端球部尿道呈對稱凸出之錐狀外觀，錐尖即為球部尿道進入泌尿生殖膈(urogenital diaphragm)、延續為膜部尿道之交界點。[^1]

> [!note] 考點：尿液由膀胱排出之尿道通過順序？
> **前列腺部→膜部→Pars Nuda→球部（C）**。[^1]
