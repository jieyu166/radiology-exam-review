# ectopic-ureter

**異位輸尿管會不會漏尿，看插入點有沒有越過外括約肌遠端——女性可以，男性解剖上不行。** 女性異位口可落在陰道/前庭/遠端尿道（括約肌遠端）→ 繞過控尿機制，結果是邊正常排尿邊持續滴尿；男性異位口最遠只到後尿道或精囊（括約肌近端），括約肌還管得到，所以不漏。記憶鉤：**「女性括約肌以下有地方可以插，男性沒有」**。

## Summary
- **症狀條件**：異位輸尿管**插入點越過（遠端於）外尿道括約肌**時才會持續漏尿。[^1]
- **女性**：常見異位插入點為膀胱頸/近端尿道、陰道前庭、陰道等——**可在括約肌遠端 → 持續性尿失禁**（且仍正常排尿）。[^1]
- **男性**：異位插入點為後尿道、精囊、輸精管、射精管（**皆在括約肌近端**）→ **不會尿失禁**（反而可表現附睪炎等）。[^1]
- **F:M ≈ 10:1**；常被低估（無症狀男性）。[^1]
- **Weigert-Meyer 法則**（完全重複系統）：引流**上極**的輸尿管插入較**內下方且異位**、易阻塞；引流下極者較上外、易逆流。[^1][^2]

> [!note] 為何「女漏男不漏」
> 女性異位口可落在外括約肌**遠端**（陰道/前庭）→ 繞過控尿機制 → 持續漏尿；男性異位口永遠在括約肌**近端** → 不漏尿。[^1]

## 影像判讀

### Weigert-Meyer 法則（完全重複系統的判讀骨架）
- **上極（upper pole）輸尿管**：終止於下極輸尿管開口的**內側且下方（medial & inferior）**，且傾向**異位插入**、終點常為 ureterocele → **易阻塞**。[^2][^3]
- **下極（lower pole）輸尿管**：插入位置較**外上方（lateral & superior）**，UVJ 瓣膜機制異常 → **易膀胱輸尿管逆流（VUR）**。[^2][^3]
- 判讀記憶鉤：上極「掉得低、跑得偏內、容易塞」；下極「站得高、靠外、容易逆流」。[^3]

### 異位開口位置與「漏尿」判讀
- **女性**：異位口可落在膀胱頸、尿道、子宮/子宮頸、陰道前庭或陰道——可在外括約肌**遠端** → **持續滴尿**（約半數因括約肌下方插入而漏尿），且仍正常排尿。約 80% 女性異位輸尿管屬重複系統。[^2][^4]
- **男性**：異位口最遠只到後尿道、精囊、輸精管、射精管、攝護腺部尿道（**皆在外括約肌近端**）→ **不失禁**；攝護腺部尿道為最常見插入點，且男性常為單一系統異位、易以反覆 UTI／附睪炎表現。[^2][^4]

### 各模態影像特徵
- **超音波（首選初篩）**：可見重複腎集尿系統、上下極不對稱腎積水；阻塞的上極呈**重度腎積水、實質變薄、皮質回音增強的 dysplasia**。膀胱內可見**薄壁卵圓形囊狀無回音結構（ureterocele）**突入膀胱腔。US **無法定位異位開口**、難以描述解剖關係，常需進一步影像。[^4][^5]
- **IVU／排泄性尿路攝影**：完全重複可見雙套集尿系統；上極無功能時表現為**上極腎盞不顯影**，下極腎盞被擴張上極壓迫移位、軸線指向同側肩部 → **「drooping lily（垂萎百合）」徵象**。單純（orthotopic）膀胱內 ureterocele 在排泄期顯影呈**「cobra head／眼鏡蛇頭」徵象**（擴張的膀胱內段顯影＋薄透亮壁形成暈環）。[^3][^5][^6]
- **CT urography／MR urography**：用以顯示**異位插入點**與整體解剖；阻塞無功能的上極輸尿管在分泌期**無對比劑進入**，並可追蹤其異位終止處（如插入膀胱後下壁／尿道）。MRU＋3D 重組對雙側重複與異位插入的全貌優於 US。[^2][^5]
- **VCUG**：評估下極 VUR；ureterocele 在早期充盈時呈**膀胱內圓形充盈缺損**，膀胱充盈後可被壓扁（efface），**排尿期可外翻（eversion）而酷似膀胱憩室**。[^3][^6]

### 鑑別診斷（影像）
- **單純（orthotopic）ureterocele**：多為單一系統、成人偶見；上極功能正常、薄壁膀胱內擴張、cobra head 徵象。[^5][^6]
- **異位（ectopic）ureterocele**：合併重複腎，常見於女童；薄壁囊伴**上極腎積水／dysplasia**，囊頸位於膀胱底偏內下。[^4][^6]
- **膀胱憩室**：缺乏 ureterocele 特有的薄壁囊與和重複集尿系統的連續性，且無排尿期外翻現象——但 ureterocele 排尿期外翻時可互相混淆。[^6]

## 鑑別與報告要點（本題四選項）
| 選項 | 是否致女性持續漏尿 |
| --- | --- |
| **完全重複系統 + 異位輸尿管** | **是**（上極異位口可在括約肌遠端）[^1][^2] |
| 完全重複 + 異位 ureterocele | 否（多為阻塞/UTI，膀胱內）[^2] |
| 膀胱輸尿管逆流（VUR）| 否（UTI/逆流腎病）[^2] |
| Primary megaureter | 否（阻塞/擴張）[^2] |
