# bowed-legs-children-ddx

**兒童弓形腿(bowed legs/genu varum)最常見病因是「生理性(physiologic)」——嬰幼兒開始走路後的正常發育性內翻角度，會於2-3歲自然轉為外翻、6-7歲恢復成人型態；其餘常見病理性病因包含佝僂病(rickets)與Blount disease，但「肌肉無力(muscular weakness)」本身並非弓形腿的常見病因。** 判讀分水嶺：**發育性弓形（developmental bowing）：內翻角度於開始走路後6個月內或2歲前出現、2-3歲轉為外翻、6-7歲恢復成人型態;先天性弓形（congenital bowing）：因子宮內姿勢異常，通常脛腓骨凸向後內側、伴隨下肢長度不等長;Blount disease：脛骨近端內側幹骺端-骨骺角度>11度（正常<8度）;其他病理性病因含rickets、scurvy、neurofibromatosis type 1（脛骨前外側弓形伴腓骨發育不全）、syphilis（saber shin）。**

## Summary
- **生理性弓形（Physiologic bowing）**：**最常見**病因，為兒童開始走路後正常發育過程的一部分。[^1]
  - 發育性弓形：內翻角度於開始走路後6個月內或2歲前出現，**2-3歲轉為外翻**，**6-7歲恢復成人型態**。[^1]
  - 先天性弓形：因**子宮內姿勢異常**所致，通常脛骨/腓骨呈**凸向後內側**彎曲，生長過程可自行重塑（惟部分需支架矯正），常**伴隨下肢長度不等長**。[^1]
- **Rickets（佝僂病）**：維生素D缺乏致骨礦化不良，為病理性弓形腿常見病因。[^1]
- **Blount disease**：脛骨近端內側幹骺端生長板異常，**脛骨幹骺端-骨骺角度(metaphyseal-diaphyseal angle) >11度**（正常應**<8度**），為病理性弓形腿的重要鑑別。[^1]
- **其他病理性病因**：Scurvy（維生素C缺乏）、Neurofibromatosis type 1（典型脛骨**前外側**弓形，常伴**腓骨發育不全**）、Syphilis（saber shin，軍刀狀脛骨）。[^1]
- **關鍵排除項——Muscular weakness（肌肉無力）**：**非**弓形腿的常見病因，與弓形腿的骨骼/生長板機轉無直接關聯。[^1]

> [!note] 考點：何者「不是」兒童弓形腿的常見病因?
> **Muscular weakness（C）**——肌肉無力並非弓形腿的常見病因機轉。Physiologic（A，最常見）、Rickets（B）、Blount disease（D，脛骨幹骺端-骨骺角度>11度）皆為常見/重要病因。[^1]
