# vascular-rings-slings

**血管環/吊帶是主動脈弓或肺動脈異常走行包繞（或壓迫）氣管食道的結構；double aortic arch 是最常見的症狀性血管環，aortic coarctation 不在此類中。** 判讀分水嶺一：**最常見症狀性＝double arch；最常見變異但多無症狀＝aberrant right subclavian artery**；判讀分水嶺二：**pulmonary sling＝左肺動脈起自右肺動脈繞氣管後，coarctation 不是血管環成因**——考點就在排除 coarctation。

## Summary
- **常見成因**：[^1]
  - **Double aortic arch**：最常見**症狀性**血管環。
  - **Right aortic arch + aberrant left subclavian artery + 左 ligamentum arteriosum**。
  - **Aberrant right subclavian artery**：最常見變異,多無症狀。
  - **Pulmonary sling**：左肺動脈異常起自右肺動脈、繞氣管後（壓迫氣管）。
- **Aortic coarctation 不是血管環/吊帶的成因**。[^1]

> [!note] 考點：血管環/吊帶的常見成因「除外」何者?
> **Aortic coarctation（主動脈窄縮）**——非血管環成因。double aortic arch、right aortic arch、pulmonary sling 皆是。[^1]

## 影像判讀

### 分型：complete ring vs incomplete ring
- **Complete ring（完全環，四面包繞、多有症狀）**[^2]
  - **Double aortic arch**：最常見的「症狀性」完全環，約占完全環 50–60%（部分系列達 75%）；源自雙側第四主動脈弓持續存在，升主動脈分為左右兩弓於後方會合包繞氣管食道。[^2][^3]
  - **Right aortic arch + aberrant left subclavian artery（迷走左鎖骨下動脈）+ 左側 ligamentum arteriosum**：第二常見的症狀性完全環（約占 30%），韌帶在後方/左側補上一面完成環。[^2]
  - **Right arch + mirror-image branching + 左側 ligamentum**（接於 Kommerell 或降主動脈時亦可成環）；mirror-image 型合併心內畸形比率極高（約 90%）。[^2]
- **Incomplete ring（不完全環，僅部分壓迫、常無症狀）**[^2]
  - **Aberrant right subclavian artery（arteria lusoria，迷走右鎖骨下動脈）**：最常見的主動脈弓變異（約 0.5% 人口），多為孤立、非完全環，僅造成食道後方輕微壓痕；症狀多在成人（4–5 十年）因血管擴張/鈣化才出現。[^2]
  - **Pulmonary artery sling（肺動脈吊帶）**：左肺動脈異常起自右肺動脈，於氣管與食道之間穿行（壓迫氣管後壁、食道前壁），嚴格上屬「sling」而非環，但常與完全氣管環並存。[^2]

### CXR 線索
- **Right aortic arch**：正位片可見右側主動脈結、左側主動脈結消失；雙弓時可見「雙側主動脈結＋中線氣管受壓」。[^2]
- **氣管壓痕方向**：右側壓痕→右弓或雙弓；左側壓痕→雙弓且右弓閉鎖段；雙側壓痕→雙弓。[^2]
- **側位片**：氣管前彎、retrotracheal 軟組織增厚、局部氣管狹窄；pulmonary sling 可見肺野過度充氣（air trapping）。[^2][^3]
- CXR 異常率高（系列達 7/8、約 87.5%）；任何有 aerodigestive 症狀的孩童，縱膈輪廓都應仔細審視，勿被明顯替代病因誤導。[^3]

### 鋇劑食道攝影壓痕型態
- **後方壓痕（posterior indentation）**：double arch、right arch + aberrant LSA、或 left arch + aberrant RSA。[^2]
- **前方壓痕（anterior indentation）**：pulmonary sling（同時伴氣管後壁壓痕，為其特徵）。[^2]
- **雙側/側方壓痕（AP view）**：double aortic arch。[^2]
- Right arch + aberrant LSA 可呈「後方接續側方」兩點壓痕。[^3]
- 多視角下持續存在的固定、搏動性外壓壓痕高度提示血管環；食道攝影正常幾乎可排除診斷，但食道攝影無法直接顯示環本身。[^2][^3]

### CTA/MRA 與 3D 判讀
- **Four-artery sign（四動脈徵象）**：軸位上四條分支血管呈對稱「梯形」排列，提示異常弓分支型態、為 double aortic arch / 血管環的特徵徵象。[^2][^3]
- **Three Ds（三 D 徵象）**：Dimple（ductus 殘跡凹陷）、Diverticulum（梭形擴張即 Kommerell）、Descending aorta 位於弓的對側。[^2]
- **Arch dominance（弓優勢）判讀**：右弓優勢占約 55–70%；優勢弓決定手術入路——右弓優勢採左側開胸、左弓優勢採右側開胸（在非優勢/較小弓側離斷）。[^2]
- **Kommerell diverticulum**：迷走鎖骨下動脈起始處的球狀/梭形擴張，提示血管環；可瘤化而需切除。[^2][^3]
- **判讀技巧**：軸位定分支起源與相互關係；冠狀重組最能顯示弓優勢與雙弓同時顯影；3D volume-rendered 利於手術規劃。閉鎖段/韌帶在增強影像「不顯影」，須由相鄰血管的牽引變形或氣管受壓「推斷」其存在。[^2][^3]

### Pulmonary sling 與氣道異常（ring-sling complex）
- **Ring-sling complex**：pulmonary sling 合併完全氣管環，常造成長段氣道狹窄。[^2]
  - **Type I**：氣管與右主支氣管受壓、氣道分支正常（IA 無額外氣道異常；IB 伴 tracheal bronchus、tracheobronchomalacia）。[^2]
  - **Type II**：長段 tracheobronchial stenosis 並氣道分支異常。[^2]
- **Complete tracheal rings（完全氣管環）**：軟骨呈環形、缺後膜，氣道呈圓而窄（可窄至 2–3 mm）。[^2]
- 其他相關：bridging bronchus、tracheobronchomalacia（吐氣相氣道狹窄 >50%）；CT 吸/吐氣相可評估動態狹窄與軟化。[^2][^3]

### DDx 與判讀陷阱
- **雙弓含閉鎖段 vs right arch + ligamentum**：閉鎖弓會牽引、扭曲頸/鎖骨下動脈起源，並有局部氣道狹窄與 ductus dimple 走向可資鑑別。[^2]
- **Circumflex right arch vs 一般 right arch + 左降主動脈**：circumflex 在 supracarinal 高度於食道後突然橫越；一般右弓近橫膈處才漸進橫越。[^2]
- **Right arch + retroesophageal left brachiocephalic artery**：罕見，僅三面壓迫、**非環**；為「第一個含頸動脈分支位於弓對側」規則的例外。[^2]
- **Arteria lusoria 為孤立發現**：多無症狀、食道後方壓痕輕微、非完全環。[^2]
- **Pulmonary sling vs 右肺動脈缺如**：sling 為左肺動脈起自右肺動脈；agenesis 為右肺動脈整段缺如。[^2]
- **食道固定壓痕的非血管 DDx**：縱膈前腸重複囊腫（foregut duplication cyst）。[^3]
- 血管環合併心內/conotruncal 畸形（tetralogy、transposition、truncus）及 22q11 缺失約 12–30%（部分系列 12.4%）；echocardiography 適合查心內病灶但不適合確立或排除血管環。[^2][^3]
