# cervical-radiculopathy

**大拇指的感覺皮節主要對應 C6 神經根——「C6 管拇指、C7 管中指、C8 管小指」是頸椎神經根皮節定位的經典記憶法，大拇指酸麻刺痛應優先考慮 C6 神經根壓迫（常見於 C5/6 椎間盤突出或骨刺）。**

## Summary
- **C6 皮節對應大拇指（thumb）**：右手大拇指酸麻刺痛時，最可能壓迫的神經根為 **C6**，常見病灶位於 C5/6 節間（該節神經根為 C6 出口）。[^1]
- **上肢皮節記憶法**：C5（上臂外側/三角肌區）、**C6（拇指/食指）**、C7（中指）、C8（小指/手掌尺側）、T1（前臂內側）——依此可依麻痛分布定位受壓神經根層級。[^1]

## 判讀骨架
- **影像對應**：MRI 頸椎見對應節間（如 C5/6）椎間盤突出/骨刺壓迫神經根出口（neural foramen），並與臨床皮節分布交叉確認以定位責任節位。
