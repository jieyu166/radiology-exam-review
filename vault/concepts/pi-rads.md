# pi-rads

**PI-RADS v2 記憶鉤：「分區配序列」——周邊帶（PZ）癌用 DWI 看密度、移行帶（TZ）癌用 T2 看結構；DCE 只當二元開關（有無早期強化），MRS 已從評分中移除。** 判讀分水嶺一：**PZ → DWI/ADC 主導、TZ → T2WI 主導**；判讀分水嶺二：**DCE＝positive/negative（陽性可升 PZ DWI 3 分至 4 分）；MRS 不納入 PI-RADS v2 評分**。

## Summary
- **PZ 病灶**：**DWI/ADC 為主導序列**（決定分數）。[^1]
- **TZ 病灶**：**T2WI 為主導序列**（有時由 DWI 修飾）。[^1]
- **DCE**：**簡化為 positive／negative**;陽性（早期局灶強化）可把 PZ 的 DWI 3 分升為 4 分。[^1]
- **MRS**：**不納入 PI-RADS v2 評分**（無 ancillary 角色）。[^1]

> [!note] 考點：關於 PI-RADS v2,何者「錯誤」?
> 「**MRS 在病灶特徵化有輔助角色**」錯誤——**PI-RADS v2 不納入 MRS**。PZ 以 DWI/ADC 主導(A)、TZ 以 T2WI 主導(B)、DCE 簡化為陰/陽性(D)皆正確。[^1]

### 參考來源
[^1]: *Prostate Imaging Reporting and Data System (PI-RADS)*. Radiopaedia.org（實際查證 accessed 2026-06-18）：「**MR spectroscopy does not form part of PI-RADS scoring**」;TZ「primarily determined by the T2W score」、PZ 以 DWI 主導;DCE 簡化為 positive/negative。
