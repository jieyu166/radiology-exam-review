# intrathecal-contrast-media

**台灣核准可用於intrathecal injection（脊髓造影）之含碘對比劑僅限特定非離子型劑型——Omnipaque(iohexol) 180/240/300與Isovue-M(iopamidol) 200/300，並非所有non-ionic對比劑皆可用於此途徑，Optiray(ioversol)與Ultravist(iopromide)未列於核准清單。**

## Summary
- **核准可用於intrathecal injection之對比劑**（依2020 ACR對比劑手冊）：[^1]
  - **Omnipaque™（iohexol）**：180 / 240 / 300（nonionic）
  - **Isovue-M®（iopamidol）**：200 / 300（nonionic）
- **未核准用於intrathecal injection**：Optiray（ioversol）、Ultravist（iopromide）——雖亦屬nonionic對比劑，但未列於核准可用於脊髓腔內注射之清單。[^1]
- **記憶要點**：intrathecal用藥限「Omnipaque或Isovue-M」，其餘常見血管內對比劑（Optiray、Ultravist）不可用於此途徑。[^1]

> [!note] 考點：台灣目前哪兩種contrast medium可以用來intra-thecal injection？
> **Omnipaque（iohexol）300 / Isovue-M（iopamidol）300**——僅此二類非離子型對比劑核准用於脊髓腔內注射；Optiray、Ultravist未核准此途徑。[^1]

### 參考來源
