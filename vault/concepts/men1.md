# men1

**MEN 1 靠「3P」記：Parathyroid、Pancreas、Pituitary，三個 P 全部可能長腫瘤；臨床上副甲狀腺最常受累（~95%，多腺體增生），不是稀罕併發症——「副甲狀腺增生罕見」是考題設計的反轉陷阱。** 判讀分水嶺一：**副甲狀腺增生是最常見表現，非罕見**；判讀分水嶺二：**胰島腫瘤以 gastrinoma（Zollinger-Ellison 症候群）最常見，其次才是 insulinoma**。判讀分水嶺三：**與 MEN2 區分——MEN2 才有 medullary thyroid carcinoma + pheochromocytoma（RET 基因）；MEN1 是 MEN1 基因、無 MTC。**

## Summary
- **同義/遺傳**：又名 **Wermer 症候群**、**體染色體顯性**；侵犯腦下垂體、胰臟、副甲狀腺等內分泌腺（3P）。**滲透率（penetrance）大致：副甲狀腺機能亢進 ~94%、胃腸胰神經內分泌瘤 ~59%、垂體腺瘤 ~50%**，反映受累頻率順序。[^1][^9]
- **原發性副甲狀腺機能亢進**：**最常見表現（~95%，多腺體增生 hyperplasia）**；故「副甲狀腺增生在 MEN1 罕見」**錯誤**＝題目正解（2016-102 正解 C）。[^1]
- **胰臟神經內分泌瘤（pancreatic NET）**：**MEN1 為 panNET 最常見之遺傳症候群，佔所有 panNET 約 10%**;MEN1 中最常見者為**多發性 gastrinoma（20–60%）與非功能性腫瘤（達 80%）**，皆呈邊界清楚、強烈強化腫塊;**gastrinoma 最常見之功能性者**（Zollinger-Ellison，多發、體積小、位十二指腸黏膜下/gastrinoma triangle、多惡性易肝轉移，CT/MRI 見胃皺襞顯著增厚）;**insulinoma 為第二常見**功能性 pNET（~10–30%、發病年輕 <35 歲）;**囊性 panNET 佔 MEN1 之 panNET 達 15%**（多大、likely benign、非功能性）。**panNET 為 MEN1 疾病相關死亡之首因**。[^1][^4][^5][^8]
- **腦下垂體腺瘤**：常見 prolactinoma。[^1]
- 可合併其他內分泌腫瘤（如腎上腺、類癌 carcinoid 等）。[^1]

> [!note] 考點：何者「為錯」？
> 「**副甲狀腺增生在 MEN1 罕見**」錯——副甲狀腺機能亢進/增生反而是**最常見**表現（~95%）。Wermer 同義、胰島腫瘤多為 gastrinoma 皆正確。[^1]（2016-102）

## 各器官影像判讀
> 以下為一般放射科判讀骨架，依器官分模態整理；功能性 pNET、carcinoid 與部分腎上腺病灶可用 somatostatin receptor imaging（68Ga-DOTATATE PET/CT）偵測。

### 副甲狀腺（Parathyroid，最常見、最早出現）
- **超音波**：甲狀腺後方**橢圓形低回音結節**；為第一線（敏感度 ~72–89%），color Doppler 可提升特異度。[^2][^3]
- **99mTc-Sestamibi（SPECT/CT）**：功能性組織延遲相**滯留顯影**（abnormal focal uptake），敏感度 ~77–85%；對異位腺體有用。[^2][^3]
- **CT（4D / 三相，unenhanced+arterial+venous/delayed）**：偵測縱膈等**異位腺體**；**MRI** 為備援（T2 高訊號、無游離輻射）。[^2]
- **18F-Fluorocholine PET/CT**：敏感度高（~95%），對異位定位優於 sestamibi。[^2]

### 胰臟/十二指腸神經內分泌瘤（Pancreaticoduodenal NET）
- **CT/MRI 強化型態**：典型 pNET 為**富血管腫瘤**，多模態三相影像中以**動脈/胰相高強化**（較周邊正常胰實質為高）最具特徵，T2 高訊號、DWI 受限、ADC 偏低；分化差或纖維化者可低強化、囊變、鈣化而不典型。[^2][^4][^8]
- **多發性**：MEN1 常**多灶**、邊界清楚的小圓形腫塊；大者可因囊變/壞死/鈣化而不均質。[^2][^4][^8]
- **Insulinoma vs Gastrinoma 強化差異**：**insulinoma** 多位於胰內、小（<2 cm）、邊界銳利，典型**動脈期均勻高強化**；**gastrinoma** 相對**血供較少、纖維基質較多**，常見**延遲/持續性強化**，且較易為**胰外（十二指腸黏膜下/gastrinoma triangle）**、小（多 <0.5 cm、可達 1–3 cm）、**多發**——故 CT/MRI 偵測小 gastrinoma 敏感度受限。[^4][^5][^8]
- **EUS（內視鏡超音波）**：對小病灶敏感度最高（~93%），可補 CT 之不足。[^2]
- **肝轉移**：gastrinoma 多惡性、應評估肝臟轉移（MRI/DWI 較佳）。[^2][^4]
- **68Ga-DOTATATE PET/CT**：非 insulinoma 之 pNET 分期金標準，依 somatostatin receptor 表現，敏感度高於 CT；insulinoma 因受體表現不一可偽陰，必要時改 GLP-1 receptor 影像或選擇性動脈鈣刺激採血定位。[^2][^8]

### 腦下垂體（Pituitary）
- **MRI 為金標準**：腺瘤常 T1 等/低訊號、強化**低於正常垂體**；**dynamic post-contrast** 有助微腺瘤偵測，評估鞍區侵犯。[^1][^2][^6]
- **CT**：角色有限，用於鈣化/骨破壞或 MRI 禁忌時。[^2]

### 腎上腺與類癌（Adrenal / Carcinoid）
- **腎上腺**：多為**無功能腺瘤**；CT/MRI 評估脂肪含量與強化；功能性者可 DOTATATE 顯影。[^1][^2]
- **Thymic / bronchial carcinoid**：建議每 1–2 年胸部 CT 或 MRI 篩檢；強化多變、可侵犯。[^2]

### 與 MEN2 區別
- **MEN1**：MEN1 基因；3P（副甲狀腺、胰島、垂體），**無 medullary thyroid carcinoma**。[^1][^7]
- **MEN2**：**RET** 基因；**medullary thyroid carcinoma + pheochromocytoma**（±副甲狀腺）。看到 MTC/pheo 想 MEN2，不是 MEN1。[^7]

## 鑑別與報告要點
| 考點 | 答案 |
| --- | --- |
| 最常見、最早受累器官 | **副甲狀腺**（~95% 增生）[^1] |
| 最常見功能性 pNET | **gastrinoma**（ZES，多發/惡性/十二指腸）[^4] |
| 第二常見 pNET | **insulinoma**[^5] |
| pNET 強化型態 | **富血管、動脈/胰相高強化、可多發、易肝轉移**；insulinoma 典型均勻高強化、gastrinoma 血供較少常延遲強化[^4][^8] |
| 副甲狀腺定位影像 | 超音波 + sestamibi（±4D-CT / fluorocholine PET）[^2] |
| 功能性 NET/carcinoid 全身評估 | **68Ga-DOTATATE PET/CT**[^2] |
| 與 MEN2 區別 | MEN2＝RET＋MTC＋pheo（MEN1 無 MTC）[^7] |
