# angiography-standing-wave

**Standing arterial wave 是血管攝影中因動脈壁彈性反射產生的波狀偽影，使用 vasodilator（如 nitroglycerin）可消除，增加 filming rate、injection rate 或 contrast volume 則無效。**

## Summary
- 機轉：contrast bolus 撞擊動脈壁產生反射波 [^1]
- 消除方法：**vasodilator**（使血管舒張、降低彈性反射） [^1]
- 增加 filming rate / injection rate / contrast volume 無法消除 [^1]

### 參考來源
[^1]: 2019 交換考詳解。
