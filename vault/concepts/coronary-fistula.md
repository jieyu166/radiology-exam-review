# coronary-fistula

**冠狀動脈直接連通到心腔或大血管，形成「短路」產生左→右分流。** 起源以 RCA（~50%）最多、LAD（~42%）次之——「左冠為主」是常見陷阱。引流目的地按頻率記：RV > RA > 肺動脈幹 > 冠狀竇，皆為低壓右側結構；記憶鉤：**「血往低處流，RV 最大最低壓，故最常」**。

## Summary
- **本質**：冠狀動脈與其他心臟結構間的單一或多發瘻管連接。[^1]
- **引流部位**：**RV > RA > 肺動脈幹 > 冠狀竇**。[^1]
- **起源**：**RCA（~50%）> LAD（~42%）**；故「左冠 > 右冠」**錯誤**（2016-098 正解 C）。[^1]
- 臨床：多無症狀偶發；大瘻管可致竊血、心衰、心內膜炎。[^1]

> [!note] 考點：何者「為假」？
> 「**左冠 > 右冠**」假——起源以 **RCA（~50%）> LAD（~42%）**。單/多發瘻管、引流 RV>RA>PA>冠狀竇、左到右分流皆正確。[^1]（對應考題 2016-098，見下方 dataview）

# Note

## 影像判讀

### 解剖分類
冠狀動脈瘻管（coronary artery fistula, CAF）可依**起源冠狀動脈**、**引流部位**、瘻管數目（單一/多發）與是否合併其他異常分類。[^2]

- **起源**：多源自**右冠狀動脈系統（約 50–60%）**；左冠約 25–42%、雙側約 5%。[^2][^5] 約 **90% 引流至右側（低壓）結構**。[^3]
- **引流部位（cameral / vascular）**：可分為冠狀動脈—心腔瘻管（coronary-cameral，引流入 RV、RA、LV、LA）與冠狀動脈—血管瘻管（引流入肺動脈、冠狀竇、心臟靜脈、上腔靜脈、支氣管動脈）。[^2]
- **病因（先天 vs 後天）**：絕大多數為**先天性**（冠狀動脈終止異常）；**後天性**少見，可見於外傷、心肌梗塞、冠狀動脈繞道手術（CABG）或經皮冠狀動脈介入（PCI）後。[^5]

### 引流部位頻率
引流目的地按頻率：**右心室（RV，最常）> 右心房（RA）> 肺動脈（PA）> 冠狀竇**；引流入左心室、左心房少見。[^3] 血流走向多為**左到右分流**（高壓主動脈/冠脈系統 → 低壓右側）。[^3]

### CTA / 冠狀動脈攝影特徵
心電圖門控心臟 CTA 因高時空解析度，已成首選非侵入性檢查；侵入性冠狀動脈攝影仍可界定瘻管起源、走行與大小。[^2][^3]

- **供應冠狀動脈擴張且迂曲**（dilated, tortuous feeding coronary artery）為核心徵象。[^1]
- 可伴**動脈瘤樣擴張**（aneurysmal dilatation）。[^1]
- 以多平面重組（MPR）與容積成像（VR）清楚顯示**引流點（drainage point）**；3D 重組有助呈現複雜走行。[^1]
- 影像上**引流點位置**比起源動脈更能決定供應動脈的口徑與迂曲程度。[^1]

### 血流動力與併發症
- **Left-to-right shunt**：大瘻管使血流繞過正常微循環，造成引流心腔容積負荷。[^3]
- **冠脈竊血（coronary steal）**：瘻管遠端心肌缺血，運動或需氧增加時可致心絞痛。[^4]
- **併發症**：心衰竭、心肌缺血、心律不整（含心房顫動）、感染性心內膜炎/心內膜動脈炎、血栓與栓塞、瘻管破裂致血心包。[^4]

### 鑑別診斷
小型瘻管多無症狀偶發；大型者依大小、起源、引流部位呈現不同臨床表現，影像須與其他冠狀動脈擴張性病變（如冠狀動脈瘤）區分，關鍵為辨識出**異常交通通道與引流點**。[^2]

- **其他冠狀動脈異常**：單純冠狀動脈瘤、冠狀動脈起源異常（如 ARCAPA）須以是否存在「異常交通通道＋引流點」鑑別；瘻管特徵為迂曲擴張供血支終止於心腔/血管。[^2]
- **冠狀動脈—肺動脈瘻管（CPAF）合併先天性心臟病**：於**肺動脈閉鎖合併心室中隔缺損（PA-VSD）**，CPAF 可作為肺循環血流來源，須與主動脈—肺側支動脈（MAPCA）鑑別；此情境下 CPAF 多源自**左冠狀動脈（約 65%）**，與一般 CAF 以右冠起源為主不同。[^6]
- **後天性瘻管**：外傷、心肌梗塞、CABG 或 PCI 後出現之異常交通，須結合病史與既往影像判斷。[^5]
