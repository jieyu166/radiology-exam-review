# atlantodental-interval

**ADI 是 C1 前弓後緣到齒突前緣的距離，測的是橫韌帶還撐不撐得住——距離越大代表韌帶越鬆或已斷。** 正常上限：X 光成人 <3 mm、兒童 <5 mm（兒童韌帶本來就較鬆，考試愛考）；CT 判讀標準更嚴（成人 <2 mm）。記憶鉤：兒童上限「翻倍」（成人 3，兒童 5）。超過閾值就想：類風濕、Down 症、創傷（橫韌帶斷裂）、感染後 Grisel 症候群。

## Summary
- **前 ADI（AADI）X 光正常值**：成人 **≤3 mm**（Radiopaedia 另分 男<3／女<2.5）；**兒童 ≤5 mm**（2016-085 正解 C；兒童韌帶較鬆、可有 pseudosubluxation）。[^1][^2]
- **CT 正常值更嚴**：成人 **≤2 mm**[^1][^2]、兒童 <2.6 mm[^1]（CT 解析度高，參考值小於 X 光）。
- **後 ADI（PADI）正常 >15 mm**；PADI 縮小代表脊髓可用空間減少（RA 評估脊髓受壓重要指標）。[^1]
- 增大病因（寰樞不穩）：類風濕、僵直性脊椎炎、crystal deposition、Down 症與**先天韌帶鬆弛**、創傷（橫韌帶斷裂）、**Grisel（感染後）**、骨骼發育不良。[^1][^2]
- 動態屈伸 X 光評估不穩；MRI 看脊髓受壓與韌帶。[^1][^2]

> [!note] 考點：兒童 ADI 正常上限？
> **X 光 ≤ 5 mm**（成人 ≤ 3 mm；CT 更嚴：成人 ≤2 mm、兒童 <2.6 mm）。[^1][^2]
