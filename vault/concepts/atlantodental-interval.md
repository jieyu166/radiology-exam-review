# atlantodental-interval

**ADI（寰齒間距，C1 前弓後緣↔齒突前緣）的正常上限：成人 <3 mm、兒童 <5 mm；超過提示寰樞不穩（橫韌帶鬆弛/斷裂）。** 判讀分水嶺一：**兒童韌帶較鬆 → 上限放寬到 5 mm**；判讀分水嶺二：**>上限或屈伸動態增大 → 寰樞不穩**（RA、Down、創傷、感染後 Grisel）。

## Summary
- **前 ADI（AADI）X 光正常值**：成人**男 <3 mm、女 <2.5 mm**；**兒童 <5 mm**（2016-085 正解 C；兒童韌帶較鬆、可有 pseudosubluxation）。[^1]
- **CT 正常值更嚴**：成人 <2 mm、兒童 <2.6 mm（CT 解析度高，閾值小於 X 光）。[^1]
- **後 ADI（PADI）正常 >15 mm**；PADI 縮小代表脊髓可用空間減少（RA 評估脊髓受壓重要指標）。[^1]
- 增大病因：類風濕、Down 症（橫韌帶鬆弛）、創傷（橫韌帶斷裂）、Grisel（感染後）、骨骼發育不良。[^1]
- 動態屈伸 X 光評估不穩；MRI 看脊髓受壓與韌帶。[^1]

> [!note] 考點：兒童 ADI 正常上限？
> **X 光 < 5 mm**（成人男 < 3 mm、女 < 2.5 mm；CT 更嚴：成人 <2 mm、兒童 <2.6 mm）。[^1]
