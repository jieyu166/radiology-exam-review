# adrenal-incidentaloma-washout

**腎上腺偶然腫 CT washout protocol 僅需兩個時間點：unenhanced + 15-min delayed（或 enhanced + 15-min delayed 算 relative washout）。5 分鐘延遲不是標準 protocol 的一部分。** Adenoma 特徵：unenhanced ≤10 HU（富脂質）、absolute washout ≥60%、relative washout ≥40%。

## Summary
- **Unenhanced CT**：≤10 HU 高度提示 adenoma（富脂質 lipid-rich adenoma）。[^1]
- **Washout CT protocol**：**unenhanced → enhanced（60–70 秒）→ 15-min delayed**。[^1]
- **Absolute washout**：(enhanced − delayed) / (enhanced − unenhanced) × 100% ≥ **60%** → adenoma。[^1]
- **Relative washout**（無 unenhanced 時）：(enhanced − delayed) / enhanced × 100% ≥ **40%** → adenoma。[^1]
- **5 分鐘延遲 CT 不是標準 washout protocol 的一部分**——標準為 15 分鐘。[^1]

## 陷阱
- 「5-min delayed」不屬於標準 adrenal washout protocol。
- Lipid-poor adenoma（unenhanced >10 HU）仍可藉 washout 診斷。

### 參考來源
[^1]: 2019 交換考詳解；Dahnert Radiology Review Manual 7th ed.：adrenal washout protocol 標準為 unenhanced + 15-min delayed，absolute washout ≥60%、relative washout ≥40% 提示 adenoma。
