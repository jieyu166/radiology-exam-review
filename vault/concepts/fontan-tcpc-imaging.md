# fontan-tcpc-imaging

**Fontan/TCPC 繞過右心、讓靜脈血被動流進肺動脈，因此路徑上任何一點堵住（狹窄/血栓）都是危機；CT/MRI 要全程查：管道狹窄→fenestration→血栓→房室瓣。** 判讀分水嶺一：**被動肺血流循環 → 對阻力零容忍，吻合處狹窄＋conduit 血栓最要緊**；判讀分水嶺二：**fenestration（人為小洞降壓）＋房室瓣功能影響單心室負荷，缺一不可**。記憶鉤：從靜脈端順著走——SVC/IVC 吻合→conduit（有無洞/有無栓）→房室瓣，像巡管線一樣頭到尾看完。

## Summary

### 循環原理與術式演進
- **Fontan/TCPC 原理**：單心室生理（tricuspid atresia、HLHS、double-inlet LV 等），無 subpulmonary pump；繞過右心，把體靜脈血（SVC＋IVC）直接被動導入肺動脈。被動、低速、近層流的肺血流對任何阻力（吻合狹窄、血栓、側枝竊血）零容忍——這是全篇判讀主軸。[^1][^3]
- **術式演進**（判讀前先認術式，影像表現與併發症取向不同）：[^3][^4]
  1. **Atriopulmonary connection（APC，舊式、已淘汰）**：右心房直接接肺動脈 → 影像見巨大擴張右心房、血流停滯易血栓、心律不整。
  2. **Lateral tunnel TCPC**：心房內以人工 baffle 把 IVC 血流導向 SVC-PA 吻合。
  3. **Extracardiac conduit TCPC（現行主流）**：心房外 Gore-Tex 人工血管把 IVC 接肺動脈。
  4. **Kawashima**：IVC 中斷（heterotaxy/azygos 連續）時，SVC 經奇靜脈引流；日後常需加做肝靜脈導入以治療肺動靜脈畸形。
- **Fenestration**：lateral tunnel/conduit 上人為開約 4–5 mm 孔，於體靜脈壓升高時容許部分右→左分流以降低 Fontan 壓力、維持心輸出（代價為輕度去飽和）——影像須確認其有無與通暢狀態。[^1][^3]

### CT/MRI 影像技術要點（避開假性血栓陷阱）
- **混合不全造成的 streaming/假性充盈缺損是最大陷阱**：SVC 與 IVC 顯影劑到達時間不同，在 conduit 與肺動脈內混合不全，產生 swirling/streaming 假影，可被誤判為血栓或栓塞。[^3][^4]
- **雙側上肢＋下肢同時打顯影劑（dual-injection）**：同時經上肢與下肢靜脈（理想為兩台高壓注射器）注入，讓 SVC、IVC 顯影同步、混合均勻，是消除 streaming 假影的首選技術。[^3]
- **延遲掃描**：以延遲（靜脈期）影像確認可疑充盈缺損——延遲約 1 分鐘已可大致顯影（但仍可見輕度不均）、延遲約 3 分鐘最均勻（代價為整體對比濃度下降）；APC 舊式或心室功能差者需更長延遲。真血栓在延遲相仍持續存在，假影則消散。[^3]
- **Bolus tracking** 觸發掃描優於固定延遲，因 Fontan 流速個體差異大、難預測。[^3]

### 併發症判讀
- **Conduit/baffle 狹窄**：最常見於與肺動脈吻合處，因摺曲、鈣化或內膜增生（pseudointimal peel）所致；任何狹窄都增加被動循環阻力。[^3]
- **Conduit 血栓／內膜增生**：慢流易栓；MDCT 上見管壁低密度增厚或中央充盈缺損（須與 streaming 假影鑑別，靠延遲相確認）；報告血栓發生率約 3–19%，部分研究無症狀者亦有約 13% conduit 內附壁血栓。[^3][^4]
- **靜脈-靜脈側枝（veno-venous collateral）**：右→左分流造成去飽和/發紺，動脈早期相最易見。[^3]
- **主動脈-肺動脈側枝（aortopulmonary collateral, APC）**：源自降主動脈、鎖骨下、支氣管/肋間動脈，增加心室容積負荷、咳血風險；MRI 可量化 APC 流量。[^3]
- **肺動靜脈畸形（PAVM）**：肝靜脈「肝因子」未到達該肺區（如 Kawashima 術後）時形成，週邊肺野見異常擴張纏結血管。[^3]
- **蛋白流失性腸病（PLE）**：發生率約 3–18%、診斷後 5 年死亡率高；影像見腹水、瀰漫腸壁增厚、腸繫膜根部嚴重水腫。[^2][^4]
- **Plastic bronchitis**：淋巴循環異常造成支氣管樹內鑄型蛋白栓，與 PLE 同屬 Fontan 高壓/淋巴病變譜系。[^4]
- **Fontan-associated liver disease（FALD）**：中央肝靜脈壓升高 → 竇狀隙鬱血。[^2][^5]
  - **鬱血型態**：門脈期 mosaic/網狀（nutmeg）不均勻強化，週邊比中央明顯；尾葉相對均質肥大（似 Budd-Chiari）；肝表面結節狀、超音波粗糙回音。[^2]
  - **再生結節（FNH-like）**：成人 Fontan 約 20–30%；多發、小、右葉週邊（多在表面 2 cm 內），動脈期強化、延遲期等強化、無洗出、無 DWI 限制——惟約 10% 可見延遲洗出而擬似惡性。[^2]
  - **HCC 警訊（須與良性結節鑑別）**：間隔增大、門脈期洗出、mosaic 結構、AFP 上升、DWI 限制、輪廓異常／包膜。[^2]
  - **監測時機**：一般建議 Fontan 術後約 10 年起系統性監測（若有 Fontan failure 徵象則提早），以超音波（＋AFP）為篩檢工具；可疑病灶（>10 mm、輪廓不規則、有包膜、快速生長）以多期相 CT 或肝細胞特異性對比劑 MRI 進一步鑑別。肝硬化累積發生率術後 5/10/20/30 年約 1.3% / 9.2% / 56.6% / 97.9%。[^5]

### MRI 量測
- **Cine SSFP** 為單心室功能評估參考標準：心室容積、射出分率、4 腔相上房室瓣脫垂/逆流（影響單心室負荷）。[^3]
- **相位對比（phase-contrast）流量**：可算 Qp/Qs、APC 流量（主動脈流量減去腔靜脈流量法或加總法）。[^3]
- **延遲增強（LGE）**：偵測心肌纖維化，與較低 EF、較高心室舒張末容積指數相關。[^3]

### 完整判讀要點（題目）
- **完整判讀四要點**（題目）：[^1][^2]
  1. **SVC/IVC 到 PA 吻合處任何狹窄**（路徑狹窄增加阻力）。
  2. **conduit fenestration 之有無**。
  3. **conduit（IVC）血栓或內膜增生**（慢流易栓）。
  4. **4 腔 cine 上房室瓣脫垂/逆流**（影響單心室負荷）。
- 其他併發：蛋白流失性腸病、肝充血/纖維化、側枝/靜脈-靜脈分流、PLE。[^1]

> [!note] 考點：何者才算判讀完整？
> **以上皆是**——路徑狹窄、fenestration、conduit 血栓/內膜增生、房室瓣脫垂/逆流四者並查。[^1][^2]
