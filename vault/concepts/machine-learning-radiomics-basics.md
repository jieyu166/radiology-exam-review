# machine-learning-radiomics-basics

**機器學習(machine learning)技術涵蓋範圍廣泛，神經網路(neural networks)、決策樹(decision trees)、k-最近鄰(k-nearest neighbors)、支持向量機(support vector machine)皆屬機器學習技術之範疇，是radiomics/AI輔助診斷之基礎演算法工具。**

## Summary
- **機器學習技術涵蓋範疇**：**neural networks（神經網路）**、**decision trees（決策樹）**、**k-nearest neighbors（k-最近鄰）**、**support vector machine（支持向量機）**皆屬機器學習技術。[^1]

> [!note] 考點：何者屬於機器學習技術？
> **以上皆是（E）**——neural networks、decision trees、k-nearest neighbors、support vector machine皆屬機器學習技術範疇。[^1]
