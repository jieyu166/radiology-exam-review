# tfcc-palmer-classification

**TFCC Palmer分類分兩大類：Class 1創傷性（1a中央穿孔、1b尺側撕脫±尺骨骨折、1c遠端撕脫、1d橈側撕脫±sigmoid notch骨折）、Class 2退化性（尺腕骨衝擊症候群，依嚴重度分a-e五級）。**

## Summary
- **Class 1（創傷性損傷）**：[^1]
  - **1a**：TFC盤中央穿孔(central perforation)。
  - **1b**：**尺側撕脫(ulnar avulsion)，可合併或不合併尺骨遠端骨折(distal ulnar fracture)**；可侵犯近端或遠端lamina（分別對應foveal與styloid附著處）。
  - **1c**：遠端撕脫，侵犯ulnotriquetral與ulnolunate韌帶。
  - **1d**：橈側撕脫(radial avulsion)，可合併或不合併sigmoid notch骨折。
- **Class 2（退化性損傷，尺腕骨衝擊症候群 ulnocarpal abutment syndrome）**：依嚴重度分2a（TFCC磨損變薄未穿孔）至2e（合併尺腕關節炎）五級。[^1]

> [!note] 考點：TFCC Palmer type 1B損傷為何？
> **Ulnar avulsion with or without distal ulnar fracture（A）**——尺側撕脫，可合併或不合併尺骨遠端骨折，可侵犯近端(foveal)或遠端(styloid)lamina。1c為遠端撕脫（ulnotriquetral/ulnolunate韌帶）、1d為橈側撕脫（±sigmoid notch骨折）。[^1]
