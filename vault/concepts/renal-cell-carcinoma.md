# renal-cell-carcinoma

**腎細胞癌（RCC）最常見亞型為 clear cell RCC（約 70-80%），其次為 papillary RCC（10-15%）、chromophobe RCC（5%）。Clear cell RCC 典型影像表現為高度強化（hypervascular）腎腫瘤。**

## Summary
- **Clear cell RCC**：最常見亞型（**70-80%**），hypervascular，CT/MRI 強化顯著。[^1]
- **Papillary RCC**：第二常見（**10-15%**），hypovascular，強化較弱。[^1]
- **Chromophobe RCC**：約 **5%**，預後較佳。[^1]
- **Collecting duct carcinoma**：罕見，侵襲性高。[^1]

## 陷阱
- Clear cell RCC 為最常見亞型，非 papillary 或 chromophobe。
