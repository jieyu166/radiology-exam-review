# salter-harris

**Salter-Harris 生長板骨折五型（SALTR 記憶）。把生長板想成夾心餅乾的『夾心層』,骨折線跟它的相對位置就是分型關鍵:I Slipped（只經夾心/生長板）、II Above（夾心＋上面的幹骺端,『最常見 ~75%』）、III Lower（夾心＋下面的骨骺）、IV Through（上下貫穿骨骺-生長板-幹骺端）、V Rammed（整層被壓砸,預後最差）。最常見＝Type II。** 判讀分水嶺一：**type II ~75% 最常見**;判讀分水嶺二：**型號越高（III–V 累及骨骺/關節面）預後越差**。

## Summary

### 分型與機轉（SALTR 記憶法）
- 生長板（physis）是骺端與幹骺端之間的軟骨層，是兒童骨骼最脆弱處;骨折線相對於生長板的走向＝分型關鍵。記憶法 **SALTR**：I = **S**lipped、II = **A**bove、III = **L**ower、IV = **T**hrough、V = c**R**ush（壓砸）。[^3]
- **Type I（Slipped）**：骨折線**僅**經生長板，使骺端與幹骺端分離（epiphysis 與 metaphysis 分開）。好發於生長板較厚的**較年幼**兒童。[^1]
- **Type II（Above）**：骨折線進入生長板後**向幹骺端轉折**而出，帶走一塊三角形幹骺端骨片＝**Thurston-Holland fragment**;**最常見，約佔 74%**。[^1]（2016-189 正解 B）
- **Type III（Lower）**：骨折線進入生長板後**向骺端（epiphysis）轉折**而出，為**關節內骨折**;較 Type II 少見。[^1]
- **Type IV（Through）**：骨折線**貫穿**幹骺端-生長板-骺端三層，同時破壞生長板與關節面。[^1]
- **Type V（cRush）**：生長板受**軸向壓砸傷**，無移位的骨折線;少見、部分學者質疑其存在，可見於運動員（如體操選手）的壓力性傷害。[^1]

### 各型平片徵象（含 type I/V 易漏陷阱）
- **Type II 為平片最易辨識者**：找 **Thurston-Holland 幹骺端骨片**＝Type II 的標誌;骺端與幹骺端一同移位。[^1]
- **Type III/IV 找關節內骨折線與台階**：骨折線穿過骺端進入關節面，須評估關節面對位（articular step-off）。[^1]
- **Type I 可只見生長板增寬或骺端輕微移位**;若無移位，平片可幾乎正常，須靠軟組織腫脹與臨床壓痛點懷疑。[^1][^3]
- **Type V 幾乎都是 occult（平片初診看不到）**：常於追蹤時因臨床畸形或生長停滯的影像變化才回溯診斷。[^3]
- **判讀紀律**：平片對移位「一致性低估」;與 3D 影像比對時，移位常被低估、甚至改變分型。對位/移位程度才是預後關鍵，不可只憑平片下定論。[^1]

### MRI 角色
- MRI 可顯示平片看不到的軟骨與生長板損傷，能**判定受傷累及生長板的哪些帶（zones）**，有助理解生長板傷害程度。[^1]
- 流體敏感壓脂序列（fluid-sensitive fat-suppressed）可偵測**骨挫傷（bone contusion）**;偵測到挫傷可改變預後評估。[^2]
- 對 type I/V 等平片陰性或可疑的生長板傷害，MRI 是釐清診斷的進階工具。[^1][^3]

### 預後與生長板損傷
- **型號越高、預後越差**為一般原則：較高分型意味較高的生長障礙（growth disturbance）風險。[^2] 但 Salter-Harris 分型**本身不是預後的獨立預測因子**;年齡、血供保存、是否開放性骨折、復位方式與品質、關節內移位、固定方式與時間、以及**受累的特定生長板**都會影響預後。[^1]
- **初始移位程度與復位準確度是最重要的預後指標**（而非單看分型）。[^1]
- **生長停滯 / 骨橋（bony bar）**：Type IV 復位不良可形成**跨生長板骨橋（transphyseal bony bar）**，造成不對稱生長或生長畸形;Type IV 並有**完全生長停滯**風險。Type III/IV 累及關節面者另有**創傷後關節炎**之虞。[^1]
- 量化參考：關節內 distal radius 骨折研究中，**生長停滯發生率達 43%**（SH-IV 較 SH-III 多），顯示關節內生長板傷害的高生長障礙風險。[^4]

### 分型延伸（Peterson／Ogden）
- **Peterson 分類**另加兩型：Type I（幹骺端骨折延伸至生長板）與 Type VI（生長板部分缺損，如除草機等開放性傷害）。[^1]
- **Ogden 分類**細分為九型並含次分型，依生長停滯風險分層。[^1]

> [!note] 考點：最常見的 Salter-Harris 型?
> **Type II（約 74%，by far the most common）**——經生長板+幹骺端，帶走 Thurston-Holland 骨片。[^1]
