# persistent-lsvc

**PLSVC 是最常見的胸腔先天靜脈異常，核心記憶：「左側縱膈血管影 → 追到 dilated coronary sinus → 問它流去哪」。** 約 90% 經 coronary sinus 引流入右心房（無症狀偶然發現）；少數直接入左心房（多伴 unroofed coronary sinus）才產生右→左分流。判讀分水嶺一：**引流路徑決定臨床意義**——coronary sinus → 右心房（最常見、無分流）vs 直接入左心房（少數、可致反常栓塞）；判讀分水嶺二：**dilated coronary sinus 是影像線索，見之需排除 PLSVC**。

## Summary

### 胚胎與解剖
- **最常見的胸腔先天靜脈異常**；一般族群盛行率約 **0.3–0.5%**，先天性心臟病（CHD）族群可達 **約 10%**。[^1][^2]
- **胚胎來源**：正常時 **left anterior cardinal vein** 退化形成 coronary sinus 與 ligament of Marshall；其**未閉合／持續存在**即形成 PLSVC。[^1]
- **多數仍保留右側 SVC**（雙側 SVC）；**isolated PLSVC（右側 SVC 缺如）**約佔 PLSVC 的 10–20%（CHD 族群整體約 0.09–0.13%），此時全身上半身靜脈回流全靠左側 SVC。[^2][^3]
- **bridging（left brachiocephalic／innominate）vein**：存在雙側 SVC 時，左右 SVC 間的橋接靜脈常**缺如或細小**（左無名靜脈不發育）；判讀時兩側 SVC 並存而無正常左無名靜脈跨越為線索。[^4]

### CT/MR 辨識路徑（追到 dilated coronary sinus）
- **管狀血管位於主動脈弓**外側／左前方，起自左 subclavian–jugular 匯合處，下行**經左肺門前方**，最終匯入後縱膈的 **dilated coronary sinus → 右心房**。[^4]
- **dilated coronary sinus** 是最敏感的間接線索；可在左心房後下方呈擴大的圓形結構，**初看可誤為心旁腫塊（paracardiac mass）**，須在連續軸切上把左縱膈血管一路追到 coronary sinus 才能定論。[^5]
- 確診靠**對比劑 CT/MR venography**＋連續軸切追蹤血管走行；冠狀面（尤其 LAO 斜冠狀切）有助顯示匯入 coronary sinus 的終末段。[^5]

### 變異組合與臨床意義
- **多無症狀、偶然發現**；引流入右心房者無血流動力學影響。[^1][^4]
- **直接入左心房 → 右→左分流**：常透過 **unroofed coronary sinus**（coronary sinus 頂壁缺損，與左心房相通）造成去氧血入體循環，**風險為反常栓塞（腦膿瘍／腦栓塞）、發紺、低血氧**。[^6]
- **介入／手術陷阱**：左側中心靜脈導管、PICC、Swan-Ganz、節律器／去顫器導線經左側放置時，會循 PLSVC → coronary sinus → 右心房，**走行異常**且增加導線置放與心律不整風險；心臟手術 **retrograde cardioplegia 經 coronary sinus 給藥**時，若有未結紮的 PLSVC 會使心肌保護液分流、保護不全，**術前須辨識並計畫處理**。[^1][^4]
- PLSVC 與 CHD（ASD、endocardial cushion defect、TOF）、竇房結／傳導異常（如 sick sinus syndrome）相關，尤其 isolated PLSVC。[^3]

### DDx（左上縱膈血管影）
- **left superior intercostal vein（aortic nipple）**：走行**位於主動脈弓後方**、口徑小（正常 <約 4.5 mm），不下行至 coronary sinus。[^5]
- **PAPVR 的 vertical vein**：血流方向為**向上（cephalad）**、源自肺靜脈，**左主支氣管前方無血管**；PLSVC 則血流向下且匯入 coronary sinus。[^5]
- **levoatriocardinal vein**：**幾乎都伴左心阻塞性病變**，位置偏後（左肺動脈後方）。[^5]
- **pericardiophrenic vein**：下行至橫膈、走行於心臟外側，口徑細小。[^5]
- **aberrant left brachiocephalic vein**：**穿行於主動脈弓下方／後方**，與 TOF 相關。[^5]

### 判讀陷阱
- 見 **dilated coronary sinus** 切勿先想心旁腫塊或心包積液——先把左縱膈血管追到底排除 PLSVC。[^5]
- **不是所有 PLSVC 都無害**：務必確認引流終點（右心房 vs 左心房）；入左心房／合併 unroofed coronary sinus 才有右→左分流意義。[^6]
- 看到左側中心導管／節律器導線走行偏左下異常路徑，反向提示可能存在 PLSVC。[^4]

> [!note] 考點：何者「錯」？
> 「**通常匯入左心房**」錯——PLSVC **通常經 coronary sinus → 右心房**（約 90%）；入左心房屬少數（約 10%，多伴 unroofed coronary sinus → 右→左分流）。其餘（最常見胸腔先天靜脈異常、dilated coronary sinus、源自 left anterior cardinal vein）皆正確。[^1]
