# ultrasound-image-orientation

**超音波影像方向：探頭 marker 永遠對應螢幕左側；縱切時 marker 朝頭 → 螢幕左＝cranial；螢幕頂＝靠探頭（淺）、底＝深。** 記憶鉤：「marker 在哪，螢幕左邊就跟著去」；縱切 marker 朝頭，所以螢幕左邊是頭那側（cranial）。判讀分水嶺：**縱切 marker 朝頭 → 螢幕左＝cranial、右＝caudal；橫切 marker 朝病人右側 → 螢幕左＝病人右側（同 CT/X 光慣例）；深度軸：頂淺底深**。

## Summary
- **指標—螢幕對應**：探頭上的 orientation marker 對應顯示於**螢幕左側**。[^1]
- **縱切（longitudinal）**：marker 朝**病人頭側（cephalad）** → **螢幕左側＝cranial（頭側）、右側＝caudal（足側）**。[^1]
- **橫切（transverse）**：marker 朝病人右側 → 螢幕左側＝病人右側（如同看 CT/X 光的慣例）。[^1]
- **深度**：螢幕**頂端靠近探頭（淺層）**、底端為深層。[^1]

> [!note] 考點：腹部縱切時,螢幕左側（操作者左手側）顯示哪個方向?
> **Cranial（頭側）**——縱切 marker 朝頭、對應螢幕左側,故左＝cranial、右＝caudal。[^1]

### 參考來源
[^1]: 超音波影像方向標準慣例（實際查證 accessed 2026-06-18，123sonography「Ultrasound 101 - Image Orientation」、PMC9619033「Ultrasound scanning techniques」）：「if the marker on your probe points to the patient's head, the **screen's left side will be cranial**」;指標側對應螢幕左側、頂端靠近探頭。
