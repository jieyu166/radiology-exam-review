# anti-nmda-encephalitis

**抗 NMDAR 腦炎好發『年輕女性（與兒童）』，以精神症狀＋癲癇/運動異常表現，常合併卵巢畸胎瘤、MRI 多正常或非特異——『老年女性好發』是錯的。** 判讀分水嶺一：**年輕、女性為主（F:M≈4:1）**；判讀分水嶺二：**找卵巢畸胎瘤**（早切除預後較佳），影像常正常故診斷靠抗體。

## Summary
- **族群**：好發**年輕患者、尤其年輕女性**（F:M≈4:1）；為第二常見自體免疫腦炎。[^1]（故「老年女性好發」**錯誤**＝題目正解）
- **臨床**：常先有病毒樣前驅期，續以**顯著精神症狀**、續發**癲癇/運動異常**、自主神經不穩。[^1]
- **腫瘤關聯**：**卵巢畸胎瘤**（約 40–60%，年輕女性比例更高，多為成熟畸胎瘤）；**早期切除腫瘤 → 較佳預後**。[^1]
- **影像**：**MRI 常正常**或僅非特異 T2/FLAIR 高訊號，與症狀**相關性不佳**；診斷靠 CSF/血清抗 NMDAR 抗體。[^1]

> [!note] 考點：何者為「假」？
> 「**老年女性**好發」為假——好發**年輕女性**（F:M≈4:1）。其餘（精神症狀/癲癇、卵巢畸胎瘤>近半且早切預後佳、MRI 非特異）皆真。[^1]
