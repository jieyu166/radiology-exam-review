# anti-nmda-encephalitis

**年輕女性突然行為異常＋癲癇，而 MRI 幾乎看不出來——抗 NMDAR 腦炎的特徵就是「臨床鬧很大、影像很正常」。** 好發年輕患者，女性為主（F:M≈4:1），「老年女性好發」是考試最常出現的錯誤選項。關鍵搜查：腹/骨盆超音波或 MRI 找卵巢畸胎瘤（約 40–60%），早期切除腫瘤可改善預後；確診靠 CSF 或血清抗 NMDAR 抗體，不靠影像。

## Summary
- **族群**：好發**年輕患者、尤其年輕女性**（F:M≈4:1）；為第二常見自體免疫腦炎。[^1]（故「老年女性好發」**錯誤**＝題目正解）
- **臨床**：常先有病毒樣前驅期，續以**顯著精神症狀**、續發**癲癇/運動異常**、自主神經不穩。[^1]
- **腫瘤關聯**：**卵巢畸胎瘤**（約 40–60%，年輕女性比例更高，多為成熟畸胎瘤）；**早期切除腫瘤 → 較佳預後**。[^1]
- **影像**：**MRI 常正常**或僅非特異 T2/FLAIR 高訊號，與症狀**相關性不佳**；診斷靠 CSF/血清抗 NMDAR 抗體。[^1]
- **誘發因子（trigger）**：除卵巢畸胎瘤外，**單純疱疹病毒腦炎（HSE）亦為 NMDAR 自體免疫的已知誘發因子**（post-HSV-encephalitis autoimmune encephalitis）。[^2]
- **年齡分布**：約 **37% 患者發病時 <18 歲**（兒童/青少年亦不少）；女性為主（F:M≈8:2≈4:1）。[^2]
- **治療與預後**：約 **80% 經免疫治療（必要時切除腫瘤）後改善，但恢復緩慢**；重症常需加護照護、目前無特異預後生物標記；自體抗體既是診斷標記、也直接改變 NMDAR 突觸傳遞。[^2]

> [!note] 考點：何者為「假」？
> 「**老年女性**好發」為假——好發**年輕女性**（F:M≈4:1）。其餘（精神症狀/癲癇、卵巢畸胎瘤>近半且早切預後佳、MRI 非特異）皆真。[^1]
