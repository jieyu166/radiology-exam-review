# biliary-atresia

**膽道閉鎖影像評估：見到完整肝外膽管可排除；鑑別包括膽道閉鎖與新生兒肝炎；US/scintigraphy/MRI 均有助診斷；確診需術中膽道攝影（非 MRCP）。**

## Summary
- 完整肝外膽管 visible → **可排除膽道閉鎖** [^1]
- 鑑別：biliary atresia vs neonatal hepatitis [^1]
- 診斷工具：US（triangular cord sign）、hepatobiliary scintigraphy（Tc-99m IDA）、MRI [^1]
- **確診**：**intraoperative cholangiogram**，非 MRCP [^1]
- MRCP 看不到肝外膽管不能確診（可能為技術限制）[^1]
