# cns-opportunistic-infection

**遇到環狀強化病灶，先想 DWI：膿液黏稠擴散受限（高 DWI/低 ADC），壞死腫瘤不受限——一個序列就能初步分流。** AIDS 患者的 CNS 伺機性感染，首選排除 toxoplasmosis（最常見），典型表現是多發環狀強化、好發基底節、eccentric target sign。記憶鉤：「膿瘍黏→DWI 亮；弓蟲找 AIDS→基底節環」，兩個考點串成一個判讀流程。

## Summary
- **DWI 區分環狀強化病灶**：**化膿性膿瘍**中心膿液**擴散明顯受限（DWI 高、ADC 低）**;**壞死/囊變腫瘤**中心多**不受限**——故 DWI 為鑑別最佳序列(option A 正確)。[^1]
- **CNS 黴菌病**：多為**免疫低下者的伺機性感染**(option B 正確)。[^1]
- **弓蟲症（toxoplasmosis）**：**AIDS/免疫低下患者最常見的中樞神經伺機性感染**;典型多發環狀強化、好發基底節,「eccentric target sign」(option C 正確)。[^2]
- 三者皆正確 → 故「何者正確」答 **D（以上皆正確）**（2016-184 正解 D）。

> [!note] 考點
> DWI 為環狀強化病灶鑑別最佳序列、CNS 黴菌多伺機性、**toxoplasmosis 為 AIDS 最常見 CNS 伺機感染** → 三者皆正確（答 D）。[^1][^2]

## 判讀骨架：免疫低下 CNS 感染的影像分流

### 一、環狀／結節強化病灶：toxoplasmosis vs primary CNS lymphoma（核心鑑別）

| 特徵 | Toxoplasmosis | Primary CNS lymphoma（PCNSL） |
|---|---|---|
| 部位 | 多發，**基底節／灰白質交界**為主[^2] | **室周（periventricular）／胼胝體**，常**室管膜下擴散（subependymal spread）**[^3] |
| 強化型態 | **環狀**或結節強化；**eccentric target sign**（偏心強化核心＋中間低訊號＋外環強化）[^4] | 在 AIDS（免疫低下）常呈**環狀**強化（壞死所致）；免疫健全者多為**均勻實質強化**[^3] |
| 數目 | 通常多發 | 可單發或多發 |
| 非顯影 CT | — | 病灶可呈**高衰減（hyperattenuation）**，反映高細胞密度[^3] |
| 擴散（DWI/ADC） | 核心擴散多**不明顯受限**（不同於化膿性膿瘍）[^4] | 高細胞密度 → **擴散受限、ADC 偏低**；惟與 toxo **ADC 有重疊**，不可單獨判讀[^5] |
| Thallium-201 SPECT／FDG-PET | **低／無攝取（代謝低下）**[^5] | **高攝取（高代謝）**[^5][^6] |
| 類固醇反應 | 對 anti-toxo 治療反應 | 對類固醇常**暫時消退（vanishing tumor）**，但會復發[^3] |

- **判讀分水嶺**：DWI／ADC 兩者重疊，**單靠擴散無法可靠區分** toxo 與 PCNSL[^5]；**Thallium-201 SPECT 或 FDG-PET 的代謝高低（lymphoma 高、toxo 低）較具鑑別力**[^5][^6]，必要時輔以灌注（PCNSL rCBV 通常不高，異於高惡性度膠質瘤）、MRS 與 **anti-toxoplasma 經驗性治療試驗（2 週影像未改善偏向 lymphoma）**作為臨床決策依據。[^3][^5]
- **陷阱**：toxo 與 PCNSL 影像（環狀強化＋擴散受限）可重疊；type sign（target sign）非 toxo 專屬，淋巴瘤亦可偽裝。[^4]

### 二、白質與室管膜病變（多不強化或線狀強化）

- **PML（JC 病毒）**：**不對稱**白質病變，**T2/FLAIR 高、T1 低**，**典型不強化、無腫塊效應（no mass effect）**，**侵犯皮質下 U-fiber** 形成與皮質間銳利邊界，好發頂枕葉；約 90% 病例呈此典型表現。[^7]
- **CMV 腦炎／室管膜炎**：**室管膜下／室周線狀 T2/FLAIR 高訊號與線狀強化（ependymitis/ventriculitis）**，室管膜面 DWI 受限為較敏感徵象；常見於 **CD4 < 50** 之 AIDS 患者。[^8]
- **HIV 腦病變（HIV encephalitis）**：對稱性室周／深部白質 T2 高訊號、瀰漫性腦萎縮，**不強化、無腫塊效應**，為排除性鑑別。[^7]

### 三、肉芽腫／腦膜為主病變

- **Cryptococcus（隱球菌）**：**基底節／灰白交界沿 Virchow-Robin 腔之膠樣假囊（gelatinous pseudocysts）**，呈 T2/FLAIR 高訊號擴大的血管周隙、通常**不強化或輕度強化**；可合併**基底膜腦膜炎**與腦積水。[^9]
- **結核（TB）**：**基底膜腦膜強化（basal meningeal enhancement）**為特徵，伴 tuberculoma（環狀或結節強化、可中心 T2 低訊號）與交通性腦積水、梗塞。[^10]

### 四、免疫重建發炎症候群（IRIS）

- 抗反轉錄病毒治療（ART）啟動後 CD4 回升，原本沉默的感染轉為**發炎性惡化**：**PML-IRIS** 病灶出現**新發強化與腫脹（mass effect）**（與典型 PML 不強化相反）；**Cryptococcal-IRIS** 出現腦膜或實質強化加劇。為治療後影像惡化之重要鑑別。[^11]
