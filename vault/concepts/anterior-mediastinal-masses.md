# anterior-mediastinal-masses

**前縱膈腔腫瘤口訣「4 T's」：Thymoma（胸腺瘤）、Teratoma/生殖細胞瘤、Thyroid（胸骨後甲狀腺腫goiter）、Terrible lymphoma（淋巴瘤）。神經源性腫瘤（neurogenic tumor）好發於「後縱膈」（脊椎旁），不是前縱膈——這是本題陷阱。**

## Summary
- **前縱膈腔常見腫瘤（4 T's）**：[^1]
  - **Thymoma / thymic tumors**（胸腺瘤）：前縱膈最常見原發腫瘤。
  - **Teratoma / germ cell tumors**（畸胎瘤/生殖細胞瘤）。
  - **Thyroid**（胸骨後甲狀腺腫 goiter）。
  - **Terrible lymphoma**（淋巴瘤）。
- **非前縱膈（本題陷阱）**：**Neurogenic tumor（神經源性腫瘤）好發於後縱膈**（脊椎旁溝，源自交感神經節/神經鞘），非前縱膈。[^1]

> [!note] 考點：何者「不是」常見的前縱膈腔腫瘤？
> **Neurogenic tumor（B）**——神經源性腫瘤好發於後縱膈(脊椎旁)。Thymoma、goiter(胸骨後甲狀腺腫)、lymphoma、teratoma皆為前縱膈常見腫瘤(4 T's)。[^1]

### 參考來源
[^1]: Radiology review manual, 6th ed., p.443（2018 交換考題詳解 p.284 引用；Tier 2 教科書）。前縱膈常見腫瘤為4 T's：thymoma、teratoma/germ cell、thyroid(goiter)、terrible lymphoma；neurogenic tumor好發於後縱膈(脊椎旁溝)。
[^2]: *Thoracic Imaging*, p.214（2018 交換考題詳解 p.328 引用；Tier 2 教科書）。心膈角(cardiophrenic angle)好發pericardial cyst、Morgagni hernia、lymph node masses、脂肪墊；neurogenic tumor好發後縱膈脊椎旁溝，非心膈角/前縱膈。

## Cardiophrenic angle（心膈角）好發病灶（2018-416）
- 前縱膈/心膈角(cardiophrenic angle)好發：**pericardial cyst(心包囊腫)、Morgagni hernia(前橫膈疝)、lymph node masses(淋巴結腫，如lymphoma)、心膈角脂肪墊**。[^2]
- **Neurogenic tumor(神經源性腫瘤)不好發於心膈角/前縱膈**——好發於**後縱膈(脊椎旁溝)**。[^2]
