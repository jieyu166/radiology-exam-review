# radiation-dose-reference

**常見檢查有效劑量（mSv）約略排序：胸部X光 ~0.02 < 乳房攝影(數位,每次)~0.2–0.4 < 骨盆/全脊椎X光 ~0.6–1.5 < 頭部CT ~2 < 鋇劑灌腸 ~6–8 < 腹部骨盆CT ~7–8。** 判讀分水嶺一：**X光 << CT；特殊攝影（鋇劑灌腸）劑量可媲美/超過頭部CT**；判讀分水嶺二：**鋇劑灌腸在『X光骨盆/頭CT/全脊椎X光』四者中劑量最高**。

## Summary
- **乳房攝影（mammography）**：數位約 **0.2 mSv/study**；雙側 4-view 有效劑量約 **0.4–0.7 mSv**（2016-116 正解 C；原選項小數遺失,應為 0.007/0.07/**0.7**/7）。[^1][^2]
- **頭部 CT**：約 **2 mSv**。[^1]
- **鋇劑灌腸（barium enema，下消化道）**：約 **6–8 mSv**——四選項（骨盆X光~0.6、頭CT~2、鋇劑灌腸~6、全脊椎X光~1.5）中**最高**（2016-121 正解 C）。[^1]
- **骨盆 X 光** ~0.6 mSv、**腰椎/全脊椎 X 光** ~1.5 mSv、胸部 X 光 ~0.02 mSv、腹部骨盆 CT ~7.7 mSv。[^1]

> [!note] 考點
> 四檢查中**鋇劑灌腸（~6–8 mSv）劑量最高** > 頭部CT(~2) > 全脊椎X光(~1.5) > 骨盆X光(~0.6)。乳攝 4-view ~0.4–0.7 mSv。[^1][^2]

> [!warning] 2016-116 選項小數遺失
> 資料庫選項顯示 007/07/7/7,係小數點遺失;原選項應為 **0.007 / 0.07 / 0.7 / 7 mSv**,正解 **C＝0.7 mSv**。

### 參考來源
[^1]: *Typical Radiation Doses*（MSD Manual Professional Edition，本於 Mettler 等）（實際查證 accessed 2026-06-17）：數位乳攝 0.21、CT 頭 2、**鋇劑灌腸（下消化道）6**、腰椎X光(側位)1.5、CT 腹骨盆 7.7 mSv。
[^2]: Mettler FA, Huda W, Yoshizumi TT, Mahesh M. *Effective Doses in Radiology and Diagnostic Nuclear Medicine: A Catalog*. Radiology. 2008;248(1):254-263. https://doi.org/10.1148/radiol.2481071451 （乳房攝影有效劑量約 0.4 mSv;一般X光 0.01–10、CT ~2–20 mSv 範圍）。

## 題目
> [!question]- 骨盆X光、頭部CT、鋇劑灌腸、全脊椎X光中,何者有效劑量最高?
> **鋇劑灌腸（~6–8 mSv）** > 頭部CT(~2) > 全脊椎X光(~1.5) > 骨盆X光(~0.6)。[^1]

> [!question]- 一般乳房攝影（4-view）的有效劑量約為?
> 約 **0.4–0.7 mSv**（數位每次 ~0.2 mSv）。[^1][^2]
