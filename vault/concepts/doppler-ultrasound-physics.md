# doppler-ultrasound-physics

**都卜勒頻移(Doppler shift)與操作頻率(operating frequency)成正比——依都卜勒方程式,提高系統的操作頻率會直接增加偵測到的頻移量,是增加系統對都卜勒頻移敏感度最直接有效的方法。** 判讀分水嶺：**都卜勒方程式 Δf = 2f₀v·cosθ/c，Δf 與 f₀（操作頻率）成正比——提高操作頻率直接增加Δf(頻移),故最能增加系統對頻移的敏感度;調整都卜勒角度、重新定位取樣容積(sample volume)、改變取樣閘(sample gate)大小屬於影像品質/訊號擷取優化,非直接增加頻移本身的物理量。**

## Summary
- **都卜勒方程式核心**：都卜勒頻移 **Δf = 2f₀v·cosθ / c**（f₀=操作頻率、v=血流速度、θ=都卜勒角度、c=聲速）。[^1]
- **增加操作頻率（operating frequency）**：**直接增加Δf**（頻移量與f₀成正比），是增加系統對都卜勒頻移敏感度**最直接有效**的方法。[^1]
- **其他選項的角色**：[^1]
  - **增加都卜勒角度（increasing angle）**：cosθ隨角度增加而變小（θ越接近90°，cosθ→0），會**降低**而非增加頻移偵測;理想都卜勒角應盡量小（<60°）。
  - **重新定位取樣容積（repositioning sample volume）**：影響取樣位置準確性，非直接增加頻移敏感度。
  - **改變取樣閘大小（sample gate size）**：影響空間解析度與訊號強度，非直接增加頻移量。

> [!note] 考點：何者最可能增加系統對都卜勒頻移的敏感度?
> **Increasing the operating frequency（C）**——依都卜勒方程式，頻移與操作頻率成正比。增加都卜勒角度反而使 cosθ變小、降低頻移;重新定位取樣容積、改變取樣閘大小非直接增加頻移的方法。[^1]
