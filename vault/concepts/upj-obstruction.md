# upj-obstruction

**解剖擴張 ≠ 功能性阻塞。** UPJ obstruction 的判讀核心,是判斷「影像所見的腎盂腎盞擴張,是否真的對應到生理上的尿流阻塞」。形態(超音波 / CT 的擴張)負責提出懷疑,功能(diuretic renogram)負責確診。記住「腎盂腎盞擴張、但輸尿管不擴張」這個 UPJ level 的指紋型態,以及「成人要找 crossing vessel、但血管鄰近不等於壓迫」這兩個判讀分水嶺。

## Summary

- **定義**:尿流在 renal pelvis → proximal ureter 交界處受阻;先天性 / 胎兒腎盂腎盞擴張最常見的原因。[^1][^2]
- **病因**:先天性內在性(輸尿管肌肉層異常、再管腔化不全、蠕動波傳遞失敗)最常見;成人與較大兒童常見 crossing vessel 參與(各來源比例差異大,見內文);少數為後天(術後、發炎、結石、腫瘤)。[^1][^2][^5]
- **影像指紋**:UPJ 層級以上明顯 hydronephrosis,輸尿管不擴張(腎盂呈氣球狀 / 倒淚滴狀擴張、近端輸尿管塌陷)。[^1][^2][^4][^5]
- **功能確診**:diuretic renogram(MAG-3 首選 / DTPA 次之)為功能性阻塞黃金標準。[^1][^2][^3]
- **成人術前**:CTA / CT urography(2D/3D 後處理)或 MR renography 評估 crossing vessel 與輸尿管—血管空間關係。[^5][^6][^9]
- **易混淆觀念**:①擴張 ≠ 阻塞;②腎功能正常 ≠ 沒有阻塞;③血管鄰近 UPJ ≠ 真正壓迫。[^1][^2][^5]

## 定義與術語

UPJ obstruction(又稱 pelviureteric junction obstruction, PUJO)指尿流在腎盂與輸尿管交界處受功能性或解剖性阻塞,是先天性 / 胎兒腎盂腎盞擴張最常見的原因。產前超音波下約影響 1:500 胎兒;**男性較多**[^1];以左側、單側為主(單側 70–90%、雙側 10–30%、Left > right 約 2:1)。[^1][^2]
（註:來源僅敘述「more common in males」,未提供確切男女比;STATdx 未明列性別優勢。）

**判讀的核心決策**:面對疑似 UPJ obstruction 時,最重要的是判斷「影像所見是否與生理(功能性)阻塞相符」——解剖擴張不等於功能性阻塞,這是整個診斷邏輯的骨幹。[^1][^2]

## 病因

### 先天性(intrinsic,最常見)

胚胎期 UPJ 區再管腔化不全,造成輸尿管肌肉層數量與走向異常,使蠕動波(peristaltic wave)無法有效傳遞通過交界處,形成「無蠕動節段(adynamic segment)」;為新生兒最典型的機轉。組織學上見膠原過多、肌肉缺乏、縱向肌肉過多。[^1][^2][^4]

### Crossing vessel(外在壓迫;成人與較大兒童重要)

異常極血管(aberrant polar vessels)可從 renal artery 主幹近端或直接從主動脈分出,跨越 UPJ。各來源對其角色與比例看法不一:Radiopaedia 約 40%、Grainger & Allison 約 10%、STATdx 認為血管很少是真正病因但術前必須考慮其與 UPJ 的關係。[^1][^2][^3]

> [!warning] 因果方向的陷阱
> RadioGraphics 的 3D MDCT 研究指出,跨越血管常位於 UPJO 管徑移行點的上方(cephalad),與移行點無直接關係,因此其致病角色受質疑——血管「鄰近」UPJ ≠ 真正壓迫,報告措辭宜保守。[^5]

> [!note] crossing vessel 的手術意涵（為何術前要找）
> 此點屬泌尿科手術文獻、非影像來源所涵蓋:**crossing vessel 主要降低 endopyelotomy(內視鏡切開)成功率**——內視鏡治療成功率約 90%,存在 crossing vessel 時降至約 40%,且未察覺之血管可致術中大出血[^12]。相對地,**dismembered / 腹腔鏡或機器人 pyeloplasty** 受影響較小、在有 crossing vessel 時成功率較佳(可將 UPJ 重建至血管前方)[^13]。故術前以 CTA 偵測 crossing vessel 對術式選擇至關重要。

### 後天性(成人)

術後 / 外傷後疤痕、緊鄰 UPJ 遠端的阻塞性結石、腎盂炎後疤痕、內在性惡性(上泌尿道尿路上皮癌)、外在壓迫 / 纖維化、主動脈瘤等。[^1][^2]

## 臨床表現

- **成人**:間歇性側腹痛(intermittent flank pain),大量飲水後加劇;經典術語「beer drinker's hydronephrosis」。[^1]
- **小兒**:疼痛、噁心嘔吐、外傷後血尿,或腹部腫塊;即使輕微外傷也有較高腎損傷風險。[^1][^2]
- **新生兒**:多為產前超音波發現的無症狀 hydronephrosis,或可觸及腫塊。[^1][^2]

## 影像診斷

### 超音波(第一線;新生兒 / 兒童首選)

最具特徵性的所見:UPJ 層級以上明顯 hydronephrosis,但輸尿管不擴張——「腎盂腎盞擴張但輸尿管正常口徑」是與下游阻塞鑑別的關鍵;腎盂呈氣球狀擴張,嚴重者見腎皮質變薄、對側代償肥大。評估範圍應涵蓋腎盞系統、renal pelvis、UPJ 及盡可能多的近端輸尿管;可用腎動脈 resistive index 嘗試區分阻塞性與非阻塞性 hydronephrosis。產前定量:AP 腎盂徑 > 10 mm、腎盂:AP 腎徑比 > 0.5。[^1][^2][^4]

### 核醫 — Diuretic renogram(功能性診斷黃金標準)

超音波顯示 hydronephrosis 且 VCUG 排除 reflux 後,做利尿腎掃描(MAG-3 首選;DTPA 次之)量化相對腎功能並界定阻塞程度。

- 顯著阻塞 → 利尿劑前後同位素皆不明顯排出(腎圖無向下斜降、示蹤劑滯留)。
- 非阻塞性擴張 → 利尿劑後排空改善(腎圖向下斜降)。

> [!note] DTPA 非首選
> DTPA 以腎絲球過濾為主,而阻塞性尿路病變中腎絲球功能下降較早較快;MAG-3 萃取率高、輻射負擔較低。[^1][^2][^3]

> [!note] 功能正常 ≠ 沒有阻塞
> 高度 hydronephrosis 病人中多數阻塞腎臟功能仍接近正常;腎臟大小與實質厚度才是重要手術決策因子。[^1][^2]

### CT / CTA(成人、crossing vessel、術前規劃)

NECT/CECT 顯示阻塞層級、水腎程度與病因(結石、馬蹄腎、重複腎)。CTA / CT urography 以軸位 + MPR + 3D volume rendering,於單一檢查完整評估 crossing vessel 與輸尿管—血管空間關係。[^2][^5][^6] RadioGraphics 描述實用判讀技巧:oblique sagittal「hilar clock-face view」呈現腎門血管—輸尿管的放射狀關係,以及 UPJO 的「inverted teardrop(倒淚滴)」外形與擴張腎外型腎盂向前 / 向後 ballooning。[^5] Helical CT 亦可用於偵測 UPJO 的 crossing vessel。[^6]

### MR urography / MR renography

無輻射、適合年輕族群與追蹤。AJR 之 feasibility study(17 名 UPJO 病人,對照利尿性核醫腎圖)顯示:MR renography(MRR)與核醫在 split renal function ratio 有極佳相關(r=0.87,p<0.01,平均差 <10%);MRR 估算 GFR 與 MDRD 為中度相關(r=0.60);腎盂訊號強度變化率(PUR)在機械性阻塞單位顯著較低(0.39±0.26 vs 2.0±1.38 min⁻¹,p<0.01),可以 89% 準確度區分機械性與功能性阻塞,有潛力成為「one-stop shop」檢查。[^9] 小兒方面,靜態與功能性 MR urography 可與 MAG3 核醫及超音波對照評估 CAKUT。[^8][^11]

## 鑑別與報告要點

| 要點 | 說明 |
| --- | --- |
| 擴張 vs. 阻塞 | UPJ 處解剖狹窄 / 擴張須靠 diuretic renogram 確認是否功能性阻塞[^1][^2][^3] |
| 看輸尿管口徑 | 腎盂腎盞擴張但輸尿管不擴張 → UPJ level;輸尿管也擴張 → 往下游找其他阻塞點[^1][^2] |
| 成人查 crossing vessel | 用 CTA(2D/3D 後處理)/ MRA,描述血管與 UPJ 相對位置(前 / 後、上極 / 下極動脈、相對移行點上下),對「是否壓迫」保守措辭[^2][^5][^6] |
| 相關解剖異常 | 腎臟上升、旋轉與血管異常(MCDK、duplication、horseshoe、ectopia)常與 PUJO 並存,易被忽略[^1][^2][^7] |
| Duplex system | 阻塞可發生在 lower pole UPJ,亦可在 upper pole UPJ[^2] |
| 鑑別:congenital megacalyces | 中央腎盂相對塌陷 / 腎盂大小正常,藉此與水腎區分[^1][^3] |
| 鑑別:extrarenal pelvis / parapelvic cyst | 無腎盞擴張、無延遲排泄;parapelvic cyst 多發時靠 CT urography 延遲期見囊腫間正常腎盂腎盞系統[^1][^2] |

## 題目

> [!question]- UPJ obstruction 在超音波上,與下游輸尿管阻塞最關鍵的鑑別點是什麼?
> 腎盂腎盞明顯擴張,但**輸尿管不擴張**(口徑正常)。擴張止於 UPJ 層級。[^1][^2]

> [!question]- 為什麼「腎功能正常」不能排除 UPJ obstruction?
> 高度 hydronephrosis 病人中多數阻塞腎臟功能仍接近正常;功能正常不代表無阻塞,腎臟大小與實質厚度才是重要手術決策因子。[^1][^2]

> [!question]- 成人 UPJ obstruction 評估 crossing vessel 首選哪種影像?判讀時最需警惕什麼陷阱?
> 首選 CTA(MDCT + 2D/3D 後處理,如 hilar clock-face view)評估輸尿管—血管空間關係;陷阱是血管「鄰近」UPJ ≠ 真正壓迫——跨越血管常位於管徑移行點上方、與移行點無直接關係,故報告措辭應保守。[^5][^6]

> [!question]- Diuretic renogram 上,「阻塞」與「非阻塞性擴張」如何區分?
> 顯著阻塞:利尿劑前後同位素皆不明顯排出;非阻塞性擴張:利尿劑後排空改善。[^1][^2][^3]

> [!question]- Diuretic renogram 首選哪種放射藥劑?為什麼 DTPA 不是首選?
> 首選 Tc-99m MAG-3(萃取率高、輻射低,適用阻塞系統);DTPA 以腎絲球過濾為主,而阻塞性尿路病變中腎絲球功能下降較早較快,僅在腎功能良好時使用。[^1][^3]

> [!question]- MR renography 在 UPJO 的功能評估上能做到什麼?有何實證支持?
> 可測 split renal function(與核醫 r=0.87)、估算 GFR(與 MDRD 中度相關 r=0.60),並以腎盂訊號強度變化率(PUR)區分機械性 vs 功能性阻塞(準確度約 89%),有潛力作為「one-stop shop」檢查。[^9]

> [!question]- UPJ obstruction 的典型先天性病因機轉為何?
> 胚胎期 UPJ 區再管腔化不全,導致輸尿管肌肉層異常、蠕動波無法有效傳遞,形成無蠕動節段(adynamic segment)。[^1][^2]
