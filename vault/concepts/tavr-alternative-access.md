# tavr-alternative-access

**當股動脈路徑不可行時，TAVR替代路徑依侵襲程度排序：subclavian（血管需≥6mm、無嚴重鈣化/扭曲）→ transaortic（右側緣、greater curvature進針）→ transapical（侵襲性最高，左心室尖端異常位置反而是其相對禁忌症，而非「非禁忌症」）。**

## Summary
- **Subclavian approach**：禁忌因素包括血管狹窄（**<6 mm**）、嚴重鈣化、過度扭曲(tortuosity)。[^1]
- **Transaortic approach**：理想aortotomy位置位於**主動脈右側緣（right lateral aspect）、沿greater curvature**，距無名動脈近端約1cm，且應距主動脈瓣至少5.5cm以確保有足夠空間置放balloon-expandable valve。[^1]
- **Transapical approach**：被視為**侵襲性最高**之路徑，通常保留於股動脈、鎖骨下動脈、transaortic路徑皆有禁忌時採用。限制左心室尖端經thoracotomy路徑之因素包括：[^1]
  - **左心室尖端位置異常**（過於medial、inferior或posterior）——此為**相對禁忌症**（非「非禁忌症」，常見陷阱）
  - 既往心肌梗塞造成之心肌顯著變薄或心肌鈣化
  - 肋膜鈣化/異常、心包鈣化
  - 左心室射血分率過低（**<20%**）

> [!note] 考點：左心室尖端位置異常（過於medial/inferior/posterior）是否為transapical approach之相對禁忌症？
> **是**（相對禁忌症）——若考題敘述「不是相對禁忌症」則為錯誤選項。[^1]

## 題目
> [!question]- About alternative access beyond the femoral arterial approach for transcatheter aortic valve replacement (TAVR), which description is wrong? (2020-213)
> **A**——左心室尖端位置異常（過於medial、inferior或posterior）**是**transapical approach之相對禁忌症，並非「非」相對禁忌症。Subclavian禁忌因素（血管<6mm/嚴重鈣化/過度扭曲）、transaortic理想進針點（右側緣沿greater curvature）、肋膜/心包鈣化限制thoracotomy路徑，皆為正確敘述。[^1]
