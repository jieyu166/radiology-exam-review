# tracheoesophageal-fistula

**氣管食道瘔管（TEF）依 Gross 分類，最常見為 Type C（食道閉鎖＋遠端瘔管），約占 85-86%。** 記憶口訣：Type C = most Common。X 光可見上端食道閉鎖（proximal pouch），遠端腸氣存在（因遠端瘔管通氣）。

## Summary
- **Type A**（~8%）：純食道閉鎖（EA），無瘔管。X 光無遠端腸氣。[^1]
- **Type B**（~1%）：食道閉鎖＋近端瘔管。罕見。[^1]
- **Type C**（~**85-86%**）：**食道閉鎖＋遠端瘔管（最常見）**。X 光有遠端腸氣。[^1]
- **Type D**（~1%）：食道閉鎖＋近端＋遠端瘔管。[^1]
- **Type E/H-type**（~4%）：無食道閉鎖，僅瘔管（H 型）。較晚診斷。[^1]
- **VACTERL association**：vertebral, anorectal, cardiac, tracheoesophageal, renal, limb。[^1]

## 陷阱
- Type C = most **C**ommon（85-86%），不要與其他型混淆。
- Type A 無瘔管 → X 光無遠端腸氣（gasless abdomen）。

### 參考來源
[^1]: 2019 交換考詳解；Dahnert Radiology Review Manual 7th ed.：TEF Gross classification，Type C（EA + distal fistula）占 85.6% 為最常見。
