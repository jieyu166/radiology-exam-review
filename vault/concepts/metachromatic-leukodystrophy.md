# metachromatic-leukodystrophy

**異染性腦白質失養症（MLD）是先天缺乏arylsulfatase A所致的溶酶體儲積疾病，最常見亞型為晚期嬰兒型（late infantile variant）；MRI T2WI白質訊號增加，於軸位呈虎斑紋（tigroid pattern）或矢狀面呈豹紋（leopard pattern），代表沿小靜脈周圍白質保留正常訊號；典型的白質病變會保留（sparing）皮質下U型纖維，並非侵犯U fiber。**

## Summary
- **病因**：先天缺乏**arylsulfatase A（芳基硫酸酯酶A）**所致的溶酶體儲積疾病（lysosomal storage disorder）。[^1]
- **最常見亞型**：**晚期嬰兒型（late infantile variant）**，約占50-80%（平均約65%）；其餘為少年型（juvenile，3-10歲發病）與成人型（16歲後發病）。[^1]
- **MRI表現**：T2WI白質訊號增加，受累區域可呈**虎斑紋（tigroid pattern，軸位）**或**豹紋（leopard pattern，矢狀面）**，反映沿小靜脈周圍白質相對保留正常訊號所致的紋理。[^1]
- **U-fiber**：**皮質下U型纖維（subcortical U-fibers）通常受保留（spared）**，並非典型白質病變會侵犯的區域。[^1]
- **併發症**：因sulfatide亦沉積於膽囊壁，**膽囊炎（cholecystitis）為已知併發症**。[^1]

## 陷阱
- 「典型白質變化會侵犯到U fiber」為錯誤——U fiber通常受保留，非受侵犯。[^1]
