**IPF（特發性肺纖維化）**的 ATS/ERS/JRS/ALAT 修訂版診斷標準中，HRCT pattern 分為三類，**不包含 NSIP pattern**。

## Summary

根據 revised ATS/ERS/JRS/ALAT diagnostic criteria for IPF，HRCT pattern 分類：[^1]

| Pattern | 特徵 |
|---------|------|
| **Definite UIP** | Subpleural/basal predominance + reticular + honeycombing ± traction bronchiectasis + absence of inconsistent features |
| **Possible UIP** | Subpleural/basal predominance + reticular + absence of inconsistent features（無 honeycombing） |
| **Inconsistent with UIP** | Upper/mid-lung predominance, peribronchovascular predominance, extensive GGO, profuse micronodules, discrete cysts, diffuse mosaic/air-trapping, consolidation |

- **NSIP pattern 不在此分類系統中** — NSIP 為獨立的 IIP 亞型，非 IPF 的 HRCT pattern 選項。
- 臨床情境正確時，CT 顯示 definite 或 probable UIP pattern 即可確診 IPF。

### 參考來源

[^1]: Respir Res. 2013; 14(Suppl 1): S2。
