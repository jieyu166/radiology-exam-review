# adrenoleukodystrophy

**男童行為退步＋腦後部對稱白質 T2 亮＋周邊強化——這是 X-ALD 的三件套。** 過氧化體病，VLCFA 堆積；白質病灶從頂枕（peritrigonal）向前推進，最先侵犯 splenium（胼胝體壓部）。三區帶記憶：中央壞死（T2 最亮）→ 中間活動性脫髓鞘（leading edge 強化）→ 前緣活動帶。鑑別：Alexander 額葉為主；Krabbe 後肢/錐體束及視放射；MS 不對稱且年長。

## Summary
- **遺傳/代謝**：X-linked 過氧化體病;**極長鏈脂肪酸（VLCFA）顯著升高**;男童典型。[^1]
- **分佈**：**頂枕/peritrigonal 深部白質對稱 T2 高訊號**,早期累及**胼胝體壓部（splenium）**,由後向前進展。[^1]
- **三區帶（zones）**：中央（壞死/膠質化,T2 最高）→ 中間（活動性發炎/脫髓鞘,**周邊 leading edge 強化**）→ 周邊（前緣活動帶）。[^1]
- **本題**：10 歲男童、行為/學業退步、雙側 peritrigonal 對稱 T2 高＋邊緣強化 → **X-linked ALD**（2016-185 正解 A）;鑑別 Alexander（額葉為主）、Krabbe（後肢/錐體束、視放射）、MS（不對稱、年長）。[^1]

> [!note] 考點
> 男童、後部（頂枕/splenium）對稱白質 T2 高＋**周邊 leading edge 強化** → **X-linked adrenoleukodystrophy**。[^1]
