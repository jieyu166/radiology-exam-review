# postoperative-stomach

**胃切除/胃腸吻合術後常見併發症：dumping syndrome、afferent loop syndrome、stomal（吻合口）阻塞、邊緣潰瘍、膽汁逆流性胃炎、殘胃癌;Zollinger-Ellison（ZES）是 gastrinoma 引起的高胃酸，屬原發疾病，不是手術併發症。** 判讀分水嶺：**afferent loop syndrome＝Billroth II 後輸入袢阻塞,US/CT 見上腹擴張充液的 C 形管狀構造（越過中線）;dumping＝食物快速排空致血管／低血糖症狀;ZES 與術後解剖無關。**

## Summary
- **常見術後胃併發症**：[^1]
  - **Dumping syndrome**：胃排空過快 → 早期（滲透/血量）或晚期（反應性低血糖）症狀。
  - **Afferent loop syndrome**：Billroth II 後**輸入袢阻塞**,影像見上腹擴張充液管狀構造（越過中線的 C 形/U 形）、可膽道擴張。
  - **Stomal（吻合口）obstruction**：吻合口水腫/狹窄/疝。
  - 其他：邊緣（marginal）潰瘍、鹼性逆流性胃炎、傾倒、殘胃癌、bezoar。
- **非術後併發症**：**Zollinger-Ellison syndrome（ZES）為 gastrinoma 分泌過量 gastrin → 高胃酸／多發頑固消化性潰瘍**,屬原發內分泌腫瘤疾病,與胃手術無因果。[^1]

> [!note] 考點：何者「不是」術後胃的併發症?
> 「**Zollinger-Ellison syndrome**」不是——ZES 為 gastrinoma 引起的高胃酸,非手術併發症。Dumping、afferent loop、stomal obstruction 皆為術後胃常見併發症。[^1]
