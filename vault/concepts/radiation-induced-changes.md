# radiation-induced-changes

## 重點
- 放射治療後中樞神經系統的影像變化依時序可分為 acute／early delayed／late delayed 三期，辨識關鍵為「治療後經過多久 + 標的病灶位置 + 放療野中含入多少正常結構」——這三點是區分 expected post-RT changes 與 tumor recurrence 的核心。
