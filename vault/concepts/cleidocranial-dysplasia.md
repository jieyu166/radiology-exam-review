# cleidocranial-dysplasia

**Cleidocranial dysplasia 的名字藏著兩個線索：cleido（鎖骨）＋cranial（顱骨），兩端同時出問題。** 影像三招牌：①多發 wormian bones＋顱縫延遲閉合；②鎖骨發育不良或缺如（肩可異常靠攏）；③超多牙齒（supernumerary teeth）。雖名稱著重「膜性骨」，但長骨的軟骨內骨化也受累——所以「膜性＋軟骨內缺陷」並存，不要只記一個。判讀核心是**廣泛性骨化延遲/不全**：哪裡的骨化最晚（顱縫、囟門、恥骨聯合），那裡就最常見到「假性變寬」的影像。

## Summary
- **遺傳/本質**：罕見骨骼發育不良，**體染色體顯性（RUNX2／CBFA1，位於 6p21）**，高外顯率但表現多變;約 35–40% 為新生突變（散發）。以**膜性骨**受累為主，但**長骨（軟骨內/enchondral）亦有缺陷**——故「膜性＋軟骨內骨化缺陷」成立。[^1][^2]
- **Wormian bones**：顱縫間多發小骨;伴囟門/顱縫延遲（或不）閉合、額/頂骨隆突、副鼻竇氣化不良。[^1][^2]
- **鎖骨變形**：發育不良或缺如（約 10% 完全缺如，多為雙側但常不對稱）→ 肩過度活動、雙肩可內收靠攏。[^1][^2]
- 其他：寬下頜、過多牙（supernumerary teeth）/阻生牙/乳牙滯留、高拱顎、鐘形窄胸、恥骨聯合與脊柱發育不良。[^1][^2]

## 各部位 X 光判讀重點

> 以下為各部位影像判讀骨架,皆為查證過之放射特徵;尚未涵蓋之細節留待補強,寧缺勿補。

- **顱骨（skull）**[^1][^2]
  - 顱縫變寬、囟門延遲/不閉合（前囟可持續存在）。
  - 顱縫間多發 **wormian bones**;額、頂骨隆突（bossing），brachycephaly。
  - Sphenoid／副鼻竇氣化不良（loss of pneumatisation）、上頜骨/中臉發育不良、鼻骨骨化延遲。
- **鎖骨／胸廓（clavicle / chest）**[^1][^2]
  - 鎖骨發育不良或完全缺如(常雙側、不對稱);肩胛帶過度活動,臨床可見雙肩前併。
  - 胸廓呈鐘形/錐形窄胸,肋骨短而斜;偶見多生肋（supernumerary ribs）。
- **骨盆／恥骨聯合（pelvis / pubic symphysis）**[^1][^2]
  - **恥骨支骨化延遲/缺如 → 恥骨聯合假性變寬**(pseudo-widening),為招牌徵象。
  - 髂翼發育不良、薦髂關節變寬;可見 coxa vara（股骨頸幹角 <120°）。
- **脊椎（spine）**[^1][^2]
  - 脊椎側彎（頸/胸/腰段皆可）、半椎體（hemivertebra）、脊椎裂等分節/骨化異常。
- **手足（hands / feet）**[^1][^2]
  - **第 2、5 指中節指骨短、遠端指骨尖細（tapering）**;掌骨細長。
  - 掌/蹠骨假性骨骺（pseudoepiphyses）、cone-shaped epiphyses;皮質變薄、髓腔變寬。
- **牙齒／顎（teeth / jaw,全口環景片 panoramic）**[^1][^2]
  - **多發過多牙（supernumerary teeth／hyperdontia）＋多發阻生齒（impacted）**,乳牙滯留,恆牙萌出延遲。
  - 高拱顎、下頜髁突細、上頜竇發育不良 → 牙列擁擠/咬合不正。

> [!note] 考點：何者為真？
> **以上皆是**——膜性＋軟骨內骨化缺陷、wormian bones、鎖骨變形三者並存。[^1]
