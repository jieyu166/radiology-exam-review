# invasive-lobular-carcinoma

**ILC 是第二常見乳癌(~5–10%),以『單列(single-file)生長、影像隱匿、少鈣化』著稱,易被低估。** 判讀分水嶺一：**組織學單列(Indian file)浸潤、E-cadherin 缺失**;判讀分水嶺二：**影像＝spiculated mass(最常見)／架構扭曲／不對稱密度;常僅單一投影可見或 mammographically occult;『微鈣化少見(<10%)、非特徵』**。雙側/多中心傾向較高。

## Summary
- **流行病學**：占所有乳癌 **5–10%**。[^1]
- **組織學**：**單列(single-cell file／Indian file)生長**、E-cadherin 缺失 → 浸潤不成團、邊界不清。[^1]
- **影像（依頻率）**：**spiculated mass(最常見)**、不對稱密度(3–25%)、架構扭曲(10–25%)、**微鈣化 <10%（少見）**;**16% mammographically occult 或看似良性**。[^1]
- **判讀陷阱**：浸潤型態使腫瘤**常僅單一投影可見**、易被低估;US 為異質低回聲腫塊。[^1]

> [!note] 考點：關於 ILC 特徵,何者「非」?
> 「**鈣化是其特徵**」非——ILC **微鈣化少見(<10%)、非特徵**。占乳癌 ~10%(A)、單列生長(B)、常僅單一投影可見(D)皆正確。[^1]
