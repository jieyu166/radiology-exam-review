# renal-angiomyolipoma

**巨大腹膜後含脂肪病灶的分水嶺在「血管」：AML 富血管（動脈瘤樣血管延入腎皮質＋腎實質缺損 / beak sign），脂肪肉瘤 hypovascular；找到「病灶內動脈瘤樣血管」即鎖定 AML。** 判讀分水嶺一：**AML → 腎實質缺損＋供應血管深入腎皮質；>4 cm 微動脈瘤破裂 → Wunderlich syndrome**；判讀分水嶺二：**脂肪肉瘤 → hypovascular、外推腎臟、無腎實質缺損**。

## Summary
- **AML**：PEComa,由**血管、平滑肌、脂肪**組成;TSC 相關者多發雙側。富血管,**>4 cm 易因微動脈瘤破裂自發出血（Wunderlich syndrome）**。[^1]
- **鑑別 AML vs 巨大腹膜後脂肪肉瘤**：[^1][^2]
  - **病灶內動脈瘤樣/大血管延伸入腎皮質 → AML**（liposarcoma 為 hypovascular）。
  - **腫瘤接觸處出現腎實質缺損（parenchymal defect / beak sign）→ 外生性 AML**。
  - 兩者皆可外推腎臟、有軟組織密度、邊界不清 → 非鑑別點。

### 影像判讀骨架（一般放射科醫師閱片所需）
- **含脂典型 AML——診斷關鍵在「巨觀脂肪」**：UECT（非顯影 CT）內見 **<−10 HU 的低密度脂肪區**即可在適當臨床情境下確診;務必用**薄層（1.5–3 mm）並以小 ROI 量測**以偵測少量脂肪,避免容積平均(volume averaging)假陰性。[^3]
- **US**：典型 AML **高回音**,回音度接近腎竇脂肪,可見**後方聲影(posterior shadowing)**;RCC 雖也可高回音但通常**低於腎竇脂肪回音**——US 單獨不足以確診,小高回音病灶仍須 CT/MRI 確認有無巨觀脂肪。[^3]
- **MRI 脂肪判讀**：典型 AML 在**脂肪抑制序列(FS)訊號掉落**;化學位移(chemical shift)反相位(opposed-phase)在**巨觀脂肪—水交界**出現 **India ink artifact(黑邊)**。注意:化學位移的整體訊號掉落(signal drop)反映的是**體素內脂肪**(較像 RCC 之微觀脂肪),而 AML 的巨觀脂肪主要靠 FS 掉訊號與交界黑邊來辨識。[^3][^4]
- **與 RCC 鑑別**：RCC **罕含巨觀脂肪**,若有多因包覆腎周脂肪、產脂壞死或骨化生(osseous metaplasia)所致,且**RCC 內見鈣化偏向 RCC**;反之 **AML 罕見鈣化**——故「含脂肪＋無鈣化」傾向 AML,「含脂肪＋鈣化」需高度警覺 RCC。[^3]
- **fat-poor / lipid-poor(少脂)AML 的判讀困難**：約 60% 少脂 AML 在 UECT 呈**均質高衰減(>45 HU)**、**T2 低訊號**、**早期均質強化**,且 FS-MRI 與 CS-MRI **多無訊號掉落**——與(尤其是乏脂的)RCC 影像重疊,常需化學位移、強化型態(早期均質強化 vs RCC 廓清/heterogeneous)輔助,難以單靠影像確診。[^3][^4]
- **依脂肪量的影像分類**(可結構化鑑別)：**fat-rich AML**(≤ −10 HU)、**fat-poor AML**(> −10 HU;tumor-to-spleen ratio < 0.71;signal intensity index > 16.5%)、**fat-invisible AML**(> −10 HU;tumor-to-spleen ratio > 0.71;signal intensity index < 16.5%)。[^4]
- **epithelioid AML(上皮樣)**：罕見變異,具**惡性潛能**;影像常為**大且異質強化的腫塊,合併內出血、壞死、無明顯巨觀脂肪**——遇到「大、異質、無脂、侵襲性」病灶不可逕自當良性 AML。[^3]
- **出血風險與處置門檻**：自發出血(可致 Wunderlich syndrome／腹膜後出血)風險與 **AML ≥4 cm** 或 **病灶內動脈瘤 >5 mm** 相關,此為傳統介入(栓塞)指標。[^3]
- **結節性硬化症(TSC)**：腎 AML 盛行率高達 **約 90%**,常**多發、雙側**,且因自發出血而有症狀;TSC 並常合併肺部**淋巴管平滑肌瘤病(LAM)**——雙側多發腎 AML 應主動聯想 TSC 並檢視肺部。[^3]

> [!note] 考點：哪一所見較傾向 AML 而非腹膜後脂肪肉瘤?
> **病灶內動脈瘤樣血管（aneurysmal vessels）**——AML 富血管;脂肪肉瘤 hypovascular。腫塊外推腎臟、軟組織密度、邊界不清在兩者皆可見,鑑別力低。[^1][^2]
