# hht-liver

**HHT（Osler-Weber-Rendu）為體染色體顯性遺傳，肝臟受侵犯常見但多數無症狀。** CT 表現：dilated hepatic arteries、portal/hepatic vein early filling（A-V/A-P shunting）、biliary abnormality。

## Summary
- 遺傳：autosomal dominant [^1]
- 臨床三聯：mucocutaneous telangiectases + epistaxis + visceral AVMs [^1]
- 肝臟受侵犯：**有肝臟 AVM 的病人很多，但 symptomatic liver involvement 少見** [^1]
- CT：dilated hepatic arteries、arterial phase 早期 portal/hepatic vein filling、biliary changes [^1]
- 可侵犯：pulmonary / cerebral / spinal AVMs [^1]
