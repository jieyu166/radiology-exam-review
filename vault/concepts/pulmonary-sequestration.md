# pulmonary-sequestration

**肺隔離症的兩型共同點是「無氣道通連、體循環動脈供血」，分水嶺在靜脈引流：extralobar 引回體循環，intralobar 引入肺靜脈。** 判讀分水嶺一：兩型**供血皆為體循環動脈**（常自主動脈），這點一樣；判讀分水嶺二：**靜脈引流——extralobar→體循環（azygos/IVC/portal）；intralobar→肺靜脈**，答「extralobar 靜脈引流入肺靜脈」為非。附加記憶：extralobar 有自己的胸膜包覆、左下葉為主（~90%）、男性多（4:1）、易合併先天膈疝；intralobar 反覆感染、成人多見。記憶鉤：Extra（獨立包覆）→ 靜脈也走體循環，Intra（共膜同住）→ 靜脈混入肺靜脈。

## Summary

### 定義與分型
- **肺隔離症（bronchopulmonary sequestration, BPS）**：一團無功能的異常肺組織，**接受體循環動脈供血**、且**與正常氣管支氣管樹無正常相通**——此兩點為兩型共同的診斷核心。[^1][^2]
- **分型依胸膜（pleural envelope）區分**：[^1][^2]
  - **Intralobar（ILS，肺葉內型）**：位於正常肺葉的臟層胸膜「內」，與相鄰正常肺**共用同一層臟層胸膜**（無獨立胸膜包覆）。
  - **Extralobar（ELS，肺葉外型）**：**有自己獨立的臟層胸膜包覆**（separate / own visceral pleura），與正常肺組織分隔。
- **相對頻率**：ILS 較常見（約佔 BPS 之 75–85%），ELS 較少見（約 15–25%）；ILS 多在青少年/成人因反覆感染被發現，ELS 多在嬰幼兒/產前因合併其他畸形被發現。[^1][^3]

### 供血動脈與靜脈引流（CTA/MRA 為診斷核心）
- **供血動脈——體循環動脈是診斷關鍵**：兩型皆由**體循環動脈**供血，最常源自**胸主動脈下段或腹主動脈**（可經膈肌或膈下發出，少數來自肋間、鎖骨下、腹腔幹等分支）。**辨識出這條異常體循環供血動脈是影像確診的核心**，也是術前規劃避免大出血的關鍵。[^1][^2][^4]
- **靜脈引流——兩型分水嶺**：[^1][^2]
  - **ILS → 引流入肺靜脈**（pulmonary veins，肺循環）。
  - **ELS → 多引流入體循環靜脈**（systemic：azygos/hemiazygos/IVC，偶可入 portal vein）。
- **影像首選**：**CT angiography（CTA）為最佳非侵入性檢查**，可同時顯示異常實質與供血/引流血管；MRA / contrast-enhanced MR 為無輻射替代（產前及小兒尤佳）；傳統導管血管攝影已少作為診斷用途。[^1][^4]

### 各模態表現
- **胸部 X 光（CXR）**：常見**下葉後基底段**（尤其左下葉）持續或反覆的實變/腫塊/含氣囊狀病灶；ILS 反覆感染時可呈不消散的肺炎樣陰影。CXR 無法顯示供血動脈，僅供初步懷疑。[^1][^2]
- **CT / CTA**：可見實變、囊狀變化、黏液栓、肺氣腫樣改變；**關鍵是追蹤到源自主動脈的供血動脈**並判定靜脈引流走向以分型。[^1][^4]
- **產前超音波 / 胎兒 MRI**：ELS 典型為均質高回音的下葉/膈下三角形腫塊；都卜勒可顯示來自主動脈的供血動脈（與 CPAM 鑑別要點）；胎兒 MRI 可進一步定性並評估合併畸形。[^1]

### 好發部位與合併畸形（hybrid lesion）
- **好發左下葉後基底段**為兩型共同傾向；**ELS 約 90% 位於左側**，男性居多（M:F 約 4:1）。[^1][^3]
- **ELS 常合併其他先天畸形**，以**先天性膈疝**最具代表，亦見於膈膨出、心血管畸形等。[^1][^2]
- **Hybrid lesion（混合病灶）**：隔離症可與 **CPAM（congenital pulmonary airway malformation，舊稱 CCAM）**共存——即一團組織同時具備隔離症的體循環供血與 CPAM 的微囊/巨囊成分；病理常見、影像見體循環供血加上多囊成分時應提示。[^1]

### 鑑別診斷（DDx）
- **不消散/反覆的下葉肺炎或肺膿瘍**：ILS 反覆感染時最易混淆，差別在於**有體循環供血動脈**。[^1][^2]
- **CPAM**：囊性為主、典型由肺循環供血、無系統性供血動脈（hybrid 例外）。[^1]
- **先天支氣管閉鎖、支氣管源性囊腫、先天肺葉性肺氣腫**等下葉發育異常。[^2]
- **下葉腫瘤/腫塊**：成人偶然發現的下葉腫塊，須以 CTA 找供血動脈避免誤判。[^1]

### 判讀陷阱
- **必須主動尋找供血動脈**：未在主動脈旁/膈附近追蹤到體循環供血動脈，可能把 BPS 誤判為單純肺炎、膿瘍或腫瘤；薄層 CTA 與 MIP/容積重組有助顯示。[^1][^4]
- **勿單以靜脈引流定診**：引流型態（肺靜脈 vs 體循環靜脈）用於**分型**，但「體循環動脈供血＋無正常氣道相通」才是**定性診斷**的核心。[^1][^2]
- **CXR 陰性不能排除**：CXR 不顯示血管，下葉持續陰影或反覆同部位肺炎應進一步做 CTA/MRA。[^1][^2]
- **術前出血風險**：異常供血動脈起源變異多（胸/腹主動脈、膈下、腹腔幹），術前影像未完整標示供血血管可致術中大出血，故影像報告須明確描述供血動脈起源與數目。[^2][^4]

> [!note] 考點：extralobar sequestration 何者「為錯」?
> 「**靜脈引流入肺靜脈**」為錯——extralobar **多引流入體循環靜脈（azygos/IVC 等）**；intralobar 才引流入肺靜脈。左下葉、男性多、體循環供血皆對。[^1][^2]

> [!note] 考點（2018-045）：pulmonary sequestration 敘述何者「錯」?
> **Intralobar sequestration通常在新生兒期以呼吸窘迫/發紺/感染表現，extralobar則在兒童期以反覆感染表現（B）**——**相反**。ILS多在青少年/成人因反覆感染被發現；ELS多在新生兒/嬰兒期表現（可因分流致high output heart failure）或產前發現。分型25%/75%(A對)、ELS靜脈引流入體循環靜脈至右心房、ILS引流入肺靜脈(C對)、ELS好發左下葉可位於膈下(D對)皆為正確描述。[^1][^2]

## 題目
> [!question]- Which of the description about pulmonary sequestration is wrong? (2018-045)
> **Intralobar sequestration usually present in the neonatal period... Extralobar sequestration present later in childhood（B）**——相反：ILS多在青少年/成人因反覆感染發現，ELS多在新生兒/嬰兒期或產前發現。[^1][^2]
