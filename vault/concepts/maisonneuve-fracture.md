# maisonneuve-fracture

**Maisonneuve fracture＝『近端腓骨螺旋骨折』＋踝關節損傷,合併『脛腓骨聯合(tibiofibular syndesmosis)撕裂』＋內踝/三角韌帶損傷;為旋前外旋(PER)機轉的不穩定傷。** 判讀分水嶺:**近端腓骨骨折＋syndesmosis 撕裂**——踝 X 光看似單純,須照全腓骨。

## Summary
- **定義**:近端腓骨螺旋骨折＋踝損傷,合併**脛腓骨聯合(syndesmosis)撕裂**(正解 D)＋內踝骨折或三角韌帶撕裂。[^1]
- 機轉:旋前-外旋(PER);不穩定、常需手術固定 syndesmosis。[^1]
- 陷阱:踝 X 光可能只見內踝/syndesmosis 增寬,須加照近端腓骨。[^1]

### 參考來源
[^1]: Maisonneuve fracture 標準教學(AJR 1999;173(3):702 等):近端腓骨螺旋骨折＋脛腓骨聯合撕裂＋內側結構損傷;PER 機轉、不穩定。
