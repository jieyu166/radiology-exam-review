# maisonneuve-fracture

**Maisonneuve fracture 的精神:扭傷的力量從腳踝『一路往上傳』,在『近端腓骨（靠膝蓋那端）』折斷。想像踝關節被外旋扭開,力量沿著脛腓骨之間的韌帶（syndesmosis）往上撕,最後在高位腓骨斷掉——三處連成一線。** 判讀分水嶺（也是最大陷阱）:**病人只喊腳踝痛、踝 X 光看似單純（只見內踝骨折或 syndesmosis 增寬）,真正的骨折卻在沒照到的近端腓骨**——所以懷疑時一定要加照全腓骨。

## Summary
- **定義**:近端腓骨螺旋骨折＋踝損傷,合併**脛腓骨聯合(syndesmosis)撕裂**(正解 D)＋內踝骨折或三角韌帶撕裂——高位腓骨→syndesmosis→內側,三處一條力線。[^1]
- 機轉:旋前-外旋(PER);因 syndesmosis 被撕開 → **關節不穩定**、常需手術固定。[^1]
- 陷阱:踝 X 光可能只見內踝/syndesmosis 增寬,**真正骨折在沒照到的近端腓骨**,須加照全腓骨。[^1]

### 參考來源
[^1]: Maisonneuve fracture 標準教學(AJR 1999;173(3):702 等):近端腓骨螺旋骨折＋脛腓骨聯合撕裂＋內側結構損傷;PER 機轉、不穩定。
