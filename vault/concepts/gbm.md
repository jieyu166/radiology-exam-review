# gbm

**GBM 的招牌＝『中央壞死＋不規則環狀強化＋瀰漫 vasogenic edema＋可出血』的侵襲性腦腫瘤；（顱外）轉移不是它的特徵（極罕見）。** 判讀分水嶺一：**厚不規則環狀強化＋中央壞死＋越胼胝體（butterfly）**；判讀分水嶺二：**局部侵犯/沿白質播散為主，顱外轉移罕見**。

## Summary
- **典型 MRI**：白質為中心、**不規則周邊（環狀）強化、中央壞死、周圍 vasogenic edema**（內含浸潤腫瘤）。[^1]
- **出血**：腫瘤內出血可見（急性可似中風）。[^1]（選項 B 為特徵）
- **強化**：厚、不規則環狀強化。[^1]（選項 D 為特徵）
- **Perifocal（vasogenic）edema**：常見且明顯。[^1]（選項 C 為特徵）
- **壞死**：中央壞死為標誌。[^1]（選項 A 為特徵）
- **擴散方式**：沿白質束/胼胝體局部浸潤、可多中心；**顱外轉移極罕見** → **Metastasis 非 GBM 之特徵**（題目正解）。[^1]

> [!note] 考點：何者「非」GBM 特徵？
> **Metastasis（顱外轉移）**。GBM 以中央壞死、出血、perifocal edema、不規則環狀強化為特徵；顱外轉移極罕見。[^1]
