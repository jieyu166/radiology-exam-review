# gastric-carcinoid-tumor-classification

**胃類癌三型的判讀口訣：Type 1（多小、預後好，慢性萎縮性胃炎相關）、Type 2（伴 MEN/ZES）、Type 3（大且孤立、轉移多、預後差，「與 ZES/MEN 無關」是本病考題最易誤判處）。**

## Summary
| 特徵 | Type 1 | Type 2 | Type 3 |
|---|---|---|---|
| 比例 | 70–80% | 5–10% | 15–25% |
| 大小/數目 | 多發、小(<1cm) | 多發、大小不一(1–2cm) | **孤立、大(>2cm)** |
| 轉移 | 少見(<10%) | 10–30%(多為局部淋巴結) | **75%有淋巴結/肝轉移** |
| 相關疾病 | 慢性萎縮性胃炎 | ZES(高胃泌素血症期)、MEN | **與上述疾病無關**；TP53基因突變過度表現 |
| 預後 | 良好 | 良好(少數具侵襲性) | **差** |

[^1]

> [!note] 考點：Type 3 胃類癌何者「有誤」？
> **與Zollinger-Ellison症候群相關**為Type 3之錯誤描述——ZES相關為**Type 2**特徵，Type 3與ZES/MEN等疾病「無關」。Type 3之大且中心壞死(A)、轉移常見(B)、女性好發(C)皆為正確描述。[^1]

### 參考來源
[^1]: *RadioGraphics* 2017;37:516-536（2018 交換考題詳解 p.110 引用；Tier 1 影像期刊，Table 4 Differentiating Types of Gastric Carcinoids）。Type 1佔70-80%、多發小(<1cm)、轉移<10%、與慢性萎縮性胃炎相關、預後佳；Type 2佔5-10%、多發大小不一(1-2cm)、轉移10-30%多局部淋巴結、與ZES(高胃泌素血症期)及MEN相關、預後佳(少數侵襲性)；Type 3佔15-25%、孤立大病灶(>2cm)、轉移75%(淋巴結/肝)、與上述疾病無關(TP53突變過度表現)、預後差。

## 題目
> [!question]- Which of the following features about type 3 gastric carcinoid tumors is wrong? (2018-130)
> **與Zollinger-Ellison症候群相關（D）**——ZES相關為Type 2特徵，非Type 3。Type 3為大且中心壞死(A)、轉移常見(B)、女性好發(C)皆正確。[^1]
