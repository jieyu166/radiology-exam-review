# mosaic-attenuation

**馬賽克衰減的三個成因邏輯各異：小氣道病「低密度是病灶，呼氣加重」；血管病「低密度是血流少，呼氣不加重」；浸潤病「GGO 才是高密度病灶」——三者都有深淺交錯，但機轉完全不同。** 判讀分水嶺一：**呼氣相 HRCT 加重→小氣道病；低密度區血管纖細→血管病（oligaemia）；高密度區為 GGO→浸潤**；判讀分水嶺二：**CTEPH 因大血管栓塞，造成符合血管領域的大範圍 mosaic，與小血管型肺高壓（idiopathic PAH、PVOD）的均勻瀰漫型截然不同**。

## Summary
- **三大機轉**：[^1][^2]
  - **小氣道病（air trapping）**：低密度區為異常,**呼氣相加重**;血管在低密度區變細（反射性血管收縮）。
  - **閉塞性血管病（mosaic perfusion）**：低密度區 **oligaemic（血流少）**;呼氣相不加重。
  - **浸潤性肺病**：GGO 為**高密度**區（如肺泡/間質病變）。
- **CTEPH（大血管型）**：慢性大（節段/葉）肺動脈栓塞 → **大範圍、區域性、符合血管領域（vascular territory）的 mosaic perfusion**;故「造成符合血管領域之大範圍 mosaic」者＝**CTEPH**（2016-204 正解 A）。[^2]
- **小血管型肺高壓**：idiopathic PAH、PVOD、mitral stenosis 多為**較均勻/瀰漫**或 centrilobular GGO，**非大範圍territory mosaic** → 故非此題答案。[^2]
- **鑑別工具**：呼氣相 HRCT（小氣道病加重）、血管口徑（血管病低密度區血管細）、CTPA（CTEPH 大血管栓）。[^2]
- **病因分布（記憶要點）**：實質性肺病約佔 mosaic 病例 **半數**、小氣道病約 **三分之一**、血管病為其餘。[^2]
- **呼氣相判讀細節**：[^2]
  - 小氣道病 → 患段衰減**不變**（氣體出不來＝air-trapping）;非氣道性病因 → 全肺**均勻升高**衰減。
  - **氣管／主支氣管後壁（膜部）內凹**＝呼氣用力足夠的確認指標;吸氣相誤判前須先檢查後壁是否內凹。
- **周邊肺動脈口徑（一線分流線）**：低衰減區周邊肺動脈纖細、符合血管領域 → 血管性病因（典型為 CTEPH）。**但此徵非 CTEPH 專屬**——小氣道病經缺氧性血管收縮亦可致低衰減區血流分流、血管變細，故須合併其他徵象判讀。[^2]
- **CTEPH vs PAH（小血管型）形態差異**：[^2]
  - CTEPH → **界線清楚的節段／次節段** mosaic（符合血栓的血管分布）;可見附壁血栓、突然中斷之血管、管徑突變（再通）、webs/bands、corkscrew 血管。
  - PAH → mosaic 較**瀰漫**;低衰減多侷限於次小葉，伴血管周圍高衰減小結節。
- **肺高壓量化線索**：主肺動脈／升主動脈直徑比 **>1**;≥3 葉之節段動脈／支氣管比 **>1** → 支持肺高壓。[^2]
- **大氣道分布線索**：支氣管擴張延伸進入**低衰減（hyperlucent）肺**→ 小氣道病;延伸進入**高衰減**肺 → 實質性病因（如 OP、DAD 機化期、NSIP）。[^2]

> [!note] 考點：何者造成符合血管領域的大範圍 mosaic?
> **CTEPH（慢性血栓栓塞性肺高壓）**——大血管慢性栓塞致區域性 mosaic perfusion 符合血管領域;idiopathic PAH/PVOD/mitral stenosis（小血管/後微血管）較均勻,非territory mosaic。[^2]
