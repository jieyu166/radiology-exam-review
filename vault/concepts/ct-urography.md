# ct-urography

**CTU 比「CECT 加一個排泄延遲期」多的那一件事，就是最前面的未顯影期（KUB）。** 三期順序：未顯影（查結石）→ nephrogram（腎實質）→ excretory（集尿系統）；「能不能看輸尿管」兩者都可以，不是鑑別點。記憶鉤：**「KUB 先到才是 CTU」**——少了這一期，就只是「CECT 加延遲」。

## Summary
- **CTU 協定（典型 3 期）**：[^1]
  - **未顯影 KUB**：評估腎結石。
  - **Nephrogram（腎實質）期**：約注射後 100 秒;可給 IV furosemide 改善排泄期。
  - **Excretory（排泄）期**：評估集尿系統/輸尿管（furosemide 後約 10 分鐘;可俯臥/咳嗽幫助充盈）。
  - 變化型：split-bolus（合併期數降劑量）。
- **與『CECT＋排泄延遲期』的差別**：**CTU 含專屬未顯影期**;後者一般無未顯影期。兩者皆可評估輸尿管 → 差別不在輸尿管評估。[^1][^2]
- **三種掃描技術**（無單一公認協定）：[^2]
  - **Triple-phase（傳統單能）**：未顯影 → 單次 bolus → nephrographic（bolus 後 100 秒）→ excretory（bolus 後 5–15 分）。
  - **Dual-phase split-bolus**：未顯影 → 打第一 bolus → 延遲 5–15 分後打第二 bolus → 合併 nephrographic＋excretory 單次擷取（第二 bolus 後 100 秒），減一期降劑量。
  - **Single-phase dual-energy**：單次 bolus + 單一 excretory 期（常 80/140 kVp）→ 後處理產生 virtual nonenhanced 影像;但對小結石敏感度較低、virtual nonenhanced 衰減值不準，尚未廣泛採用。
- **影像判讀要點**：excretory 期 axial source images 是橫斷評估輸尿管的最佳影像;務必用**寬窗**（如 width 1200/level 350 起手，再手動微調）避免小病灶被高密度排泄對比劑遮蔽。[^2]

> [!note] 考點：CTU 與『CECT＋排泄延遲期』最佳鑑別?
> **CTU 包含一個未顯影（nonenhanced）期**。兩者皆能看輸尿管(故 B 錯);CTU 含顯影期(C 錯);非同義(A 錯)。[^1][^2]
