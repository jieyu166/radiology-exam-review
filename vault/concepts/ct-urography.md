# ct-urography

**CTU 比「CECT 加一個排泄延遲期」多的那一件事，就是最前面的未顯影期（KUB）。** 三期順序：未顯影（查結石）→ nephrogram（腎實質）→ excretory（集尿系統）；「能不能看輸尿管」兩者都可以，不是鑑別點。記憶鉤：**「KUB 先到才是 CTU」**——少了這一期，就只是「CECT 加延遲」。

## Summary
- **CTU 協定（典型 3 期）**：[^1]
  - **未顯影 KUB**：評估腎結石。
  - **Nephrogram（腎實質）期**：約注射後 100 秒;可給 IV furosemide 改善排泄期。
  - **Excretory（排泄）期**：評估集尿系統/輸尿管（furosemide 後約 10 分鐘;可俯臥/咳嗽幫助充盈）。
  - 變化型：split-bolus（合併期數降劑量）。
- **與『CECT＋排泄延遲期』的差別**：**CTU 含專屬未顯影期**;後者一般無未顯影期。兩者皆可評估輸尿管 → 差別不在輸尿管評估。[^1][^2]
- **三種掃描技術**（無單一公認協定）：[^2]
  - **Triple-phase（傳統單能）**：未顯影 → 單次 bolus → nephrographic（bolus 後 100 秒）→ excretory（bolus 後 5–15 分）。
  - **Dual-phase split-bolus**：未顯影 → 打第一 bolus → 延遲 5–15 分後打第二 bolus → 合併 nephrographic＋excretory 單次擷取（第二 bolus 後 100 秒），減一期降劑量。
  - **Single-phase dual-energy**：單次 bolus + 單一 excretory 期（常 80/140 kVp）→ 後處理產生 virtual nonenhanced 影像;但對小結石敏感度較低、virtual nonenhanced 衰減值不準，尚未廣泛採用。
- **影像判讀要點**：excretory 期 axial source images 是橫斷評估輸尿管的最佳影像;務必用**寬窗**（如 width 1200/level 350 起手，再手動微調）避免小病灶被高密度排泄對比劑遮蔽。[^2]

### 適應症與各期分工
- **主要適應症**：成人血尿（尤其無痛肉眼血尿、高危族群）評估上尿路 urothelial 病灶;結石定位與成分線索;腎腫瘤/腎盂腫瘤分期;腎盂輸尿管解剖（術前、移植）。[^1][^3]
- **各期看什麼**（影像判讀分工）：[^2][^3]
  - **未顯影期**：結石（高衰減）、基準衰減值（判斷後續是否真強化）、urothelial carcinoma 常見的腫瘤內鈣化、suburothelial 出血的高密度壁。
  - **Nephrographic/corticomedullary 期**：腎實質腫瘤（RCC vs 浸潤型上尿路癌）、腫瘤強化、腎柱假瘤排除;UTUC 因早期強化在此期可見「focal 壁增厚＋異常 hyperenhancement」。
  - **Excretory（延遲 7–15 min）期**：集尿系統與輸尿管全程充盈、腔內充盈缺損、壁增厚的「負影」輪廓。**兩期是否都必要**有文獻爭議——有研究主張單用 excretory 期即可達相近 UTUC 敏感度，但 nephrographic 期對偵測非充盈缺損型（壁增厚/浸潤型）腫瘤仍有助益，故多數方案仍保留雙期。[^4]

### 輸尿管一般異常表現與基本概念（CTU for evaluation of the ureter）
- **CTU 可呈現的輸尿管異常型態**：先天異常（congenital abnormality）、腔內充盈缺損（filling defect）、擴張（dilatation）、狹窄（narrowing）、走行偏移（deviation in course）。[^2]
- **輸尿管重複（ureteral duplication）**：為**泌尿道最常見的先天異常**，發生率約**佔人口 1%**。[^2]
- **擴張 ≠ 阻塞（重要判讀陷阱）**：**擴張的輸尿管不必然代表阻塞**（如懷孕、術後、逆流性擴張、慢性阻塞已代償）；反之，**阻塞的輸尿管也不總是擴張**（急性阻塞早期、腎周滲漏減壓）。故單憑輸尿管口徑不能斷定是否阻塞，須合併腎盂腎盞擴張程度、臨床病史等綜合判斷。[^2]
- **輸尿管膨出（ureterocele）**：定義為輸尿管穿越膀胱壁處的**球狀擴張（bulbous dilatation）**;CT/影像上呈**投射入膀胱腔內的圓形充盈缺損（filling defect）**。[^2]

> [!note] 考點：關於 CT urography 評估輸尿管，何者「正確」?
> **a+b+c+d 全對**——CTU 可呈現先天異常/充盈缺損/擴張/狹窄/走行偏移;輸尿管重複為最常見先天異常（約占人口1%）;擴張輸尿管不必然阻塞、阻塞輸尿管不必然擴張;ureterocele 為輸尿管穿膀胱壁處球狀擴張，呈投射入膀胱腔的圓形充盈缺損。[^2]

### Urothelial carcinoma（UTUC）判讀重點
- **三種形態型態**：[^3]
  - **乳突型（papillary）**：腔內單發或多發**充盈缺損**;可見「oncocalyx」（整個 calyx 被實質組織取代而消失）或「phantom calyx」（infundibulum 阻塞致 calyx 擴張但不顯影）。
  - **平坦型（flat）**：同心或偏心**壁增厚**而管腔不一定變窄;excretory 期較難辨識，需靠 nephrographic 期 hyperenhancement。
  - **浸潤型（infiltrative）**：腎竇/輸尿管周圍脂肪密度增加與條紋化;腎臟外形（reniform shape）多保留，可與 RCC 的膨脹性腫塊區分。
- **強化是關鍵鑑別**：UTUC **早期強化**（與基準比有真實 enhancement），與不強化的血塊明顯不同;務必對照未顯影期確認非結石、確認是否真強化。[^3]
- **多發性（multifocality）**：上尿路癌易**多中心**——同側多處 focal 壁增厚＋hyperenhancement 提示多發 TCC;且常合併膀胱病灶，判讀務必掃描輸尿管全程與膀胱（field cancerization）。[^2][^3]
- **PPV 與信心分級**：CTU 對 UTUC 的整體陽性預測值約 53%，但歸類為「大腫塊（large mass）」者 PPV 可達約 83%——病灶越大、形態越明確，惡性可能越高。[^5]

### 鑑別診斷與陷阱
- **腔內充盈缺損 DDx**（對照未顯影期是分水嶺）：[^3]
  - **結石**：未顯影期高衰減;但 UTUC 本身會鈣化，任何尿路鈣化都要先排除其旁是否有腫塊或 focal 壁增厚再判為單純結石。
  - **血塊**：稍高密度、**不強化**、可隨體位/不同時相移動。
  - **黴菌球（fungus ball）／脫落乳突（sloughed papilla，papillary necrosis）**：可模擬腫瘤;papillary necrosis 有 teardrop/ball-on-tee、末期「signet ring」徵象。
  - **fibroepithelial polyp**：長條、平滑、有蒂之良性病灶，常被對比劑完整包繞（除蒂部）;有時隨不同擷取而位置移動，但不一定能與惡性區分。
- **壁增厚 DDx**：發炎（pyeloureteritis、結核）多為對稱環狀增厚、平滑緣、追蹤可消退;**suburothelial 出血**呈瀰漫腎盂壁增厚＋管腔變窄，靠未顯影期壁高密度與 UTUC 區分;pyeloureteritis cystica 為多發毫米級充盈缺損，較均質、數目相對少於多發 UTUC。[^3]
- **假性缺損／技術陷阱**：[^2][^6]
  - **充盈不全／輸尿管蠕動**：未充盈段或塌陷段可造成假性壁增厚或假性缺損;同段需在不同影像／時相重複確認後再判讀。
  - **彎曲輸尿管的部分容積**：tortuous 輸尿管斜切可產生假性充盈缺損。
  - **外壓**：血管（如 retrocaval ureter、髂血管）、淋巴結、後腹腔病灶造成外在壓迫須與內生病灶區分。
  - **輔助充盈手法**：低劑量利尿劑（furosemide）、水化、俯臥、咳嗽可改善集尿系統與輸尿管全程均勻充盈，降低偽病灶。
- **輻射劑量考量**：多期掃描劑量高;以 split-bolus 合併期數、dual-energy 產生 virtual nonenhanced 取代真實未顯影期，皆為降劑量策略（後者對小結石敏感度與衰減值準確度較差）。[^2]

> [!note] 考點：CTU 與『CECT＋排泄延遲期』最佳鑑別?
> **CTU 包含一個未顯影（nonenhanced）期**。兩者皆能看輸尿管(故 B 錯);CTU 含顯影期(C 錯);非同義(A 錯)。[^1][^2]

## 題目
> [!question]- About CT urography for evaluation of the ureter, which one is correct? (2017-250)
> **a+b+c+d（D，全對）**——CTU 可見先天異常/充盈缺損/擴張/狹窄/走行偏移;輸尿管重複為最常見泌尿道先天異常（約占人口1%）;擴張輸尿管不必然阻塞、阻塞輸尿管不必然擴張;ureterocele 為輸尿管穿膀胱壁處球狀擴張，呈投射入膀胱腔的圓形充盈缺損。[^2]

> [!question]- Which findings at CT urography favors fibroepithelial polyp over urothelial carcinoma? (2016-260)
> **兩者無法可靠區分，常需 ureteroscopy（A）**。Fibroepithelial polyp 為長條、平滑、有蒂之良性病灶，常被對比劑包繞（除蒂部），但影像上不一定能與惡性區分。[^3]
