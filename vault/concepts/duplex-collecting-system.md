# duplex-collecting-system

**泌尿道先天異常中「最常見」的答案就是重複集尿系統。** Weigert-Meyer 法則決定哪一極出哪種麻煩：上極輸尿管插入點偏內下方且異位，容易阻塞和形成 ureterocele；下極輸尿管插入正常但斜行短，容易逆流（VUR）。記憶鉤：**「上極往下走→塞；下極短路→逆流」**。

## Summary
- **流行病學**：**泌尿道最常見的先天異常之一**;約佔人口 ~0.8%、達就診泌尿症狀者的 8%。多無症狀偶然發現。[^1]
- **Weigert-Meyer 法則**：**上極輸尿管 → 異位、插入較內下方（易阻塞、ureterocele）**;**下極輸尿管 → 正位插入但短斜行、易 VUR**。[^1]
- 併發症：感染、逆流、阻塞、ureterocele。[^1]

> [!note] 考點：泌尿道最常見的先天異常?
> **Ureteral duplication（重複集尿系統）**。其次為 UPJ 阻塞、原發性巨輸尿管、VUR。[^1][^2]
