# organizing-pneumonia

**OP 是「周邊/支氣管中心 consolidation＋atoll（reversed halo）sign＋移行性＋類固醇反應佳」的肺病，COP 多可逆，但合併 CTD（polymyositis/antisynthetase）的 fibrosing OP 則不易消退。** 判讀分水嶺一：**周邊或 bronchocentric consolidation＋GGO＋air bronchogram＋perilobular opacities＋atoll sign＋migratory**；判讀分水嶺二：**COP（cryptogenic）類固醇反應佳；fibrosing OP（fOP）不完全消退、與 fibrotic NSIP 重疊、好發 polymyositis/antisynthetase syndrome**。記憶鉤：atoll sign＝環礁（外圍 consolidation 包住中央 GGO），是 OP 的看圖標誌。

## Summary
- **典型影像**：**周邊或 bronchocentric consolidation**;GGO、air bronchogram、**curvilinear perilobular／nodular opacities**、**atoll sign（reversed halo）**、peripheral sparing;病灶常**移行(migratory)**。[^1]
- **治療/預後**：[^1]
  - **多數（尤其 cryptogenic OP, COP）對類固醇反應佳、可消退**。
  - **Fibrosing OP（fOP）**：部分 OP 經長期治療仍不完全消退;與 **fibrotic NSIP overlap**;**好發於 polymyositis／antisynthetase syndrome 等 CTD-ILD**,不如典型 OP 易逆轉。[^2]

## 影像判讀

### 定義與分類
- **OP 為一種組織病理／影像-臨床型態**，非單一疾病：肺泡管與肺泡腔內充滿肉芽組織栓（Masson body）。可分為**cryptogenic OP（COP，無可辨識病因，即特發性）**與**secondary OP（次發性）**。[^3]
- **次發性 OP 的觸發因子**：感染、藥物、結締組織病（CTD）、發炎性腸道疾病（IBD）、移植後、**胸部放射治療**等。[^3] 故影像看到 OP 型態時，臨床病史（用藥、CTD、放療、移植）是判別 COP vs secondary 的關鍵。

### HRCT 特徵與徵象
- **典型（classic）型態**：**周邊（subpleural）與 peribronchovascular（支氣管血管周圍）consolidation**，常合併 GGO;為 OP 最具代表性的兩種分布。[^3]
- **非典型（atypical）型態**：結節、crazy-paving、線狀／帶狀影（lines and bands）、**perilobular（小葉周邊）consolidation**、以及**reversed halo sign（atoll sign）**。[^3]
- **Reversed halo sign（atoll sign）**：中央 GGO 被新月形或環形的緻密 consolidation 包圍;最初被描述為**對 COP 相對具特異性的徵象**;OP 是 reversed halo 最常見的成因。[^1][^4][^5] 但此徵象並非 OP 專屬（見下方 DDx）。

### 分布與移行性
- 偏好**下肺野、周邊／胸膜下與支氣管血管周圍**分布;**peripheral sparing**（緊貼胸膜的最外層反而清空）可見。[^1]
- **移行性（migratory）**：病灶在追蹤影像上此消彼長、位置改變，是 OP 的重要動態特徵，有助與固定性病灶（如腫瘤、慢性感染）區別。[^1]

### DDx（reversed halo sign 的鑑別）
- **OP 以外可出現 reversed halo 的疾病**：侵襲性肺黴菌感染（IFI，如 mucormycosis／aspergillosis）、paracoccidioidomycosis、Pneumocystis 肺炎、結核、社區型肺炎、lymphomatoid granulomatosis、GPA（Wegener）、lipoid pneumonia、sarcoidosis、肺腫瘤、肺梗塞，以及放療後與射頻燒灼（RFA）後變化。[^4]
- **環壁／病灶內有小結節** → 偏向**活動性肉芽腫病（結核、sarcoidosis）**而非 OP。[^5]
- **免疫低下病人出現 reversed halo** → 先當作**感染（IFI）**處理直到證實為止;mucormycosis 與侵襲性 aspergillosis 多見於嚴重免疫抑制（血液惡性腫瘤）。[^5]
- **IFI vs OP 的量化鑑別點**（biopsy-proven 比較研究）：**reversed halo 內出現網狀影（reticulation）**見於 93% IFI、0% OP;**外緣 consolidation rim 厚度** IFI 約 2.0 cm vs OP 約 0.5 cm（**rim >1 cm 偏向 IFI**）;**合併肋膜積液** IFI 73% vs OP 0%。三者皆強烈傾向 IFI 而非 OP。[^6]
- **其他需鑑別的 consolidation 性 ILD**：chronic eosinophilic pneumonia（同為周邊上肺野 consolidation，臨床看嗜酸性球與過敏史）、感染性肺炎、肺癌／lymphoma（含黏液型腺癌可呈持續性 consolidation）、肺梗塞、肉芽腫病。[^4]

### 判讀陷阱
- **Reversed halo 並非 OP 專屬**：單憑此徵象不能定診 OP，須整合臨床情境與其他伴隨影像（環壁結節、rim 厚度、肋膜積液、免疫狀態）;免疫低下者尤其常需切片。[^4][^5][^6]
- **持續不消退的 consolidation 勿一律歸 OP**：fibrosing OP 與 fibrotic NSIP 影像重疊;持續性周邊 consolidation 亦見於黏液型腺癌、lymphoma，必要時切片。[^2][^3]
- **次發性 OP 易被當成原發疾病**：看到 OP 型態務必回查放療、藥物、CTD、移植病史，否則漏掉真正病因。[^3]

> [!note] 考點
> OP 核心＝**周邊/支氣管中心 consolidation ＋ atoll(reversed halo) sign ＋ 移行性 ＋ 類固醇反應佳**;**fOP（fibrosing OP）較不可逆、常見於 CTD（PM/antisynthetase）**。[^1][^2]
