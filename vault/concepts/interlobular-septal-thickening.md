# interlobular-septal-thickening

**小葉間隔增厚看「質地」定方向：平滑→液體（肺水腫）；串珠 beading→細胞/沉積（sarcoid、淋巴管癌、lymphoma、amyloid）；不規則→纖維化。** 判讀分水嶺一：**肺水腫＝平滑型——它「最不會」造成串珠/不規則增厚**，考題若問「最不可能 beading」答案即肺水腫；判讀分水嶺二：**sarcoidosis、lymphoma、amyloidosis 皆可造成結節/串珠型，lymphangitic carcinomatosis 兩型（smooth 或 nodular）皆可**。

## 判讀骨架

### 1. 解剖基礎：secondary pulmonary lobule（次級肺小葉）
- 次級肺小葉約 **1–2 cm**，是 HRCT 可辨識的最小肺單位，由結締組織間隔包圍。理解小葉內外結構的位置，是判讀「中央型 vs 周邊型」病灶分布的根本。[^anat][^ra]
- **中央（centrilobular）**：終末／呼吸性細支氣管，伴行**小葉中央動脈（centrilobular artery）**，在 HRCT 上表現為小葉中心約 1 mm 的點狀影；旁有動脈周淋巴管。中央型病灶 → 細支氣管／血管源（如 RB-ILD、過敏性肺炎、tree-in-bud）。[^anat][^ra]
- **周邊（interlobular septa，小葉間隔）**：較厚的結締組織間隔，垂直於肋膜走行，內含**肺靜脈小支（pulmonary venules）與淋巴管**，引流向肺門及肋膜面。小葉間隔增厚＝這些周邊結構（靜脈／淋巴／間質）被液體、細胞或纖維填塞所致。[^anat][^ra]
- 沿小葉間隔＋支氣管血管束＋肋膜下＋葉間裂分布者 → **perilymphatic（淋巴道周圍）分布**，是 sarcoidosis、矽肺、淋巴管癌、amyloidosis 的共同分布特徵。[^perilymph][^anat]

### 2. 看「質地」定方向
- **Smooth（平滑型）**——最不具特異性，多為**液體／靜脈淋巴鬱積或瀰漫浸潤**。代表：**pulmonary edema（肺水腫）**（產生 Kerley B lines）、lymphangitic carcinomatosis（亦可結節型）、病毒性肺炎（COVID-19、CMV）、yellow nail syndrome、藥物毒性／放射性肺炎、Erdheim–Chester。[^anat][^ra]
- **Nodular / beaded（結節／串珠型）**——較具特異性，多為**沿淋巴道的細胞或沉積物**：lymphangitic carcinomatosis、**sarcoidosis**、**矽肺／煤工塵肺（silicosis / CWP）**、lymphoma／lymphoproliferative、LIP、**amyloidosis**。串珠樣（beaded）間隔增厚是 perilymphatic 沉積的典型表現。[^anat][^perilymph]
- **Irregular（不規則型）**——伴**結構扭曲／纖維化**：UIP/IPF、間質纖維化、asbestosis（常伴牽引性支氣管擴張、蜂窩肺）。[^anat]

### 3. 判讀分水嶺（鑑別決策）
- **平滑、對稱、雙側＋小葉中心模糊＋胸腔積液／心臟擴大／血管充血 → 肺水腫（心因性）。** 肺水腫是**最不會**造成串珠／不規則增厚者——考題若問「最不可能 beading」即指肺水腫。[^anat][^ra]
- **結節／串珠、分布不對稱（約 50% 為局部或單側）、肺小葉整體結構保留（preservation of lobular architecture）、可伴肺門腫塊或單側肋膜積液 → lymphangitic carcinomatosis（癌性淋巴管炎）。** 「結構保留」是與纖維化、水腫鑑別的關鍵。[^lc][^perilymph]
- **結節／串珠＋上肺及肺門周圍為主＋對稱性縱膈／肺門淋巴結腫大 → sarcoidosis。** 對比 LC 多為基底部、結構保留且常伴單側病灶。[^perilymph][^lc]
- **結節／串珠＋上肺為主＋職業暴露史（±蛋殼樣鈣化淋巴結） → 矽肺／塵肺。**[^anat][^perilymph]

### 4. Crazy-paving（鋪石路徵）之 DDx
- 定義：**毛玻璃影（GGO）＋疊加的小葉間隔／小葉內間隔增厚**，呈不規則多邊形「鋪石」狀；非 PAP 專屬，須以分布、臨床急緩、病程綜合判讀。[^cp][^ra]
- 經典：**PAP（肺泡蛋白沉積症）**——geographic GGO 加平滑網狀增厚，常與正常肺界線分明。[^cp]
- 其他須鑑別：**肺水腫／ARDS、瀰漫性肺泡出血（DAH）、Pneumocystis（PCP）肺炎、機化性肺炎（OP）、黏液性腺癌（mucinous adenocarcinoma）、類脂性肺炎、COVID-19、感染**。臨床線索（急性發燒／免疫低下→感染、PCP；咳血／血色素降→出血；慢性緩進→PAP、腺癌）為主要區辨依據。[^cp][^ra]

## Summary
- **Smooth（平滑）**：**pulmonary edema（肺水腫，Kerley B）**、lymphangitic carcinomatosis（亦可結節）、病毒性肺炎、藥物／放射性、yellow nail syndrome、Erdheim–Chester。為最不具特異性之型態。[^anat][^ra]
- **Nodular / beaded（結節/串珠）**：lymphangitic carcinomatosis、**sarcoidosis、矽肺／煤工塵肺、lymphoma/LIP、amyloidosis**；屬 perilymphatic 沉積。[^anat][^perilymph]
- **Irregular（不規則）**：纖維化（UIP/IPF、asbestosis），伴結構扭曲。[^anat]
- **判讀分水嶺**：平滑對稱＋積液＋心擴大→肺水腫；結節串珠＋不對稱＋小葉結構保留→癌性淋巴管炎；結節串珠＋上肺／肺門＋對稱淋巴結→sarcoidosis。[^lc][^perilymph]
- **Crazy-paving DDx**：PAP、肺水腫、DAH、PCP、OP、黏液腺癌、感染——以臨床急緩與分布定向。[^cp]
- **題目**：問「最不會」造成 beading/不規則（即結節型）增厚者 → **pulmonary edema**（屬平滑型）（2016-112 正解 C）；sarcoidosis(A)、lymphoma(B)、amyloidosis(D) 皆可造成結節型。[^anat]

> [!note] 考點：何者「最不會」造成串珠/不規則小葉間隔增厚？
> **Pulmonary edema（肺水腫）**——典型為**平滑型**。sarcoidosis、lymphoma、amyloidosis 皆可造成**結節/串珠型**。[^anat]
