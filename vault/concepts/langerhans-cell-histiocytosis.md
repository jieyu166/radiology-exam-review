# langerhans-cell-histiocytosis

**骨骼 LCH 的招牌是「洞中有洞」（beveled edge，hole-within-a-hole）——內外骨板侵蝕不對稱；脊椎壓扁成餅叫 vertebra plana，兒童最常見的病因就是它。** 判讀分水嶺一：**骨骼表現＝無硬化緣溶骨＋beveled edge（hole-within-a-hole）＋button sequestrum；脊椎→vertebra plana（兒童最常見病因）；下頜→floating tooth**；判讀分水嶺二：**臨床分型：單發（eosinophilic granuloma）→ 多發單系統（Hand-Schüller-Christian：顱骨缺損＋尿崩＋突眼）→ 多系統（Letterer-Siwe，嬰幼兒最惡性）**；肺型（PLCH）屬同譜但需分開看 → [[plch]]。

## Summary
- **疾病譜（臨床分型）**：[^1]
  - **單發（unifocal）＝eosinophilic granuloma（EG）**：單一骨溶骨灶。
  - **多發單系統（Hand-Schüller-Christian）**：經典三聯——溶骨性顱骨缺損＋尿崩症（diabetes insipidus）＋突眼。
  - **多系統（Letterer-Siwe）**：嬰幼兒、侵襲性、預後差。
  - **肺 LCH（PLCH）**：年輕**抽菸**成人之肺囊腫＋結節——**與骨/系統性 LCH 同疾病譜但表現不同 → 見 [[plch]]**。
- **族群**：骨骼 LCH 好發**較大兒童（平均 7±4 歲）與年輕成人（36±12 歲）**，M:F ~2:1。[^1]
- **骨骼為最常受侵器官**；部位：**顱骨 ~50%**（兒童 calvarium、成人 skull base）、**肋骨 ~15%**、**脊椎 ~15%**、骨盆/股骨 ~15%、下頜 ~7%。[^1]
- **影像特徵**：**溶骨、無硬化緣**；**beveled edge / hole-within-a-hole**（內外板不對稱）；**button sequestrum**；活動期可呈侵襲性（permeative、骨膜反應）、癒合期可硬化。[^1]
- **特徵徵象**：**脊椎 → vertebra plana**（**兒童 vertebra plana 最常見病因**，好發胸椎）；下頜 → **floating tooth**（lamina dura 喪失）。[^1]

> [!note] LCH vs [[plch]]（同疾病譜、不同表現）
> **骨/系統性 LCH**＝兒童為主、溶骨 hole-within-a-hole、可多系統；**[[plch]]（肺）**＝年輕**抽菸**成人、上中肺囊腫＋結節、抽菸相關。兩者同屬 Langerhans 細胞疾病譜但**年齡/器官/影像/關聯不同**，分開判讀。[^1]

> [!note] 與 [[rib-tumor]] 的連結
> LCH 是**兒童**肋骨/骨病灶 DDx 之一（溶骨、可侵襲似惡性亦可自癒）。[^1]

## 題目
> [!question]- 骨骼 LCH 的特徵性溶骨表現？脊椎與下頜的招牌徵象？
> **無硬化緣溶骨＋beveled edge（hole-within-a-hole）＋button sequestrum**；脊椎 → **vertebra plana**（兒童最常見病因）、下頜 → **floating tooth**。[^1]

> [!question]- 骨/系統性 LCH 與肺 LCH（PLCH）有何不同？
> 同屬 Langerhans 細胞疾病譜但表現不同：**骨/系統性 LCH＝兒童溶骨/多系統**；**PLCH＝年輕抽菸成人之肺囊腫＋結節**（見 [[plch]]）。[^1]
