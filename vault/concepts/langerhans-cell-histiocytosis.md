# langerhans-cell-histiocytosis

**骨骼 LCH 的招牌是「洞中有洞」（beveled edge，hole-within-a-hole）——內外骨板侵蝕不對稱；脊椎壓扁成餅叫 vertebra plana，兒童最常見的病因就是它。** 判讀分水嶺一：**骨骼表現＝無硬化緣溶骨＋beveled edge（hole-within-a-hole）＋button sequestrum；脊椎→vertebra plana（兒童最常見病因）；下頜→floating tooth**；判讀分水嶺二：**臨床分型：單發（eosinophilic granuloma）→ 多發單系統（Hand-Schüller-Christian：顱骨缺損＋尿崩＋突眼）→ 多系統（Letterer-Siwe，嬰幼兒最惡性）**；肺型（PLCH）屬同譜但需分開看 → [[plch]]。

## Summary
- **疾病譜（臨床分型）**：[^1]
  - **單發（unifocal）＝eosinophilic granuloma（EG）**：單一骨溶骨灶。
  - **多發單系統（Hand-Schüller-Christian）**：經典三聯——溶骨性顱骨缺損＋尿崩症（diabetes insipidus）＋突眼。
  - **多系統（Letterer-Siwe）**：嬰幼兒、侵襲性、預後差。
  - **肺 LCH（PLCH）**：年輕**抽菸**成人之肺囊腫＋結節——**與骨/系統性 LCH 同疾病譜但表現不同 → 見 [[plch]]**。
- **族群**：骨骼 LCH 好發**較大兒童（平均 7±4 歲）與年輕成人（36±12 歲）**，M:F ~2:1。[^1]
- **骨骼為最常受侵器官**；部位：**顱骨 ~50%**（兒童 calvarium、成人 skull base）、**肋骨 ~15%**、**脊椎 ~15%**、骨盆/股骨 ~15%、下頜 ~7%。[^1]
- **影像特徵**：**溶骨、無硬化緣**；**beveled edge / hole-within-a-hole**（內外板不對稱）；**button sequestrum**；活動期可呈侵襲性（permeative、骨膜反應）、癒合期可硬化。[^1]
- **特徵徵象**：**脊椎 → vertebra plana**（**兒童 vertebra plana 最常見病因**，好發胸椎）；下頜 → **floating tooth**（lamina dura 喪失）。[^1]

- **骨骼判讀骨架（最常受侵；判讀重點）**：[^2]
  - **顱骨**：地圖狀（geographic）溶骨、**無硬化緣**；**內外骨板侵蝕不對稱 → beveled edge／hole-within-a-hole（洞中有洞）**；殘存骨片可呈 **button sequestrum（鈕扣狀死骨）**。
  - **長骨**：髓質溶骨，活動期可見**侵蝕性（permeative）邊界、層狀骨膜反應、軟組織腫塊**——影像可酷似惡性（Ewing／骨髓炎），癒合期反轉為硬化。
  - **脊椎**：椎體均勻塌陷成扁餅 → **vertebra plana**，**椎間盤高度與後側元素通常保留**（與感染／轉移破壞終板的型態不同）。
  - **下頜／齒槽骨**：lamina dura 破壞、牙齒懸浮於溶骨灶 → **floating tooth**。
- **肺型 LCH（PLCH）判讀骨架**：年輕（20–40 歲）**吸菸**成人；HRCT **上、中肺為主、肋膈角（costophrenic angle）相對保留**；**結節（常 <10 mm、邊緣不規則、中心小葉分布）＋ bizarre 形（雙葉、三葉、分支狀）囊腫**的混合，**隨病程由結節→空洞結節→厚壁囊腫→薄壁囊腫演進**，晚期類似肺氣腫；**易併發自發性氣胸**。詳見 → [[plch]]。[^3]
- **中樞神經系統（CNS）判讀骨架**：[^4]
  - **下視丘—垂體軸／漏斗（infundibulum）**：**漏斗增厚／強化 >3 mm**（兒童系列約 32%），對應**中樞性尿崩症**，可伴後葉 T1 高訊號 bright spot 消失。
  - **顱骨／顱底骨病灶**為最常見的顱內相關異常（約 55%）。
  - **松果體增大**（約 45%）。
  - **神經退化性（neurodegenerative）變化**：**小腦齒狀核（dentate nucleus）T2／FLAIR 對稱高訊號**（約 29%），為非腫瘤性退化表現；部分文獻另述基底節受侵，惟有系列未觀察到基底節／下視丘訊號變化（數字分歧並列）。
- **腹部／淋巴：肝、脾、骨髓（hematopoietic system）受侵屬「危險器官（risk organ）」**；可見肝腫大、脾腫大、淋巴結腫大。[^5]
- **臨床分期（影響預後與處置）**：**單系統（single-system，單部位或多部位）vs 多系統（multisystem，≥2 系統）**；多系統再依**危險器官（肝／脾／骨髓）陽性與否**分高、低風險——**risk-organ 陽性（多見於嬰兒）預後最差**，單系統存活率近 100%。[^5]

> [!note] LCH vs [[plch]]（同疾病譜、不同表現）
> **骨/系統性 LCH**＝兒童為主、溶骨 hole-within-a-hole、可多系統；**[[plch]]（肺）**＝年輕**抽菸**成人、上中肺囊腫＋結節、抽菸相關。兩者同屬 Langerhans 細胞疾病譜但**年齡/器官/影像/關聯不同**，分開判讀。[^1]

> [!note] 與 [[rib-tumor]] 的連結
> LCH 是**兒童**肋骨/骨病灶 DDx 之一（溶骨、可侵襲似惡性亦可自癒）。[^1]

## 鑑別與報告要點
| 部位 | 特徵 |
| --- | --- |
| 顱骨（~50%） | 無硬化緣溶骨、**beveled edge / hole-within-a-hole**、button sequestrum[^1] |
| 脊椎（~15%） | **vertebra plana**（兒童最常見病因）[^1] |
| 肋骨（~15%） | 溶骨、可膨脹[^1] |
| 下頜 | **floating tooth**[^1] |

#### 各系統鑑別診斷（DDx）
| 表現 | 主要鑑別 | 區辨重點 |
| --- | --- | --- |
| 兒童溶骨性骨病灶（侵蝕性） | Ewing sarcoma、骨髓炎（osteomyelitis）、轉移／神經母細胞瘤 | LCH 可自癒、beveled edge／button sequestrum；Ewing 多有 onion-skin 骨膜＋大軟組織腫塊；骨髓炎臨床發炎徵象＋膿瘍[^2] |
| vertebra plana | LCH（兒童最常見）；成人需想轉移、骨髓瘤、感染、外傷 | LCH 椎間盤與後側元素保留；感染／轉移破壞終板與椎間盤[^2] |
| 上中肺囊腫（成人吸菸者） | LAM（女性、瀰漫均勻圓囊、無上肺偏好）、中央小葉肺氣腫（無真正囊壁）、PCP 肺囊腫 | PLCH 上中肺、bizarre 形囊腫＋結節、肋膈角保留、吸菸史[^3] |
| 漏斗增厚／中樞性尿崩 | 生殖細胞瘤（germinoma）、神經類肉瘤病（neurosarcoidosis）、轉移、淋巴球性漏斗神經垂體炎 | 影像常難區辨；LCH 多伴骨／肺病灶與齒狀核退化訊號，臨床年齡與全身分布輔助判斷[^4] |
