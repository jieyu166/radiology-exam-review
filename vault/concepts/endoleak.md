# endoleak

**EVAR 後漏血的「從哪進來」決定型別：I＝端點沒封好、II＝側枝回流（最常見）、III＝接合處或fabric破孔、IV＝本身多孔、V＝瘤囊脹大但找不到漏。** 判讀分水嶺：高壓危險型需處理＝**I 和 III**；type II 最常見、壓力低、多觀察即可。記憶鉤：I（入口端點）、II（側枝 In-flow）、III（接合破損）、IV（孔洞 porosity）——依「誰讓血進來」排序。

## Summary
- **Type I**:**graft 附著處(landing zone)封合不全**(Ia 近端/Ib 遠端/Ic iliac occluder)——高壓、破裂風險,須處理。[^1]
- **Type II**:經**分枝血管(腰動脈/IMA)逆流**回填瘤囊——**最常見**,多良性。[^1]
- **Type III**:**移植物結構缺損**(IIIa 模組接合分離、IIIb 織料破孔/骨折)——高壓,須處理。[^1]
- **Type IV**:**移植物織孔 porosity**(常術中、抗凝停止後自限)。[^1]
- **Type V**:endotension(瘤囊增大但無可見漏)。[^1]
- 本題(何者正確):A(最常見 type I)、B(type II 與破裂)、C(type III 與 porosity)皆錯;**D(type I 因封合/裝置失敗)正確**(2016-096 正解 D)。[^1]

> [!note] 考點:何者「正確」?
> **type I＝附著處封合(device)失敗**(正解)。錯者:最常見其實是 type II;與破裂相關是 type I/III(非 II);porosity 是 type IV(非 III)。[^1]

# 分型(I–V)

依「血從哪裡進入瘤囊」分型,直接決定壓力高低與處置急迫性。[^2][^3]

- **Type I —附著(seal/landing zone)失敗**:移植物與血管壁貼合不全,高壓血直接灌入瘤囊。**Ia** 近端附著處、**Ib** 遠端附著處、**Ic** aorto-uni-iliac 修補時 iliac occluder plug 封合不良。屬高壓型,破裂風險高,須**立即處理**。[^2][^3]
- **Type II —分枝血管逆流回填(最常見)**:瘤囊已於近遠端完全隔絕,但經被覆蓋的原生分枝**逆向**灌流回填瘤囊;最常見來源為 **IMA(腸繫膜下動脈)** 與**腰動脈(lumbar)**,通常有一條主要 inflow、合併一條以上 outflow。EVAR 後最常見的 endoleak、也是再介入的主因;多為低流量、常自行栓塞消退。[^2][^3]
- **Type III —移植物結構缺損**:**IIIa** 模組組件接合脫離(junctional separation)、**IIIb** 織料破孔/撕裂(fabric tear/hole)。高流量、瘤囊高壓,機轉與風險近似 type I,**須立即處理**;新世代移植物已少見(一研究:第一/二代 12.7% vs 第三代 1.3%)。[^2][^3]
- **Type IV —移植物織料孔隙(porosity)**:經移植物織料微孔短暫滲漏,多見於術中剛置放、停抗凝後自限;隨織料技術進步現多屬歷史名詞。[^2][^3]
- **Type V — endotension**:瘤囊持續增大但影像找不到可辨識的漏(診斷需排除其他型);非直接高壓,保守觀察標準尚不明確。[^2][^3]

# CTA 多期協定

CTA 是 endoleak 偵測與分型的主流(gold standard)影像工具。[^2][^4]

- **理想三期**:**non-contrast(平掃)→ arterial(動脈期)→ delayed(延遲期)**。平掃用來區分高密度(鈣化、術中殘留對比、移植物金屬/endograft material)與真正的對比外漏,避免把鈣化/金屬假影誤判為漏。[^2]
- **delayed phase 為關鍵**:endoleak 在延遲期最明顯;**低流量 type II** 對比進入瘤囊慢,常**只在延遲期**才看得到,動脈期可能完全陰性。省略延遲期會漏掉慢速 type II。[^2][^4]
- **流量決定顯影時相**:dynamic CTA 量到高壓型(type I/III)瘤囊對比達峰與主動脈幾乎同步(時間差約 2 秒),type II 則明顯延後(約 5 秒),可用「顯影時間差」輔助分型。[^4]
- **劑量取捨**:多期增加輻射與成本,實務上常退而求其次用單一動脈期或雙期(dual-bolus)做常規追蹤,**三期保留給問題排解(problem solving)**——如分型不確定、瘤囊增大卻找不到漏。[^2]
- **dynamic/4D CTA**:寬探測器(如 320-row、16 cm)可做時間解析(cine)掃描,提升 endoleak 偵測與分型的敏感度/特異度。[^4][^5]

# 各模態 / 各型影像辨識

- **DUS / CEUS(超音波)**:常規 EVAR 後追蹤可用,無輻射、可近性高、成本低;CEUS 對 endoleak 偵測敏感度/特異度高,文獻顯示偵測與分型至少不亞於 CT,並能**即時**顯示血流進入瘤囊的方向、有助釐清型別與解剖。準確度高度依賴操作者經驗。[^6][^7]
- **MRI/MRA**:增強 MRI 無輻射,對**隱匿性(occult)endoleak** 有時優於 CTA;但因成本高、檢查久、可近性差、部分移植物非 MR 相容,不列入常規追蹤。[^2]
- **DSA / C-arm CT**:導管血管攝影提供時間解析,作為**非侵入影像無法分型**或**瘤囊增大卻看不到漏**時的問題排解工具;C-arm CT(cone-beam,如 DynaCT)輔助 DSA 可更可靠定位與分型(一前導研究:單獨 DSA 可分型 9/12,加 C-arm CT 提升至 11/12)。[^3][^8]
- **多模態合併**:DUS + CTA + MRI 合併可把 endoleak 偵測/分型率提升至約 91%。[^2]

# 處置原則

- **Type I**:屬高壓、破裂風險高,**強制處理**。Ia 以近端 endograft 延伸 ± EndoAnchor、cuff/Palmaz stent、必要時 FEVAR/BEVAR/ChEVAR;頸長不足或上述失敗時改經導管栓塞(coils/Onyx/glue)。Ib 多以遠端 endograft 延伸達成遠端封合。[^2]
- **Type II**:**先觀察為原則**。低流量、常自限,僅在**瘤囊持續增大(常用閾值:6 個月內 ≥5 mm)** 時介入;主流為栓塞,目標是封掉 endoleak nidus(類比血管畸形的中央 nidus)及供血枝,途徑含經動脈(IMA 最易)、transiliac paraendograft、直接穿刺瘤囊(translumbar/經腹)、transcaval。[^2]
- **Type III**:高壓,**須立即處理**;標準作法是在原移植物內再襯一段新 endograft(relining)。[^2]
- **Type IV**:多術中自限,停抗凝後緩解,通常無需特別介入。[^2]
- **Type V(endotension)**:可選擇觀察(非直接高壓);若瘤囊持續增大,選項含 extension cuff、relining、轉開放手術。[^2]

# 判讀陷阱

- **type II vs 隱匿 type I/III**:面對「疑似 type II 且瘤囊增大」務必先排除偽裝成 type II 的**隱匿 type Ib 或 type III**——此時被覆蓋的主動脈側枝其實是**出口(exit)而非入口**血管。CTA 通常足夠;不確定時用導管攝影(± cone-beam CT)確認漏的源頭。[^2]
- **delayed phase 不可省**:低流量 type II 動脈期可陰性,須延遲期才現形;只做動脈期會漏診慢速漏。[^2][^4]
- **鈣化 / 金屬與術中對比假影**:沒有平掃對照時,瘤囊壁鈣化、移植物金屬高密度或術中殘留對比易被誤判為漏;先看 non-contrast 區分。[^2]
- **壓力分層別記錯**:高壓危險、需立即處理的是 **type I 與 III**;type II 雖最常見但低壓,多觀察。考題常以「最常見=type I」「type II 與破裂相關」「porosity=type III」設陷阱,皆為錯敘。[^2][^3]
