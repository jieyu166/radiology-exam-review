# immune-checkpoint-inhibitor-pneumonitis

**免疫檢查點抑制劑肺炎治療成功後，約三分之一病人「可以」重新使用免疫檢查點抑制劑——但其中25-28%會於再治療期間復發肺炎，這是本題判讀關鍵（並非「絕對不能」重新使用）。**

## Summary
- **irAE本質**：為免疫檢查點抑制劑治療特有的一組獨特併發症(unique set of complications)。[^1]
- **ICI相關肺炎(pneumonitis)**：呈現廣泛之臨床與影像表現光譜(wide spectrum)，其影像型態反映疾病嚴重度。[^1]
- **類肉瘤病樣肉芽腫及淋巴結病(sarcoid-like granulomatosis and lymphadenopathy)**：為另一項重要之胸腔irAE。[^1]
- **治療**：暫停免疫檢查點抑制劑並給予類固醇治療(corticosteroid therapy)。[^1]
- **重新使用ICI之可行性**：現有數據顯示**約三分之一(one-third)病人**於肺炎成功治療後**可以重新使用**免疫檢查點抑制劑（並非絕對禁止，此為常見誤判方向）；惟其中**25-28%**於再治療期間會經歷**復發性肺炎(recurrent pneumonitis)**。[^1]

> [!note] 考點：ICI相關肺炎治療後可否重新使用ICI？
> **病人於肺炎成功治療後「無法」重新使用ICI治療（E為偽）**——實際上約三分之一病人可重新使用，其中25-28%會復發肺炎，此敘述錯誤。irAE為ICI特有併發症(A)、ICI肺炎表現光譜廣且反映嚴重度(B)、類肉瘤病樣肉芽腫為另一重要胸腔irAE(C)、治療為停藥+類固醇(D)皆為正確描述。[^1]

## 題目
> [!question]- Which choice is false about immune-related adverse events (irAEs)? (2018-034)
