# neurofibromatosis-type-1

**NF1的典型找法：視神經膠質瘤、蝶骨發育不良、神經根神經纖維瘤——但「雙側聽神經鞘瘤」是NF2的招牌，不是NF1，這是兩型神經纖維瘤病最常見的混淆陷阱。**

## Summary
- **NF1典型發現**：視神經膠質瘤(optic nerve glioma)、蝶骨發育不良(sphenoid dysplasia)、神經根神經纖維瘤(nerve root neurofibroma)、神經纖維瘤(plexiform neurofibroma)、café-au-lait斑點等。[^1]
- **非NF1典型發現**：**雙側聽神經鞘瘤(bilateral acoustic/vestibular schwannoma)為NF2之特徵性表現**，並非NF1。[^1]

> [!note] 考點：NF1發現，何者「除外」？
> **Acoustic schwannoma（C）**——為NF2特徵，非NF1。Optic nerve glioma(A)、sphenoid dysplasia(B)、nerve root neurofibroma(D)皆為NF1典型發現。[^1]
