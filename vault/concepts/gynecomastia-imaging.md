# gynecomastia-imaging

**男子女乳症（gynecomastia）主要位置在乳暈下方（subareolar），可以是左右不對稱，並非必定對稱分布。早期 nodular/florid phase（< 1 年）仍可逆；慢性 dendritic/quiescent phase（> 1 年）因纖維化而不可逆。**

## Summary
- **乳房攝影（mammography）**：可呈 nodular、dendritic 或 diffuse glandular 型態之 density。[^1]
- **超音波（ultrasound）**：subareolar 之 hypoechoic mass，可有或無 color Doppler flow。[^1]
- **分布**：主要位置在 subareolar，**可以左右不對稱**——並非必定對稱分布。[^1]
- **可逆性與病程階段**：早期**nodular/florid phase（< 1 年）仍可逆**（ductal proliferation + periductal edema），可因去除病因或藥物治療而消退；慢性**dendritic/quiescent phase（> 1 年）因纖維化（stromal fibrosis）而不可逆**，影像上呈 dendritic/flame-shaped pattern。[^2]
- **鑑別診斷**：主要鑑別為**男性乳癌（male breast cancer）**——乳癌通常偏離乳暈中心（eccentric）且邊緣不規則，而 gynecomastia 多以乳暈為中心（concentric）。[^2]

## 陷阱
- 「主要位置在 subareolar，必定為對稱分布」為錯誤敘述——gynecomastia **可為單側或左右不對稱**。[^1]
- 「Gynecomastia is usually irreversible」為錯誤——早期 nodular/florid phase **仍可逆**，僅慢性 dendritic/quiescent phase 才不可逆。[^2]
