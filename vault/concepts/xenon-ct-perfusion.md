# xenon-ct-perfusion

**氙電腦斷層（Xe/CT）灌注造影靠吸入性氙氣作為可擴散示蹤劑定量腦血流；標準吸入方式為氙氣與氧氣混合（非純氙、非與二氧化碳混合），確保病人在檢查中維持足夠血氧。**

## Summary
- **吸入方式：氙氣與氧氣混合吸入**——多數研究採約 28% 氙氣＋40% 氧氣＋32% 室內空氣的混合比例，而非純氙氣吸入或與二氧化碳混合（純氙或高濃度可能導致缺氧或麻醉效應）。[^1]
- **原理**：氙為原子序 54 之惰性氣體、快速擴散穿過肺泡與血腦障壁，經吸入後濃度快速平衡，**不需動脈導管取樣**（面罩測得之終末潮氣氙濃度即可準確反映動脈濃度）。[^1]
- **臨床用途**：定量腦血流（CBF），早期偵測腦缺血部位與範圍、界定可逆性損傷區域，評估側枝循環與腦血管儲備能力（常搭配 acetazolamide 誘發試驗）。[^1]

## 判讀骨架
- **與其他氙相關造影技術區分**：Xe-133（放射性同位素，經 SPECT 造影腦血流評估）與穩定同位素 Xe/CT（非放射性，靠 CT 衰減變化定量）為不同技術，勿混淆；兩者皆可評估腦血流但原理與造影模態不同。

### 參考來源
[^1]: *Neurosurg Clin N Am*. 1996 Oct;7(4):605-16（2018 交換考題詳解 p.60 引用；Tier 2）。Xe/CT 吸入標準為氙氣與氧氣混合（常見比例約 28%氙+40%氧+32%室內空氣）；氙為快速擴散惰性氣體，不需動脈導管取樣；用於定量腦血流、缺血範圍、側枝循環與腦血管儲備評估。
